BlueSkyBike折叠电动自行车概念设计

发表于 2013-07-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

BlueSkyBike是工业设计师Richard Heath和蓝天设计工作室之间合作的一项重要成果。这项旨在掌握了悉尼市郊城市交通需要的代步工具设计创意增强了更多承载工具的实用性，将短距离代步、存储容量以及驾驶工具的外观设计风格完美的结合在一起，折叠存放方便快捷，承载量够大，外观设计简洁大方。工业设计师Richard Heat有比以往强调该短距离交通工具实用性的需要，目前设计师正在寻找合适的生产厂家，将这个可以折叠电动自行车从图纸变成现实的产品。

Chitin概念车设计

发表于 2013-07-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

Chitin概念车，甲壳类轻型结构结合内部平静与沉稳的架构。虽采用仿生原理，但概念的外观并没像甲虫那么丑，风格前卫，就像一块漂亮的手表，一件衣服或首饰。整个车身框架都采用领先的轻质材料，时尚的内饰，所有控件都被替换为一个个单触摸屏，控制和监控所有系统。

Forklift概念运输车设计

发表于 2013-07-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

日常生活中，一提到运输车，我们首先想到的就是体积庞大的重型卡车。如今，土耳其设计师Hamit Kanuni Kuralkan设计出了一款紧凑型Forklift运输车。这种新式车型不仅拥有优雅的外形，同时其超高的性能也让人敬畏。通过本田先进的技术，整个车体可以轻松抬起，在保证了货物的轻便运输的同时，也保证了它的安全性。

Bombardier巴士车设计

发表于 2013-07-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

当我们忙忙碌碌工作一天下班后，都渴望拥有一个轻松舒适的乘车环境，日常生活中的公交车更是屡见不鲜。如今，俄国设计师Vitaly Kamyshev为我们带来了一款更加优秀的巴士车—Bombardier。这款新设计不光为乘客提供了足够的舒适健康，在传统公交车的基础上，更多了一分空间的秘密与安全，独立车厢的设计满足了更多人们的需要。车厢内置的高科技多媒体设备更是让乘客感受到了它卓越的性能。

Wally 概念车设计

发表于 2013-07-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

考文垂大学的毕业生Matthew Hockham 为豪华游艇品牌Wally设计了一款跑车，将Wally 品牌与众不同的设计语言融入到了这辆概念汽车里。该款概念车运用了Wally 品牌的未来主义风格：锋利的边缘，黑色的玻璃曲面，以及高科技感的灰色基调。其侧面造型的设计灵感也正是来源于Wally Power 118 的特征线。无论是外部轮廓还是内部性能，都让该车成了汽车领域的新宠儿。
