智能减速带

发表于 2012-02-09 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

减速带是通过高出路面的凸起来迫使汽车减速的。不过，即使司机已经主动将速度降了下来，还是要忍受减速带的颠簸。假如能够给那些主动减速的司机一些奖励该有多好？

这款智能减速带就能够判断行车的速度，当车的时速低于30公里时，减速带就会放平，不会迫使车辆颠簸。而当车的时速高于30公里时，减速带就会升高，惩罚一下超速的司机。所以，只要你主动降低速度，就不必忍受减速带带来的颠簸了。

设计师：Jae-yun Kim & Jong-Su Lee

三层分类垃圾桶

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

折叠太阳灶

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

太阳灶是一款很久的设计了，通过圆锥型的反光板来将阳光聚焦在焦点上达到加热的目的。这种产品非常的清洁环保，适合在阳光充裕的地方使用。对于出外野营的人而言，也是个非常不错的选择。

但是普通的太阳灶体积都过于庞大，不过在2007德国红点设计大奖的获奖作品中就有这样一款折叠太阳灶，可以让太阳灶变得更加便携。

这款太阳灶的原理与普通的太阳灶完全相同，只是采用了折叠的设计，折叠起来时，看起来就像是一个普通的瓶子：高约25厘米。而当需要利用太阳灶加热的时候，只需将它的外壁伸展开来变成一个圆锥体，这样就可以像普通的太阳灶一样使用了。

设计师：Javier Bertani，Ezequiel Castro，Vera Kade

逗趣的斜跨自行车

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

2010年首尔自行车设计大赛上，一辆造型独特的斜跨自行车从3000多个作品中脱颖而出，让人一望便觉印象深刻。这款自行车有两个车头，要想在乘骑的时候更改行进方向，只需乘骑者向反方向侧身即可，无需操作自行车掉头。有趣的是，它的乘坐方式也十分特别————设计师将古典的女士骑马姿势引用到了设计中来，若是男性乘骑，其姿势不禁让人莞尔。此外，乘坐者要想在这款逗趣的自行车上掌握好平衡，也不是一件容易的事情呢！

设计：Michael Killian
