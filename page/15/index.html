多层滑盖式记忆卡盒

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着数码产品的普及，人们手里的记忆卡也多了起来，“可以存放记忆卡的钱包”是个很不错的创意。不过，我们今天为大家介绍的这款产品更酷，而且能够存放更多的记忆卡。

这款卡盒为不同型号的记忆卡设计了不同的卡槽，可以将记忆卡更好的存放在里面。多层的滑盖设计，能够在纤薄的身躯内存储更多的卡。彩色的外壳也更能突显时尚的色彩。

而且这款卡盒是由一次性杯子制成的，每七个杯子就能做成这样一款卡盒。充分体现了环保的创意。

设计师：Tom Kenworthy

反光裤

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

反光条是一种很常见的安全设备，可以在夜间反射周围的光线，从而对路人和司机起到一定的警示作用。但是要在普通的衣服外侧加上反光条似乎有些影响美观，所以就有公司设计出了这样一款反光裤。

它将反光条设计在了裤腿和裤兜内侧。夜间骑车的时候只要将裤腿挽起、裤兜外翻即可达到与传统反光条相同的效果。更重要的是，平时不用的时候也丝毫不会影响到衣物的美观。

轻质塑料电动车

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

一群在汽车设计工作室实习的高中生设计了这辆塑料外壳的电动车，这款车不仅造价低廉，而且十分节能。学生们利用旧赛车的底盘做基底，然后在上面按照设计好的外壳曲面焊接金属框架，最后将热缩塑料精细地覆盖在上面。这辆车采用电动发动机驱动，由21节锂离子电池供电，整车重量不足1200磅，其中塑料和框架部分仅重39磅。由于质量较轻，这辆车开起来十分节能，所需电能如果折合成汽油的话相当于每加仑汽油可以行使300英里。

可降解塑料环保易拉罐

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

在美国，每半分钟就会有1万6千个易拉罐被当作垃圾处理，如何将其有效回收，环保再利用确实是个大课题。而设计师则采用了新型可降解塑料，用于设计环保易拉罐。它采用通过玉米提炼出来的生物可降解塑料作为罐子主体，可以有效隔热，并支持加热至110度。除了有机环保材料占有优势以外，这款新型易拉罐也能如同铝制罐一样，能够轻松压缩、收集。

设计：Haoshi Design Studio

鳄鱼摩托

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

经佛罗里达州尔南多野生动物复原计划组织的许可，一名瑞典技工和皮革专家合作制成了这款货真价实的鳄鱼摩托。摩托时速可达每小时120英里，各项仪表都被嵌入短吻鳄的头部，而为了提高乘骑舒适度，摩托车座经过加工，并未采用原始的鳄鱼皮。这款鳄鱼摩托将于2010年5月在劳德代尔堡拍卖，起拍价为51000英镑，所得善款会全部用于动物保护工作。

设计：Jim Jablon & Benny Ohrman
