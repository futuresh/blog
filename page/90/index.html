最节约空间的床

发表于 2012-01-03 | 更新于 2018-07-24 | 分类于 环保&公益创意

家里有孩子可是房子不够大的朋友恐怕都在为孩子的屋子该怎么布置而发愁，尤其是有双胞胎的家庭，肯定更愁。毕竟现在的房价是一天比一天高，不晓得那辈子才能降下来。既然不能改变房子的大小，就应该尝试着去充分利用空间。今天我们在这里给大家介绍的套儿童床，绝对能给您带来惊喜。

其实这床已经不能单纯叫做床了，它甚至可以被称作一整套的家俱。不过最重要的是，它将这一整套的家俱完美的组合在了一起，却只占用了非常小的空间，而且布局合理，很适合家里地方不大的朋友使用。他有很多件配套物品可选：床、抽屉、书架、衣橱、电脑桌、椅子等。您可以根据自己的需要自由搭配。这款床给人非常清新实用的感觉，在充分利用了空间的同时还兼顾美观，虽然占用空间很小，但丝毫没有拥挤的感觉。特别适合家里有两个孩子的家庭。

无线自行车转向灯

发表于 2012-01-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

既然汽车和摩托车可以通过配备转向灯来提高安全性，那么为什么不给自行车也配上一款呢？

这套全新的无线转向灯就是专门为自行车设计的。它分成前灯和后灯两部分，前灯非常宽大，安装在车把上之后可以非常方便的用拇指来控制。而它最大的亮点在于前后灯通过无线传输来传递信号，当你启动了前面的转向灯后，后车灯就会马上接受到信号并亮起相应的转向灯，为你在夜间行车提供了更高的安全保障。

由于车灯的安装、拆卸非常简单，因此，将车停好之后，可以将车灯拆下带回家中，以防丢失。

该产品由碱性电池提供电量。

售价40英镑

柱形插线板

发表于 2011-12-31 | 更新于 2018-07-24 | 分类于 环保&公益创意

我们专利之家曾经为大家介绍过好几款“旋转插线板”的设计。而今天的这款设计更是将这种旋转的元素运用到了极致，它直接将插线板设计成了圆柱形，这样每个插口部位都可以360度旋转，从而获得更高的自由度，避免插头过大而挡住相邻的插口。

会呼吸的椅子

发表于 2011-12-28 | 更新于 2018-07-24 | 分类于 环保&公益创意
