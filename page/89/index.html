索尼新款环保数码相机

发表于 2012-01-03 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了更好的推广环保的理念，索尼曾经推出过“odo系列环保数码产品”，可以通过数码产品本身的机械运动来获取电力。日前，索尼公司又发布了odo系列的另一款全新的产品。

秉承之前的odo系列的设计理念，这款全新的数码相机同样采用了机械发电的原理：只需将这个手柄的圆环部位在桌子上不断的推动使其转动即可产生电力。而每转动15秒钟所产生的电力就可以拍摄一张照片。圆环的中空部位则是相机的取景器。

出于省电的考虑，这款数码相机并没有配备液晶屏，因此必须要连接到电脑上才能够查看照片。

挂在车牌上的刹车灯

发表于 2012-01-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

虽然开车的人都知道，红色的尾灯表示停车，但是如果前面的车突然急刹车，或者后面开车的人没有太注意，那么刹车灯似乎并不能起到太大的提示作用。

这款附加在车牌上的刹车灯就能起到一定的辅助作用。当探测到刹车的时候，上面的高亮度LED灯就会不停的闪烁三秒钟，即使后面的司机精力不集中也很难不注意这样炫目的灯光。

它并不需要连接到汽车的电力系统上，而是通过八个纽扣电池供电，每工作15,000个小时才需要更换一次电池。而这样的工作时间足够你驾驶两三年的了。

安全套做的衣服

发表于 2012-01-03 | 更新于 2018-07-24 | 分类于 环保&公益创意

衣服、安全套、艾滋病以及慈善事业他们之间有着什么样的联系呢？有一位非常聪明的35岁的巴西女艺术家正试图通过她自己的方式来唤起人们对艾滋病的关注。这位艺术家叫艾德里安.波蒂尼，在一次参加的某艾滋病预防组织的活动时，她接触到很多被艾滋病病毒感染的孩子们并由此激发了她的这个念头。她用十多年的时间设计出多款完全用废弃的安全套为原料制作的衣服、裙子和套装。

这种行为非常环保，不仅仅会提高人们对于艾滋病的重视，与此同时，由于这些衣服采用的是废弃的安全套，这些东西通常是会被扔掉或者焚烧掉的，但是一旦被做成了衣服，自然会对环保事业作出一分贡献。因为焚烧会产生大量有害气体，垃圾当然最终会被送去垃圾场，这些行为都是不环保的。但是多亏了防腐剂公司的合作和波蒂尼对于生活的热爱，她才能够将自己的才能以及对环保事业的支持表达的如此淋漓尽致。

背后的故事

目前为止波蒂尼为制作一套礼服所使用的安全套已经达到8万个。简直不可思议！这到底是一套什么样的礼服呢？实际上是一套婚纱。你能够想象新娘穿着这么一套婚纱么？同时波蒂尼还制作了200个雕塑、20幅画和160个小塑像。

不知是幸运还是不幸，在普通人看来，这些靓丽的服装并不适合穿戴，而仅仅应当摆在橱窗里供人欣赏。如果你有兴趣，可以去参观一个叫做“向艾滋宣战，以服装为誓”的艺术展。该展览将于2007年3月11日在美国加州大学洛杉矶分校的福勒博物馆举行。更多的信息请访问福勒博物馆主页。

后续活动

波蒂尼不会停止自己的服装设计生涯。他有三个计划。首先，她准备策划一个以男性为主题项目，名字叫做“中世纪的艺术——衣服只为护身”。他已经用安全套创作勒很多个男性半身雕塑。

他还要造一座房子，名字叫做“维纳斯之屋”，或许你已经猜到了，这座房子里面的人和家具都会以安全套为原料来制作，不过同时也会包含一些性教育的档案，这些档案包含性传播疾病的历史以及各种统计数据。他只是希望当人们在家里谈论性的时候，能够营造出一种更加放松而舒适的环境。

她的第三个愿望，就是希望有朝一日能够举办一个名为“时尚明星秀”的活动，在这个活动中她所设计的服装能够得到很多名人的认可，这些名人都会穿着她设计的衣服说“我很喜欢，你呢？”如果你是名人而且喜欢她的创意就告诉她。我敢肯定她一定会欣喜若狂的。

想知道更多关于波蒂尼的情况，请访问她的网站（http://www.adrianabertini.com.br/ ）

还是那句老话，当你保护我们的地球时别忘了你自己，毕竟你也是她的一份子。
