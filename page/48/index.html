不碍事的自行车锁

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

自行车锁可以保障自行车的安全，但是在乘骑自行车的时候，为了安全而购买的“重量级”车锁，收纳起来也着实烦人。所以，设计师便将车锁给藏匿在了自行车车身上。这款新型自行车的车把可以拆卸，配合车把内部的链条，可以将自行车锁拷在方便的地方；当需要骑车的时候，车锁便就回归原位，充当车把手，一点儿都不会碍事。

设计：Bach Nguyen

魔方充电器

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 电池

设计师根据电磁感应原理设计出这款魔方充电器。当您在玩魔方游戏，不停转动魔方的时候，魔方中心设备的磁感线就会被不断切割，从而产生电流，并被储存到内置电池当中。当手头的小电器缺乏电量支持的时候，您便可以将魔方拉伸，通过连接线，将魔方电池里所储存的电量转移出去。

廉价的太阳能涂层

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 电池

作为少数几种取之不尽、用之不竭的能源，太阳能是一种非常理想的环保能源。但是，由于太阳能板的造价过于高昂，因此在推广上存在一定的困难。假如能够像刷涂料一样将太阳能电池版改造成为一种涂料，那该有多好？

日前，一家位于美国硅谷的公司推出了一项全新的太阳能涂层技术，可以大大降低太阳能电池的造价。只要通过一种特殊的喷涂装置将可以吸收太阳能的纳米涂料直接喷涂到像铝箔一样的金属薄片上，就可以制成这种电池。

传统的太阳能电池版由于需要使用大量的硅，因此造价很高。而这种全新的技术完全不需要使用硅，因此可以将成本大大缩减，甚至比使用煤来发电更为廉价。以往的太阳能发电设备，每瓦的成本最低也要3美元，而如果使用这种全新的技术，每瓦电力所消耗的成本只有30美分。

如果能将这项技术大力推广，一定能够节约大量的能源。

登机牌机场导航针

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

为了可以容纳更多旅客，安排更多航班，国际机场的建筑规模一般都不小，除了机场工作人员，要想在首次抵达的时候，在纷繁复杂的指示牌中找到通往目的地的路，实在不容易。所以，便有设计师团队计划改造登机牌，让登机牌上的导航针帮助老人、小孩和一些语言不通的外国游客更加方便地找到登机口。登机牌上会附着一块超薄电池，给导航针电子纸和RFID（无限射频识别技术设备）提供能量支持。正是因为有了RFID技术的保障，导航针的指示准确度大家完全可以信任。

设计：Kang Eun-Kyung and Park Ji-Eun

农夫采摘辅助机械臂

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 都市农业

在日本，约有三分之二的农夫年龄超过65岁。由于年龄的增大，身体机能的衰退，在收取作物的时候，老农夫们往往力不从心。来自东京农业工业大学的研究员特别根据老人的身体状况，为他们推出了这款采摘辅助机械臂。它可以让使用者的力量增加62%。这款机械装备还配置了运动检测传感器、语音识别设备和8个电动装置来矫正使用者的姿势，缓解关节压力。经过15年的研究，这款机械臂将会在年底向市场推出。而充满激情的研究者计划继续升级该项设备，给它添加卡路里计算、心率检查等相应装置。

设计：Shigeki Toyama 【来源】

全能混合动力车

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

看到这张图片，相信大多数人都会被这款汽车的外观所吸引，其超流线的造型的确非常吸引眼球。不过它可不是徒有其表哦，其最具吸引力的还在于它的全能动力设计。

该产品与前面介绍过的一款“超酷环保汽车”一样，可以通过轻巧的车身和流线的造型来降低油耗从而达到省油的目的，每加仑汽油可行驶约150英里。该车既可燃烧汽油也可燃烧甲醇。由于整车采用碳纤维制成，使得其车体重量降至270千克。

所不同的是，这款汽车还提供有多种动力选择。太阳能电池版所吸收的太阳能能够供给其行驶15到25千米。有趣的是，阳光不够强烈时，它甚至可以像自行车一样通过内置的脚踏板来对电池进行充电。

该车采用3座设计，将会于2012年投产，预计花费为2万美元。

多功能电源转换器

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

你有没有过这样的经历，当你登上飞机的时候突然发现自己的手机没电了？如果旅程仅仅2个小时还好说，可是如果是国际航班呢？除非你找到新电池，否则，你肯定会抓狂的！你可能注意到很多航班的座位下面都有一个小电源插座，不过这并不是普通的插座，需要特殊的插头才能插进去。很少有人会带着这种特殊的插头到处跑，更不用说很多人根本就不知道有这种东西的存在了！如果我告诉你有这样一个小东西可以一端插在这种特殊的插头上，另一端可以随意插入普通的插头，你一定会对它很感兴趣吧？

这东西的体积似乎有点大，不过他的功能比它的体积还要强大。它可以将飞机上的电流转换成你所需要的任何制式的交流电。这样你就可以在飞机上为你的手机、MP3、便携DVD等设备充电了。为了更好的兼容，都采用了三头的插座。而且，这个小东西还有另外一个用处，就是可以用在汽车上，通过汽车点烟器就可以产生普通的民用电流。是不是很棒？

现在购买只需60美元，有了他你就再也不用为自己的随身设备没有地方充电而苦恼了！
