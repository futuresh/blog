真正的原叶茶包

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款货真价实的原叶茶包，其外包装材料便是采用纯天然的绿色茶树叶制成，省却了生产达标外包装纸的成本，也减少了因此带来的环境污染问题。不仅是为了节约、环保，设计师团队之所以采用如此特别的材料制作包装还有一个重要原因——他们都考虑到浓茶中单宁酸对人体的影响。当绿茶温度保持在60到70摄氏度期间时，茶饮料的饮用是安全的，并对人体有益。但是，若茶饮料温度达到80到90摄氏度，茶叶的浸泡时长超过2分钟，那么茶叶中的单宁酸便会被分泌出来，这时饮用，它会影响人体对铁、钙等有益物质的吸收。正是因此，用原生茶叶作为外包装，使用者便可以通过泡茶时茶叶颜色的变更，知晓茶饮料的浸泡时长和温度变化，泡出更健康的茶。茶饮料能够提神醒脑、促进消化，但是浓茶也会伤身，尤其是老年人更应当注意呀！

设计：hyung gyun jung, il seop so, han bi jung & jae young heo

会“呼吸”的花盆

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

在家里或办公室里种上几盆植物是一件既惬意又环保的事情，但是有些人由于工作忙碌却经常忘记给自己种的植物浇水、晒太阳。为了帮助人们更好的照料自己的植物，就有设计师设计出了这样一款会“呼吸”的花盆。

它采用了较为柔软的外壳，因此可以膨胀收缩，内部的独特结构可以在植物生命力旺盛时模拟动物的肺进行“呼吸”，因此，当你发现自己的花盆“停止呼吸”时，就要及时给它们浇水、晒太阳，以便使其重新焕发生机。

设计师：Jae-Han Song

太阳能蔬菜灯

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 都市农业

如果在夜间看到草地上有许多发光的蔬菜种在地里，可不要以为这是什么神奇的新品种，因为它们很可能就是这样一款太阳能蔬菜灯。

与许多太阳能路灯一样，这款太阳能蔬菜灯也是将白天的太阳能储存起来，到了晚上再供给路灯照明之用。只是这几款灯的外形被做成了蔬菜的样子。白天让“蔬菜”吸收阳光，晚上再用来照明。

无论是寓意还是用途，都非常环保，而且美观实用。

设计师：Arthur Xin

藤条自行车锁

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

帅气的自行车当然要配一把漂亮的锁，不过传统的自行车锁外形呆板，根本没有美感可言。最近在首尔举办的自行车设计比赛上，这把仿藤条设计的车锁格外引人注目。它一改传统车锁陈旧的外形，抛弃了沉重的大块锁头和容易生锈的大铁链，大胆采用了绿色硅胶包裹的柔软细钢丝锁链和拨号式密码锁。锁链上面还搭配绿叶作为点缀，无论是锁到柱子上还是缠绕在车身上，都充满了勃勃生机。

设计：sono mocci

用阳光和二氧化碳制造的环保生物燃料

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 都市农业

美国一家生物公司正在开发一项新科技，希望利用转基因单细胞植物将太阳能和二氧化碳转变成为乙醇和生物柴油等生物燃料。采用这种方式生产出来的环保燃料可以用于驱动汽车，大量降低汽油生产的制作成本和环保维护成本。公司称，该种植物仿佛太阳能板一般，可以安置在玻璃板下，并通过阳光照射和灌入玻璃板内的二氧化碳发生生物反应，分泌出燃料油脂。目前，这项新技术在实验室环境下已经能够成功获得生物燃料，在理想环境下，每英亩特殊植物可以在一年内分泌出25000加仑乙醇和15000加仑生物柴油，生产潜力不低。目前，该公司已在得克萨斯州设立工厂，以期将该项技术商业化，并提升特殊植物的产能，尽早将该项新技术推向市场。

开发：Joule Unlimited
