这次我们介绍的环保银行也被称作“反向自动贩卖机”，当人们把可循环利用的废物放进去之后，它便吐出相应的钱，作为对环保行为的奖励。这种机器目前除了可以接受易拉罐，塑料等可回收材料外，还接受一些透明的玻璃。目前它已经开始在澳洲使用，处在试验阶段。

这辆轮椅专为后腿不灵活或带有腿疾的狗狗设计，它通过护带捆绑到狗的腹部和臀部，从而帮助狗狗实现行走、起卧、上下楼梯甚至跳跃等运动。这辆轮椅使用多节动态支撑系统，创造性的弹性铰链关节是保证轮椅平稳工作的关键。虽然这款轮椅目前还处于原型机状态，但其发展前景令人十分看好。

背包自行车

发表于 2012-01-22 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

在能源价格日益上涨的今天，自行车作为一种非常环保的交通工具正在重新受到人们的重视。而为了鼓励人们更多地使用自行车，同时能够达到防盗、便携等问题，就有设计师设计出了这样一款背包自行车。

它独特的结构使得其可以变成一款背包背在身上。比起我们之前在猜猜看中为大家介绍的一款背包自行车而言，这款产品的轮胎更大，因此骑行速度更快。

车身由塑料制成，车身总重约5.5千克。

设计师：Chang Ting Jen

相关产品：“环形折叠自行车”“折叠车轮”

最环保的住宅

发表于 2012-01-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

近日，英国政府公布了他们的一款零排放（zero emission）住宅设计，在这里面，所有的动力均采集自环保的能源。

图中所指示的位置分别为：
1 风力捕捉器，用来提供夏天的通风
2 太阳能电池版，用来为房屋提供热水和电力
3 高隔热度墙体
4 生物发酵池

由于采用了高隔热度的墙体，这座住宅的热量散失率将会比传统住宅减少60%。通过太阳能和生物发酵来提供电力，同时辅以雨水收集器等节水装置来达到最大可能的节能效果。

这款住宅完全符合英国政府将于2016年实施的相关排污标准。

可叠放的衣架

发表于 2012-01-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

塑料衣架虽然可以节约成本，但是市面上的很多塑料衣架都不够结实，倘若是T恤等较为轻便的衣服尚可，如果换成比较厚重的大衣恐怕就会很容易变形。假如手边又没有比较结实的木质或金属衣架，那么最简单的办法就是将多个塑料衣架放在一起共同撑起一件衣服。

而这款新颖的衣架就将这种生活中常见的小问题融入到了衣架的设计中。每个衣架上都有一个凹槽，这样一方面可以节约成本，另外一方面，当需要多个衣架共用时就可以更好的将衣架叠放在一起了。

这款衣架与我们之前介绍的“方便裁减的浴巾”和“方盐”同为无印良品（Muji）设计大赛的获奖作品

环保可乐瓶

发表于 2012-01-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款环保可乐瓶是设计师特地为可口可乐公司设计的新包装。它采用百分百甘蔗原料制成，可以充分降解，不会污染环境。且由于新包装采用了方形设计，与原包装相比，体型更为修长，可以节约运输空间。而瓶身特别的纹路设计能让新包装的压缩比例增大，空瓶体积可以收缩至原瓶的66%，便于回收利用。

设计：Andrew Kim
