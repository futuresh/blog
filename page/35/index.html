桌子上的无烟小壁炉

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 环保&公益创意

看着桌上这只小巧玲珑无烟小壁炉，想象一下毛栗在跳跃的火焰中噼啪作响的香甜温暖，这份惬意不只可以在舒适的家里享受，还可以带到野外，为亲友聚会平添温暖热烈的气氛。

这只小壁炉的主要材料为厚重的意大利玻璃及铝，里面装饰着一几根仿圆木，一点儿沙及一只小灯，烧的是生物固体燃料，比木头的燃烧效率要高约40%，无烟，无灰，非常环保。也许不能象真的壁炉那样实用，但也能给你及家人带来特有的寒冬中的温暖。

紧凑的休闲桌椅

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

假如在朋友家里看到这样一个巨大的东西，你认为它会是用来做什么的呢？

其实，这是一套休闲桌椅。每个椅子上都设计有一个凹槽可以将桌子插到椅子中，而且每个椅子也都被设计成了直角，只要将它们拼在一起就可以很好的将桌子包裹起来，丝毫看不出它的庐山真面目。

不仅非常使用，而且相当的节约空间哦。

紧凑的单身厨具

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 都市农业

厨房里面的锅碗瓢盆虽然都不是大件的东西，但是真到了搬家的时候，要搞定这些小东西花费的力气可一点都不比大件的家具少。如果能够让这些东西更紧凑一点，甚至一只手就提走了，那该多好？

来试试这款紧凑的单身厨房吧。它采用了紧凑的模块化设计，锅碗瓢盆甚至包括菜板这样的东西都被一股脑的收纳在这样一个小盒子里面。需要的时候将它们展开，要搬家了都收纳到一起就可以了。非常适合不经常自己做饭又偶尔需要改善生活的单身一族使用。

另外，这款设计还整合了一个WiFi模块，以方便从互联网直接获取相关的饮食信息。

设计师：Alex Bradley

海绵牙刷

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

传统的牙刷都是用刷毛来清洁牙齿的，这样虽然可以将口腔清洁得更加干净，但是这种设计对于儿童和老人而言却并不实用。考虑到儿童和老人的牙龈更易受到刷毛的伤害，就有设计师用海绵替换掉了传统的刷毛。

这样就可以在刷牙的同时对牙龈起到一定的按摩作用，较之传统的刷毛更加柔和。而且针对儿童设计的海绵上还印有各式各样的卡通图案，以博取儿童的好感。

设计师：Fabio Dabori

电动滑板车租赁系统

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

与自行车类似，滑板车同样是一种非常环保的交通工具，不过传统的滑板车比较吃力，电动的则要好得多。于是澳大利亚新南威尔士大学的一名学生就设计了这样一款电动滑板车租赁系统。

首先，将现有的电线杆改造成充电和停车两用的租赁系统，用户可以直接从电线杆上取下滑板车，到达目的地后再将滑板车放到附近的电线杆上即可。当然，这个电线杆同样需要支持电动滑板车租赁系统。

设计师：Anton Grimes

俯卧自行车

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

与传统自行车不同，这款设计采用了俯卧的骑行方式。据说这样可以有效缓解下腰、胳膊以及臀部的肌肉紧张，而将支撑部位转移到盆骨和前臂上。同时，原先用来安装脚踏板的位置可以用来放置巨大的储物袋，以便存储更多的东西。而且，这种俯卧的方式还可以有效的减小风阻，从而更加省力的骑车。

为了达到最佳的便携效果，这款自行车还采用了折叠的设计，尤其是车轮，在不用的时候还能够分三段收缩起来。

设计师：Blair Hasty
