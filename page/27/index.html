新型淋浴喷头

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款新型淋浴喷头。淋浴的控制设备全部集中在淋浴水管上，从上到下，各个旋钮分别控制水温、淋雨时间、水压，管道上还有水量标识。在淋浴之前，您需要估计自己所用水量，根据水量标识，将控制器移至相应位置。淋浴器期间，控制器会匀速下滑，您可以根据它与下滑下限之间的距离来估计剩余水量，从而调整水压。当控制器滑到初始位置的时候，您需要将控制器重新提拉上去方能继续淋浴。因为这款淋浴喷头的控制器提拉上限为建议淋浴水量，所以，如果您需要重复提拉几次才能结束沐浴，就应该思考一下自己的用水模式，节约用水了。

设计：Sang-in Lee & Dae hyun Kim

感染提示创可贴

发表于 2012-03-06 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

处理小伤口使用创可贴十分方便，几乎每家都会存放一些以备不时只需。不过，使用创可贴不慎也容易引发感染。德国夫琅和费研究所的科研人员近日研制出了这种新型创可贴，其中带有的变色指示纸可以根据皮肤的PH值进行变色，并以此来及时提醒使用者伤口已经感染了。这种新型创可贴使用起来安全放心，给您更加贴心的保护。

尿液电池上市了！

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 电池

我们曾经介绍过一种利“用尿液发电”的技术，当时这项技术还只是处在测试阶段。但是现在，一款利用尿液来充电的电池已经正式在日本面市了。

该系列的产品涵盖了几乎所有的电池型号，而常用的五号电池则可以达到500毫安时的电量。电池即使放置不用，其中的电量也可以保持十年以上不泄露。而你要获取电量，唯一需要做的就是：向里面加入一点尿液（或等效的其他液体）。

而且由于不含油汞等剧毒物质，这种电池并不会因为随意丢弃而造成环境污染。

当然，该产品也有一定的不足，由于采用电解反应供电，所以其中的电极属于易耗品，因此只能支持约3～5次的反复充电。

概念折叠房车

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

对于喜欢到处旅行的人们来说，房车确实给他们带来了极大的便利。但是，驾驶着笨重的房车穿越狭窄的乡村街道也不容易。如今，由荷兰设计师推出的这款“富乐宝”折叠房车，根据传统旅行睡袋原理设计。折叠后，房车体积会缩小，便于携带和运输；而将其展开，休息空间与中型房车大小相当，可容纳一个普通家庭。此外，由于其符合空气动力学的结构设计，房车折叠起来省时省力。带着它去旅行，您一定会省心不少！

设计：Niels Caris

西方故事里面的女巫大都借助魔法扫把飞行穿梭，而这款摩托的设计也源于此。只要配合轮滑鞋，再借助带有动力装置的单轮摩托，使用者便可驾驶这款设计最为精简的摩托车穿梭街头。这款摩托时速可达40公里每小时，却只有24公斤重。它配有500瓦的马达，可以一次性行驶50公里。当然，为了使用起来更加趁手，设计师还特别为成年人、青年和小朋友定制了大小不同的摩托型号。
