花瓣弹性座椅

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

受到大自然鲜花绽放时优美形态的启发，葡萄牙设计师设计了这款“花瓣”形状的公共座椅。它的五个座位由高密度的聚丙烯和回收塑料制成，靠背则由橡树藤条编织而成。座椅底部还加装了从废旧汽车上的减震器，坐起来更加舒适。在钢筋水泥的城市中，这款身姿优美的座椅或许可以给您带去大自然的感受！

设计：Alexander Castro

快餐专用手指贴片

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

在前几天介绍“手型餐具”时，我们就曾经提到过，人手是最灵活的餐具。虽然这种手型餐具可以在一定程度上模仿人手的动作，可是毕竟还是不如直接用手来的灵活。那么为什么不试着直接用手呢？

这款新颖的手指贴片就是专门为直接用手就餐而设计的。它可以很好的适应手指的形状，与手指贴合在一起，原理类似于“一次性脚底贴片”。平时可以存放在一段弯管内，需要的时候只要将手指放入弯管的两端，即可将贴片贴在手指上。

对于品尝薯条等只需两根手指即可解决的快餐，非常实用。既可以直接用手抓取，又不必担心卫生问题。而且它采用可降解的环保材料制成，也不必担心会污染环境。

环保纸板机箱

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

除了必要的电子元件以外，这款颜色古旧的机箱可以说是当今世界上最为环保的机箱了！它采用可回收纸板做成机箱框架，并将主板、硬盘、各个数据接口等电子元件牢牢地固定在纸板内部，只要将各个接口与电脑显示器、键盘、鼠标相互连结，启动电源，电脑便会开始工作，其工作状态与普通机箱一无二致。不过，鉴于纸板并不良好的散热性能，即便纸板机箱已经预留了散热口，但若是在非空调房内使用，还是有些让人担心呢！

侧身轮胎

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

如果汽车转弯过快就有可能会翻车，但是摩托车在遇到这种情况时则可以通过倾斜车身的方式来改变重心，从而降低翻车的几率。可是汽车并非两轮设计，似乎不可能像摩托车一样侧过身来。但是有了我们今天为大家介绍的这款轮胎，汽车也可以侧身了。

这款轮胎将传统的轮胎分成了八个可以独立活动的部分，这样一来，转弯时，轮胎就可以在水平方向交错开来，向弯道内侧倾斜，从而实现类似于摩托车的侧身效果。

水滴磁性挂钩

碳纤维电动自行车

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一辆超越现代理念的自行车，车身由质量超轻的碳纤维制成。电力装置隐藏在车身内部，使用高导电性碳纤维取代电线作为传导，为车灯和在每个车轮两个轮毂之间的电动机提供能量。这款车由一个顺时针转动轴提供后轮扭力驱动整车前进，替代了传统的齿轮和链条装置，使它显得既简洁又时尚。

设计：Ralf Kittmann

融化的终结者摩托

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

见识过汽车版侉子，我们再来看看这款融化的终结者摩托。它改装自铃木摩托，看起来就像是融化了的机器人附着在摩托车上。除了怪异的外形、被改装的排气管和刹车装置之外，它还配有音响、蓝牙设备、触摸屏GPS导航系统等实用酷炫装备。

方便电池

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 电池

我们所使用的大多数电子设备都离不开电池作为驱动能量，但安放电池的电池盒往往因为需要，设计得十分紧凑，要想从中取出废旧电池，有时还真是麻烦。设计师便将常见的标准电池做了改装，让电池头尾变得扁平，而肚腩依旧鼓鼓囔囔，在保证电池供应足够电量的前提下又能方便我们取用电池。如此一来，只要按动电池头尾，我们就能让电池从电池盒中翘起来，轻松取出了。

设计：Young Suk Kim

长柄鞋托

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这个支架能够让你不用弯腰就可以轻松的换上鞋子。只要将下面的夹子对准鞋子，然后用力握住手柄，夹子就会把鞋夹住，然后轻松完成穿鞋的过程。

对于普通人而言，这种产品或许作用不大，但是对行动不便得老人或是伤员而言，这种个小东西可是能帮上大忙的。
