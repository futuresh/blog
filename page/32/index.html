墙上的毛皮动物

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

昂贵、华丽的皮毛大衣，为穿戴者带来的不仅仅是身体的温暖，更重要的是身份的象征与地位的显示。可是废旧毛大衣能做什么呢？一个女性街头艺术家组织将回收的废旧皮毛大衣当作创作素材，把它们按照各种动物的轮廓进行裁剪，然后粘贴在公共场所的墙壁上。远远看去，这些动物惟妙惟肖，逼真至极。这些艺术家们还正在探索立体表现手法：将动物模型包裹上皮毛，然后再根据所塑造动物的习性，摆出独特的造型。这些作品吸引了大批路人的目光，这种新的艺术形式，更重要的是想要唤起人们对野生动物的保护意识，以减少人们对动物的残杀。

吸尘器扫帚

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

吸尘器作为新一代的清理工具，在清理细微灰尘的时候，的确要比传统的扫帚好用得多，尤其是在对付地毯等比较容易附着灰尘的柔软表面时，优势就更为明显了。不过在对付纸团等大块的垃圾时，却并不那么好用。因此，将传统的清洁工具与吸尘器相结合就成为一种新兴的设计理念。我们专利之家之前也曾经为大家介绍过“吸尘簸箕”和“吸尘垃圾箱”。可是却唯独少了扫帚，所以今天就来为大家介绍这款新型的吸尘扫帚。

它的扫帚头是由很多柔软的吸尘管制成的，在打扫大块垃圾的同时也能够同时将细微的灰尘吸入其中。同时，配套的簸箕也采用了一体化的嵌入式设计，不用的时候可以直接将扫帚卡在簸箕的凹槽内，这样就可以更好的节约空间了。

这款吸尘扫帚与与“吸尘簸箕”解决的问题非常相似，二者相比，你认为哪一个更为实用呢？

相关产品：“吸尘鞋”

立式订书机

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

订书机虽然体积很小不会占用太多的空间，但是对于很多上班族而言，办公桌的空间可是寸土寸金，能节约一点就绝不浪费，所以也就有了今天的这款设计。它将传统订书机改造成了这种直立的设计。因此，可以占用尽量少的空间。

而且它的外形设计也非常美观，就像是一个夹子。使用方法也跟夹子一摸一样，只要把需要装订的纸张放在订书端，然后用力捏住远端即可完成装订。

设计师：Gabriel Nathan

环保擦伞器

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

雨伞是下雨天的必备雨具之一，但是大型商场和写字楼通常都不允许人们带着湿漉漉的雨伞进入，所以一般都会在门口提供套伞器，将雨伞用塑料袋包裹起来。既然超市购物都不提供免费塑料袋了，那么为了一把雨伞而浪费一个塑料袋显然也是不合理的。所以，就有一家日本公司开发出了这样一款擦伞器。

它的使用方法非常简单，只需要将雨伞放入到里面来回抽拉几下即可将雨伞彻底擦干。而擦下的水则会被收集到下面的一个小盘子里，大约每擦干3000把伞才需要清理一次盘子。这样不仅不需要使用塑料袋，可以避免白色污染，而且也无需任何的能源，因此非常环保。

可单手操作的摇杆水龙头

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

厨房里使用的水龙头普遍都可以调整角度，以方便清洗碗筷。可是要调整水温和流量则还是要移动一下手，通过阀门来操作。

这款全新的摇杆水龙头则可以让你一次性单手调整好位置、水温和流量。

它创造性的将阀门设计在摇杆的外端，这样，当你调整出水的位置时也可以同时利用阀门将水温和压力一并调节好。大大的简化了操作流程。而且，由于水龙头是被固定在墙上的，不用的时候还可以将摇杆完全贴近墙壁，节约空间。

环保营养花盆

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

不少都市人都喜欢在室内养些绿植，这样不仅能够装点家居，美化环境，还能改善室内空气质量。但是植物想要继续成长，小花盆往往不能满足根部的需要。如果您家里使用的是这款环保营养花盆，那么只要将花盆打碎，然后连同花盆将植物一起埋到更加宽敞的土里就好了，而由营养物质凝结而成的花盆还可以为植物的继续生长提供能源支持。

多用门票

发表于 2012-02-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款名为MulTicket的门票内部整合了多种功能，它内置有你的个人信息以及银行帐号，通过联网的设备，你可以直接使用这个设备进入任何一个景点，同时景点也会将自己的介绍和指示信息录入到这张门票里面。你可以一边走一边查看上面的介绍，而且如果你有朋友在同一景点内，他还可以通过指示帮你找到自己的朋友。

当然，这种门票要推广开来需要一整套联网的设备和数据库。不过通过这样的东西，起码可以有效的节约纸张的浪费。
