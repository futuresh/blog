花苞路灯

发表于 2012-01-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款以太阳能和风能为能源的概念路灯。灯头采用了花朵的造型，整个“花朵”可以在获得能源时自动控制开合。这款灯的灯柱上设有自动感应装置，当有人靠近时，它可以自动打开灯柱上的LDE灯方便近距离照明。此外，这款路灯还能够将获得的多余能量传送给城市中缺少电能的地方，更好地节约了能源。

设计：Philips Design

节约的洗手液脖箍

发表于 2012-01-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

用压力瓶来盛洗手液不仅使用方便，而且由于洗手液不直接与外面接触，所以还可以充分保证洗手液的清洁。不过由于压力瓶的瓶颈设计过长，很多人都会直接压到底部，然而实际上却并不需要使用这么多的洗手液，这也就会造成不小的浪费。

而这款由橡胶制成的脖箍就用最简单的办法解决了这个棘手的问题。只要将它套在压力瓶的瓶颈上即可达到节约洗手液的目的。

虽然思路简单，却可以解决不小的生活难题。

发电的背包

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们专利之家曾经介绍过像“风力充电器”“太阳能充电器”“手摇式充电器”甚至“自行车充电器”这样方便户外运动的便携充电设备。不过这些产品都是比较常规的，而且其中很多并不方便在野外行走的同时进行充电。

其实我们行走时消耗的能量除了一部分应用于运动和保持体温之外，有很大一部分都被浪费掉了。如果能够对此加以利用，对于野外生存而言，也会是一款非常有用的发明。

这款由美国密歇根理工大学。亚利桑那州立大学以及NanoSonic联合开发的新式发电背包就可以将行走时背包与身体接触而产生的压力转换为电力，将这些原本白白浪费掉的能量加以利用。该产品由一种名为polyvinylidene fluoride (PVDF)的新型压电材料制成。看起来就像是普通的尼龙布料一样。

这样所产生的电力足够给LED灯、MP3播放器以及手机等便携设备充电之用了。
