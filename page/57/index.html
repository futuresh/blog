植物环绕的环保椅

发表于 2012-01-29 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款爬满蔓藤的公共座椅，是设计师为环保社会而设计的。座椅上配备了细小的太阳能板，可以为夜间照明提供能源。此外，设计师还将雨水收集器和过滤装置安装在座椅上，方便帮助植物生长。相信这款座椅在放松路人身心的同时，更能唤起他们的环保意识。

马桶洗衣机

发表于 2012-01-29 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着人们环保意识的提高，各种各样的节水设备也层出不穷。由于冲厕所需要的水的清洁度相对而言较低，因此将洗澡、洗手的水二次利用拿来冲马桶就成为了一个很重要的设计方向，我们之前介绍的“浴室节水系统”就是这样一款对洗澡水进行再利用的设计。

除了洗澡、洗手之外，还有没有其他的可供再次利用的废水呢？对了，就是洗衣用水！无论是手洗还是机洗都会产生大量的废水，而这部分废水又非常适合冲洗厕所。因此，也就有了这样一款马桶洗衣机。

它将洗衣机设计在了马桶的水箱上方，既可以节约空间还能够提升滚筒洗衣机的高度，这样在操作时就不需要弯腰了。当然，它最重要的作用还在于对洗衣剩下的水进行重复的利用，达到节约资源的目的。

设计师：Sevin Coskun

饮料倾倒器

发表于 2012-01-29 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

倒饮料看起来是非常简单的事情，似乎不需要任何额外的工具，但是细心的商家却总是能够从中找到商机。这款饮料倾倒器就是这样一款看似有些“多余”的产品。

普通的饮料当然很容易倒入杯子中，但是两升的大瓶装饮料可就没有那么容易“一手掌握”了。尤其是对于小孩子和老人而言更是如此，一不小心还会将饮料洒出来。所以，也就有了这样一款饮料倾倒器。它可以将两升装的饮料牢牢地固定住，想要喝饮料的时候就可以像图中那样，抓住纤细的框架将饮料倾斜过来即可稳稳地倒入杯中。对于小孩和老人而言，非常实用。

售价19.95美元

分体式房车

发表于 2012-01-29 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

房车虽然可以为喜欢旅行的人提供一个四处移动的家，但是也还是存在一些不便。比如，当你到了一个新的地方想要轻装上阵游览一下周围的景色时，如果开着体积庞大的房车就会比较麻烦。而这款分体式房车就很好的解决了这个问题。

它将房车的车头设计成了一个可以独立行驶的小汽车。这样一来，既可以拖着后面的车厢四处旅行，又可以在需要的时候单独驾驶“车头”出行。

设计师：Christian Susana

全拆解自行车

发表于 2012-01-28 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

为了节约空间、方便移动和收纳，各式折叠自行车都各显神通，尽展变形奇能。这款新型自行车大小与一般越野车类似，特别的是，它的各个部件都能够充分拆解，就连轮子也能折叠起来。经过完整拆卸之后的自行车可以紧凑地放在一个方形包装盒内，节约了不少空间。

外置回收站

发表于 2012-01-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

Windows系统里的回收站是用来存放暂时删除的文件的，但是由于占用大量的硬盘空间，需要对回收站进行定时的清理，而且，很多人甚至都是直接用“shift+del”将文件彻底删除。这样虽然节约了硬盘空间，但是一旦发现有用信息包含在被删除的文件里时，再想恢复就很困难了。

这款移动硬盘被设计成了垃圾桶的样子，只要将它链接到电脑上，它就能够自动备份删除的文件，这样当你在要寻找误删除的文件时就会非常方便。尤其是对于公司用户，这样的产品能够避免很多不必要的麻烦。

该产品通过蓝牙与电脑相连，适用于PC机和MAC机。
