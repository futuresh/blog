透明的滑翔机

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款滑翔机最为引人注目的地方在于其驾驶舱采用了通体透明的设计。不仅如此，它的原材料也都非常环保：框架由可回收的亚麻生物复合材料制成，透明的机身则由聚碳酸酯制成，同样可以回收利用。

两翼由光电池板覆盖，利用太阳能来提供动力。为了达到更高的安全性，它还配有燃料引擎以备不时之需。

设计师：Roland Cernat

阶梯架子

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了节约空间，很多人都会在家里配上一款顶到天花板的架子。但是对于一般人而言，想要够到最上方的东西就不得不借助凳子的帮助。我们前面介绍的一款“带挂梯的书架”就利用配套的梯子来帮助人们拿取东西。但是如果是在家里的话，配上一个梯子未免有点小题大作了。还是凳子和椅子来的比较实在。

这款阶梯架子就将下面的三个储物格设计成了可拉伸式的，可以像抽屉一样随意抽拉。但是它们真正的用途在于帮助那些“海拔”不够的人轻松的够到上面的东西。

相关产品：“阶梯变形椅”

“可怜的鱼”节水洗手池

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

节约用水是每个人都应该做的，不过如何把口号变成实际行动还是需要大家提高节约意识的。这款名为“可怜的鱼”的洗手池，上方特别带有一个鱼缸，当您洗手时，鱼缸里面的水位就会随之下降，直到您关闭开关水位才会恢复。如果您用水过多，鱼缸里面的鱼就会面临缺水的危险，这种让您直面“残酷”的现实的方式，或许对某些人来说比粘贴“节约用水”的警示牌更加有效。不过您不用担心水龙头里面的水是否干净，鱼缸里面的水和水龙头里面的水是分隔开来的，您看到的只是视觉效果。

设计：Yan Lu

吸盘式洗浴包装

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款便捷的吸盘式洗浴包装由玉米淀粉提取的材料制成，十分环保。特别的强力吸盘设计更是使它可以方便吸附在浴室墙上。您可以给洗浴包里灌入自己喜欢的浴液和洗发水，并按照需要随意摆放在顺手的地方，不用在墙上钻孔安装置物架，一样可以方便地洗浴。

设计：Jung Hyun Jee

长脖三轮车

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款三轮车的造型有些类似于桌面吸尘器，可别看它古怪，这款设计刚刚荣获了Car Design News Contest 2008第二名。

它最大的特点在于采用了伸缩式的长脖设计，驾驶室设计有一个可伸缩的橡胶脖子，这样一来驾驶员就可以根据自己的需要随意调节驾驶室的高度，从而获取最为舒适的驾乘体验。

浴室节水设备赏

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

浴室往往是最容易浪费水的地方。为了能够充分利用水资源，更好地节约用水，许多设计师都在卫浴设施上推出自己的精心设计。这里，我们为大家选取了一些节水设备，别看它们貌不惊人，但是它们统统都是节水好手呢。

这款双筒洗衣机能节约大约50%的洗衣用水：第一遍的洗衣水被存储到机器底部的容器中，经过过滤处理后，可以再利用。此外，它还配有特殊的除臭系统，通过鼓入新鲜氧气杀死散发臭味的细菌，让您的衣物清新飘香。

设计：Max Germano

这款沐浴装置可以收集洗澡水，并通过内置的净化过滤系统，提供再次使用。这款产品还借助电子控制系统，使得水流能够循环流动，并保持一定的温度。

设计：Victor M. Aleman

这款漂亮的一体式浴室设备工作原理很简单：冲厕所时，水一半来源于清水，一半来自洗手盆使用过的废水。

设计：Jang Woo-Seok

这款水龙头与常见同类产品相比，有着很大的不同：它使水流采用溢出的方式，多余的水可以储存在水槽中，方便下次使用。

设计：Ernesto Messineo

这款看上去如同汽车脚踏板的水龙头开关，让您在无法腾出双手时也能为节约用水做出一份贡献。据称，这种脚踏式开关能节约大约50%的用水量。

这款节水马桶造型简单、结构紧凑，它利用洗手池的废水来冲洗马桶，能节约高达83%的用水。设计者估算，一个4口人的普通家庭使用这款产品，一年可以节约1.75万升的洁净水。

设计：Ariel Roja

这款利用洗澡水来冲洗马桶的座便器，拥有先进的带除菌功能的表面涂层和无孔纳米清洗系统，保证了马桶内外清洁干净。此外，它还使用了自动合盖技术，让您远离充满细菌的冲水过程。

设计：Alison Norcott

这是一款老少皆宜的一体化浴室。它内置可上下移动式脸盆，提供了多种应用方式。洗澡用的热水50%来源于自带热交换器。此外，具有混合空气技术的喷头耗水量仅为普通喷头的六分之一。

这套沐浴设施采用了混合空气式喷头，依靠快速加热功能和蒸汽功能，将热水转化为热蒸汽，可以有效地节约三分之二的水。如果单单就蒸汽功能而言，同一般淋浴设备相比，它只需要一般设备的十二分之一的水资源。

设计：Alemina Vranas

这款节水的洗手台和马桶一体设备，洗手台和马桶可以分开摆放，管道方便弯曲，能根据您的需要进行安装，十分灵活方便。

这款智能淋浴装置，它可以过电子显示器直观地告诉使用者即时水温、水压和洗浴时间，而且调节这些数据的扳手全部套在管道上，操作方便。另外，使用者可以预先设定洗浴时间，时间结束时，这款装置还能够给出用水统计，让您可以合理安排洗浴速度，防止浪费水源。

设计：Sang-in Lee & Dae Hyun Kim

舒适的新型自行车

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

设计师对自行车外形进行了修正，新成品中车辆的前后轮大小设计并不相同，如此便能将车辆重心前移。与此同时，座椅模样也得到更改，增加了靠背，载物框则被放置到座椅下方而不是车把上。整体看来，这款新型自行车不论是乘骑舒适度，还是载物容量都得到了一定程度的改观。

设计：Nick Foley

光电池便携冰箱

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 电池

这款小巧的冰箱是夏天外出野营的好帮手，它可以让饮料和水果长久保持冰凉和新鲜。冰箱的关键在于盖子上面的那两个光电池。如果冰箱里没有装满，你可以使用低耗模式，这样就只有一个光电池提供电源，而另一个则可以翻过来，用作照明灯。电池没电了也没关系，这款冰箱的特殊结构设计也能让里面的食物保温很长一段时间。

设计：Ruben Iglesias
