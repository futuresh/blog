弹力学步车

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

为了帮助孩子尽快学会走路，很多家长都会为自己的孩子配备一个学步车，但是市面上的学步车未必能够对孩子学习走路起到太大的帮助，这是因为学步车本身承担了大部分的重力，因此儿童腿部并没有得到有效的锻炼，也就无法达到预期的效果了。而这款新颖的弹力学步车就通过一个弹性的装置使得学步车的效果大大改进。

它在学步车的座位四周设置有一圈弹性装置，可以根据孩子的不同坐（站）姿给与相应的支撑，而且由于其弹性的设计，还能够使得幼儿以“跳跃”的方式通过学步车前进，就像是一款专门为幼儿设计的蹦床一样。相对于传统的产品而言，这样的设计更有利于锻炼幼儿的腿部肌肉，从而更快的学会走路。

设计师：Wilson Song

马力汽车

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

在蒸汽机出现之前，马车一直都是最为重要的交通工具，虽然效率相对低下，但绝对是一种无污染的清洁动力。那么在现在油气动力如此充裕的今天，是否可以有一种全新的方式对马这一原始的动力加以利用呢？

来自迪拜的一家名为Fleethorse的公司就设计了几部汽车，可以利用马做动力来驱动汽车前进。图中这部公交车就是其中之一，它在车厢中放置传送带，让马在传送带上奔跑产生动力从而供给车辆前进的动力。当然，车中还同时配备有备用的动力系统，以便在马力不足的情况下使用。

万能车钥匙

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款万能钥匙的头部有一个探测器，只要将该装置插入汽车的车锁内，它就能记录下车锁内的情况。尾部的USB接口能与电脑连接，钥匙内部的软件能确定车锁位置的键盘编码。只要利用这些编码，您就能复制一款一摸一样的新车钥匙了。而为了提高这款设备的安全系数，在将其连接电脑，运行软件的时候，电脑会与线上系统沟通，判断您是否已被授权使用该项设备。目前，这款万能钥匙只适用于福特汽车，不过研究公司已经计划将其推广到更多汽车上。

设计：Steve Randall 【官方网站】

自己发电的健身房

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 电池

你是否想过去健身房锻炼实际上是在浪费能源？因为我们在运动的同时，白白的把自身的能量释放出去了。如果能利用这部分能量，岂不是既经济又环保？

日前在香港就出现了这样一种健身房，内部的一部分照明和电视所需要的电力完全由在里面健身的人提供，当你在运动的同时，也为健身房的电力系统提供电量。

可是，目前来说，这种特殊的设备只是为了吸引顾客，因为他们通过这种方法每年节约的电费只有183元，而这样一套系统的售价却高达15,000圆。不过，如果这种系统能够推广开来，而且通过某些方法来节约成本，相信会有不错的前景。

变形摩托车

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款在国际电子消费展上亮相的变形摩托车，虽然不像电影《变形金刚》里博派或狂派那样可以变身为会说话的机器人，但它能从单轮车瞬间变成标准双轮摩托车。这辆折叠车采用了赛格威系统，当单轮行驶时能够保持良好的平衡性能，车身紧凑灵巧，很适合在拥挤的街道行驶。当到达开阔地带后，驾驶者只需按下变形按钮，便可完成变形过程。据称，这辆摩托车最高时速可达35英里每小时，预计将于明年上市。
