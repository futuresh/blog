防水太阳能电池

发表于 2010-05-22 | 更新于 2018-07-24 | 分类于 电池

普通的太阳能电池大都是放置在陆地上的，但是我们的陆地空间很多都被高楼大厦覆盖着，因此能够用于放置太阳能电池的地方并不多。既然如此，何不利用一下海洋表面的空间呢？

这款新颖的防水太阳能电池可以直接漂浮在水面上。通过这种方式就能大大拓展获取太阳能的区域。而为了防止风暴破坏太阳能电池，当有风暴发生时，这款产品还能潜入水下避风，等到风暴过后再浮出水面。

家庭里的发电机：风立方

发表于 2010-04-29 | 更新于 2018-07-24 | 分类于 电池

这是一个像风扇一样的设备，它由叶片、发电机、伸缩轴等构成，只要将它安装在墙上，然后拉出滑动槽触发电机开关，借助风力它便可运转，从而实现发电。每个单体的最大输出效率为每小时21.6千瓦，每天能为一个普通家庭节约6%左右的电力。

东芝燃料电池

发表于 2009-07-16 | 更新于 2018-07-24 | 分类于 电池

日前，东芝公司在日本发布了名为Dynario的小型甲醇燃料电池原件（Direct Methanol Fuel Cell，DMFC），它可以方便地移动，随身携带。只要将甲醇灌入设备，它就会自动发电。发出的电量可以冲给内置的锂电池，或是通过USB输出接口输出，其输出电压为5伏，电量为400毫安。机器本身甲醇容量为14毫升，大约能一次性充满两块手机电池。

便捷的指环充电器

发表于 2008-06-30 | 更新于 2018-07-24 | 分类于 电池

这是一家芬兰公司特别为iPhone、iPod系列苹果产品打造的别致充电设备。这款无线充电器由指环接收端和充电核心两部分组件组成，从而打造出一个无线充电环境。虽说该款充电器主要针对苹果产品设计，但是其他音乐播放器和电脑等设备也能通过兼容接口使用。

在充电的时候，充电核心可以在周边环境打造出一个电场，如此一来，只要指环接收端与需要充电的设备互联，那么充电两组件之间无需实体线材连接，即可给电子设备充电。只要指环接收端处于电场覆盖范围内，充电核心不论放在何处，都不会影响充电效果。此外，只要设备与指环接收端互联，那么一旦该设备进入电场范围，充电也会自动开始，而指环也会发出红光做出提示，当充电完毕，指环还会变色提醒。

设计公司希望，在进一步开发中，可以将指环直接融入到电子设备的内部构建当中去，如此便可减少附件的设置，使用也更加方便。目前看来，这款成品对公共场所，如机场、车站等地的公共服务设施建设很有裨益。只要服务机构设置好发射端，那么旅客们便可以轻松充电，而不用着急寻找空闲插座了。
