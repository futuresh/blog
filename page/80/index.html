环保地毯

发表于 2012-01-10 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款地毯采用大面积的蓝色为主色调，可以在夏日里为人们带来一丝清凉的感受。但是，它的中央却留下了一小块白色的区域。其实这正是这款地毯的点睛之笔，近距离观看就会发现原来这块白色的区域是一小块浮冰，上面还有一只北极熊。

作品表现的是全球变暖带来的北极冰川融化的严重后果。蓝色的海洋固然美丽，但是当整个地球都被这片蓝色淹没之后，人类恐怕也就再也无暇欣赏这种美丽了。

太阳能清便器

发表于 2012-01-10 | 更新于 2018-07-24 | 分类于 环保&公益创意

人们常说狗是人类最好的朋友，但是当狗狗在公共场所留下粪便时，你恐怕就会觉得它们不够友好了。不必着急，来看看这款太阳能清便器吧。

它完全由太阳能提供电力，因而非常清洁环保。另外，它的操作过程也非常简单。用户只要通过激光指示器即可控制它的行进方向。例如，将激光指示器指到目标狗粪上，清便器即可将狗粪清洁干净。而且这款产品还可以将粪便储存起来，将其分解并最终重新撒回到土壤中。

设计师:Joe Dougherty

购物袋套装

发表于 2012-01-10 | 更新于 2018-07-24 | 分类于 环保&公益创意

“限塑令”已经启用了四个多月了，很多人已经逐渐养成了去超市购物携带环保购物袋的习惯，但是很多需要单独称量的物品却仍然需要利用超市里面的手撕袋，这种行为同样会对环境产生破坏。当然，鱼虾肉类等鲜活食品用塑料袋可以起到防漏的作用，但是水果、蔬菜却不见得一定要用塑料袋，可是随身带着那么多环保购物袋的确不够方便，而且也无法进行有效的分类，那就不妨来试试这款购物袋套装吧。

它由好几个方便购物的小网兜和一个可以翻出来成为大手提袋的小腰包组成。平时网兜被放在腰包里面，选好了蔬菜、水果之后将他们放到网兜里，然后拿取称量，最后结账的时候再一起放到大的手提袋里面即可。

用凸透镜聚光的太阳能发电系统

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 电池

相信很多人小时候都玩过类似的游戏：用放大镜聚焦太阳光将火柴点燃。那么当我们感慨太阳能效率低下的时候，是不是可以借鉴一下这种方法呢？IBM公司就做出了这样的尝试。

IBM公司利用凸透镜进行聚光然后将其投射到太阳能电池板上，借此增大太阳光的强度，从而增加太阳能电池的发电量。通过这种方法，成功的将230瓦的太阳能转化成了70瓦的电力，转化效率约为传统太阳能系统的5倍。

不少人或许会担心过高的热量会将太阳能电池融化。不必担心，IBM在这套系统中整合了温控系统。通过在普通太阳能电池的PV板上加入了液态金属制冷系统。当温度升高时就能够迅速的将高达1600度的高温降低到85度，从而避免烧坏太阳能电池。IBM计划将这项技术出售给太阳能企业，帮助他们提升产品效率。

滚筒汽车

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

未来的汽车究竟会是什么样子？每个人或许都有自己独特的看法，我们今天为大家介绍的就是一个非常有趣的汽车设计。

它的整体被设计成了一个横躺着的巨桶，车轮位列两侧，直径与驾驶室相仿。驾驶室则采用了单排双座的设计。两个车轮分别由两个独立的马达驱动，采用燃料电池驱动。
