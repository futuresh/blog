智能垃圾桶

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

费城政府正准备在近期内大力推广这种智能公共垃圾桶，有了它们的帮助，政府可以在未来的十年中节约1290万美元的开支。这款智能垃圾桶可以自行对肚子里的垃圾进行压缩，并能在垃圾装满了的时候发出讯号通知清洁部门派人清理。垃圾桶工作所需的能量由顶端的太阳能板提供。普通垃圾桶平均每星期需要清理19次，由于这款智能垃圾桶可以压缩垃圾，比起其他同行，它能多承装4到8倍的垃圾，这意味着它每星期只需要清理5次。假如原来需要33名清洁工清理市区垃圾筒，那么，在更换了智能设备以后，8名清洁工就能满足清洁需要了。政府的确能节省一大笔钱。

澳洲风能汽车旅行

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

人类利用风能旅行的历史已经很长，但大多数时候却局限于海上。两名致力于风能开拓的德国达人便将风能利用转嫁到了汽车旅行上，利用这一绿色能源在澳洲土地上进行了一次环保旅行。目前，他们已经驾驶着这款风能汽车行走了5000公里路程。这款轻量级汽车的直接驱动能源主要来源于蓄电池和风筝，而蓄电池的电能也是停车休整时，通过附加设备收集风能转换而来的。

设计：Dirk Gion & Stefan Simmerer [官方网站]

婴儿车专用刹车器

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

电影里经常会出现这样的惊现镜头，婴儿车因为突发事件从母亲身旁脱离，然后……

恐怕不仅是电影，现实生活中同样会出现这种意外。而为了避免事故发生。就有人专门设计了这样一款婴儿车专用刹车器。该刹车器由两部分组成，刹车器本身固定在婴儿车的轮子上，遥控器则由父母携带。一旦儿童车与遥控器的距离超过2米，刹车就会启动。防止儿童车滑走，保护宝宝的安全。

除此之外，这款刹车器还可以记录行车里程等数据。

设计师：James William Wansey

纳米技术植物照明灯

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 都市农业

为了减少原始用料的浪费和环境污染，科学家们推出了一种新型照明技术，可以不必另行铺设电源线路和架设照明灯具，而是让街道两旁的树木为夜晚的街面提供光源。台湾的苏博士称，给树木注射的金纳米粒子可以诱导植物叶子发出红光；比起目前较为经济的LED照明来说，这种植物发光过程不会散发出像荧光粉之类的有毒化学物质。如能将该技术推广，那么街道照明不仅可以节约能源，还能增加植物吸收二氧化碳的机会，有助于改善城市居住环境。

技术阐述：Dr Yen-Hsun Su（台湾国立成功大学）

带浴缸的厨房

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 环保&公益创意

在厨房里面洗澡听起来有些不可思议，更不用说在厨房里洗盆浴了。不过既然房价这么高、住房如此紧张，为了节约空间又有什么是不可能的呢？

这款融合了一个浴缸的厨房就是这样一款看似搞笑，却多少也透露出一丝无奈的设计。它设计有一个超大号的厨房水槽，平时可以用一个额外的模板将多余的部分封盖起来就像普通的水槽一样。不过当你需要放松一下身心的时候，就可以将上面的这层模板拿掉，躺在里面舒舒服服的泡个澡了。

设计师：Philippe Malouin

数字彩纸装饰球

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 环保&公益创意

聚会的时候少不了用彩纸装点会场，重要人物出场的时候往往会从高空喷洒彩带，激起聚会高潮。不过这样可有些浪费资源，还增加了清洁难度，要想获得同样的效果又节约资金，可以考虑使用这款电子彩纸装饰球。整个装饰球相当于一款晶莹剔透的球形屏幕，通过特殊的麦克风的控制，可以在它表面模拟出彩纸喷洒的景象，同时还能显示字母。

螺旋缠绕的摩天大楼

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着电脑技术的发展，越来越多外形奇特的建筑构想可以变成现实展现在大家面前。这是一座高1300英尺，由位于街道两侧的两个不同建筑单体，通过螺旋向上的缠绕方式最终扭曲在一起的摩天大楼。建筑的外立面为环保性的仿制材料，可以根据太阳光照射方向和风向智能调整角度。外立面下层安装有纤维镜片，白天时这些镜片可以将太阳光反射进室内，提供天然照明；夜间，则由太阳能电池供电，照亮楼内空间。这座高楼集合了居住、购物、图书馆等多种建筑功能，将会成为曼哈顿市的地标性建筑。

设计：IwamotoScott Architecture

全球最亮的LED灯

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 环保&公益创意

作为一种新兴的照明设备，LED灯以其节能高亮的特性被广泛应用于各个领域。而日前来自韩国的研究人员将这种照明用具的亮度提高到了一个全新的水平。

这款最新研制的LED灯的单个灯泡亮度达到400流明，是传统LED灯亮度的四倍，而其耗电功率却仅有8瓦。在600毫安的电流下，其平均亮度达到350流明，最高亮度更是达到了420流明。

如此高效的节能产品一定能够为全球范围内的节能环保事业贡献一份力量。
