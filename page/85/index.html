老人专用电脑

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

相对于DOS这样的字符串界面而言，Windows已经非常平易近人，但是对于很多老年人来说，这种图形界面还是过于复杂。于是就有人设计出了这样一款更加简单易用的电脑。

这款产品将电脑的几个主要的功能分门别类储存在几个“模块”里面，每个模块都对应着电脑上面的插槽，用户可以通过模块来启动特定的功能，并利用触摸屏直接对数据进行拖拽。有了这样的产品，老年人也能够更好的享受高科技带来的便利。

设计师：Sylvia Spencer

便捷洗涤液瓶

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

不论是从环保的角度，还是节约的角度考虑，使用同品牌的袋装洗涤剂重新装填已经用完的洗涤剂瓶要更加合算。为了让这一装填过程更加方便、快捷，设计师便在这款新瓶上设计了橡胶漏斗，闲时它可以附着在瓶身上，当需要装填补充液的时候，只需将其翻转，一个正好卡合在瓶口的漏斗便能成型。

便携式空气加湿器

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

很多室内都比较干燥，可是普通的空气加湿器又不方便随身携带。别着急，有了这款空气加湿器，你就可以随时随地享受湿润的空气了。

它通过高涵水材料吸附大量的水，当你需要加湿空气的时候，只需要把它打开，里面的湿气就会迅速扩散开来。这款产品还内置有过滤装置，可以有效吸附空气中的灰尘和细菌，达到过滤空气的目的。

当使用过一段时间后，要清洗里面的灰尘，只需要用清水浸泡半天即可。最重要的是，这款产品非常环保，整个使用过程不需要任何额外的电力供应。不过由于体积所限，该产品只适合小范围的加湿。

平衡拐杖

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

对于许多老年人来说，拐杖是外出散步时必需的随身物品。不过如果不小心没有拿稳让拐杖倒在地上，捡拐杖对于活动不方便的老人来说是十分痛苦的。这款自平衡拐杖巧妙地应用了不倒翁原理，在拐杖的半球形底部内安装了配重，在无人使用时它也能保持直立状态。不仅方便使用者腾出双手做些其他事情，也省去了弯腰捡拐杖的麻烦。

节约瓶

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

平时使用沐浴露、洗手液、洗发水等清洁液的时候，常常因为吸管的原因无法取用剩下的液体，不得不把它们浪费掉。尽管有时我们会把瓶盖拧开，通过敲击瓶底，或者向瓶内灌水来取用最后的残余，但是这样也十分麻烦。这款节约瓶则可以充分利用清洁液，它在瓶底留有一个凹槽，吸管则深入凹槽底部。如此一来，所剩无几的液体也能在这个凹槽里汇集起来，得到充分利用。而瓶身的透明设计也方便查看剩余数量。

设计：Seonkeun Park & Jinsun Park
