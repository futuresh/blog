世界上最小的燃料电池

发表于 2012-02-11 | 更新于 2018-07-24 | 分类于 电池

来自伊利诺伊大学的研究者们设计了这款世界上最小的燃料电池，它的尺寸为3 mm x 3 mm x 1 mm。电池的外形小巧精美，所用的材料也很简单：水，金属氢化物，几个电极，还有一片薄膜。

这块电池能产生0.7伏的电压和0.1毫安的电流。别看外形小，它连续使用的时间能达到30个小时呢。或许你会觉得0.7伏和0.1毫安实在太小，远不能满足任何一种电子产品的需求。给点耐心吧，这种迷你电池才刚刚起步。其实对移动电子产品如智能手机等来说，迷你电池技术不啻为一大利好消息。这种电池是利用表面张力而非重力来控制水的流动，因此无论怎么动，它都能正常工作。

背包转向灯

发表于 2012-02-11 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

由于自行车没有像汽车一样的转向灯，在转弯时骑车人通常会打手势告知后面的车辆和行人。不过手势毕竟不够明显，有时候还是容易发生意外。为了更多人的安全，设计师设计了这款带有转向灯的背包。整套系统由位于车把处的无线操控器和带有LED灯的背包组成。当您想要转弯时，只需按动操控器上的按钮，背包后醒目的LED灯便会显示即将转向的方向。此外，当您刹车或突遇紧急情况时，LED还会显示红色灯以提醒后面车辆和行人。这项设计获得了2010年的红点奖。

设计： Lee Myung Su

滑动曲柄传动装置

发表于 2012-02-11 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着技术的不断进步，自行车正朝着无链条时代发展。这辆新型自行车以纤细的金属钢丝代替链条，并采用曲柄机构替换了脚蹬处的飞轮。它的工作原理很简单：当踩踏脚蹬时，与脚蹬一体的心形曲柄会带动肾状偏心轮转动，偏心轮凹槽内的滑块便会来回运动，滑块通过钢丝与后轮相连，以此带动后轮转动。此外，您可以通过变换脚蹬处的心形构件大小来调整车子性能，使用控制杆改变滑块高低位置，以此控制行车速度。据称，这种设计能很大程度上改善自行车的灵活性和机动性。

W.O.W——来自本田的宠物汽车

发表于 2012-02-09 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

与普通汽车相比，这款来自本田的W.O.W小型MPV的亮点并不在于传统的汽车配置上，而在于它是一款实实在在的为喜欢养狗的人准备的汽车。

首先，该车的两边的侧门都采用了双面推拉门，后门则是对开门的设计，可以提供更大的进出空间，不仅是人，对于一些大型犬也非常方便。

其次，内饰与其他车辆的最大不同在于，采用了全木地板的设计，不仅能够让狗狗感觉像是呆在家里，而且相对于织物而言，不容易吸附狗毛，方便清理。

另外，在仪表盘上，副驾驶座的一侧，还配有一个专门的狗窝，让你的狗狗可以在里面舒舒服服的打个盹，而不会骚扰到主人。

最为抢眼的一项功能是：该车配备有特殊的“犬脑信号接收器”，可以通过阅读狗狗的脑电波并向汽车发送相关指令，从而实现对汽车的行进方向的控制。不过，该项技术目前还不成熟还有待进一步完善。
