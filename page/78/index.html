自动驾驶之机器人公路火车技术

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

为了让自助出行更加便捷，不少公司都在开发自动驾驶技术，而谷歌公司则是公认的技术先锋。现在，除了自动驾驶，另一种便捷的概念驾驶技术也进入了我们的视野，那就是“机器人公路火车”技术。该技术若能实现，那么一辆带有导航功能，并由专业司机驾驶的“火车头”引导汽车就可以借助通讯沟通，带动自助加入“公路火车”行列的汽车向前行驶。除了“火车头”汽车司机，其他的“车厢”汽车司机只要将驾驶系统与导航汽车互联，就可以放心地让汽车向前行驶了。当然，在到达目的地，或是需要切换路径的时候，“车厢”司机可以自行切断与导航汽车的联系，将车辆开出队伍，十分方便。根据技术设计人员描述，一辆“火车头”汽车可以拖挂、引导6到8辆“车厢”汽车，而因为车队是由专业司机领导驾驶的，那么道路安全指数想必可以提高不少呢！

小巧的概念车

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

提到概念车我们通常都会想到炫酷的外表，豪华的配置。不过我们今天要为大家介绍的这款概念车虽算不得奢华，但也称得上精致。

这款由Zaha Hadid设计的Z.Car概念车只有4米长，算是小个子了，不过内室和座椅设计非常符合人体工学原理，驾驶非常舒适。而且小巧的身躯非常适合在城市中穿梭，要找个停车位也很简单。车身采用全静音设计，燃料电池的使用更使得这款汽车非常的环保。前景窗视野很开阔，后视摄像头则保证了倒车的安全。

像这这样设计前卫、节能环保、配置齐全的汽车将会使今后概念车设计的一大方向。

节能插头
