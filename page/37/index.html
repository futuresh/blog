新型自行车链条

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款全新的自行车链条。与传统产品不同，它并不是一节一节拼接起来的，而是类似于一条履带，齿轮也由原先的凸起改为了下凹的设计，而“履带”内侧的凸起则可以与腰带的下凹相互咬合，达到传动的目的。

由于摒弃了拼接式的设计，因此这款链条的耐用性更强，也更容易清洁，噪音也要小很多。

当然，这款产品的价格也很贵，两款已经发布的自行车的价格分别为930美元和990美元。

带留言机的开关面板

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

留言机是一种很方便的设备，你可以将留言以声音的形式录下来，以便更为直观的将消息通知其他人。不过对于那些需要特别提醒的事情，例如“千万不要忘记带钥匙”“记得关灯”之类的事情，用普通的留言机似乎就不足以解决问题了，毕竟对于一个经常会忘记带钥匙或者关灯的人而言，忘记听一下留言机上的留言的可能性就更大了。所以，我们需要一些更为特别的办法，“强迫”人们听取留言。

这款新颖的开关面板就通过内置的留言机达到强迫人们听取留言的目的。它内置有留言机，可以录制时长为20秒的留言。最为特别的地方在于，它将留言的播放与电灯的开关整合在了一起。每当你打开或者关闭开关的时候，它就能够自动播放留言。例如，在厕所的电灯上装上这样一个面板，每当有人打开电灯时，都会自动播放“请随手关灯”的语音提示，这样就可以提醒人们随手关灯节约能源了。当然，你也可以根据需要在不同的开关上设置相应的留言。用这种方式来强迫人们听取留言，应该会收到不错的效果。

潜水汽车

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

水陆两栖汽车已经不是什么新鲜产品，可是能够潜水的汽车是不是能够引起你更多的注意呢？

Rinspeed公司计划在今年的三月份举办的日内瓦车展上展出这款名为sQuba的全球第一款潜水汽车。这款车平时可以像普通汽车一样在公路上行驶，必要的时候可以像潜水艇一样潜入10米深的水下航行。

当以潜水模式行驶时，车身的动力由车尾部的两个涡轮和前方的两个喷气马达提供。能够驾驶这样一款汽车潜水，一定非常酷。

圈灯

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

自行车虽然非常环保，也能够锻炼身体，但是由于很多地方已经取消了自行车道，因此跟汽车抢道非常危险。而这款产品则为自行车驾驶者提供了一种额外的安全保障。

这款激光灯可以通过自行车的动力发光，并在自行车四周形成一个激光圈。用来警示周围的司机。平时，这个圈显示为绿色，而当有其他车辆闯入该圈时，就会自动变红，从而提示骑车人注意安全。

设计师：Leonardo Manavella

遥控代步鞋套

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

现代都市的交通拥堵现象日益严重，面对出行问题，每个人都会遇到一些略显尴尬的选择。打个比方，从居住地到目的地有2公里远，步行太累，而开车又有些浪费；选择公共汽车，走到车站的路程也不见得短。这个时候要是有一个灵巧、轻便的代步工具就好了。那么这款遥控代步鞋套肯定适合你，它配有发动机和变速箱，通过手上的遥控器控制刹车和启动，为保证使用者的安全，还贴心地在鞋套后侧设置了尾灯。它可以很方便地从滑行状态转变为步行状态，不影响使用者的正常行走。时速达每小时10英里，适合工作地点离家不远的上班族使用。

新型后座充气安全带

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意
