水陆两栖跑车

发表于 2011-11-01 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们前不久才刚刚介绍过一款来自Rinspeed的 “潜水汽车”。这款汽车还没有正式亮相， Rinspeed公司又推出了一款名为的Splash水陆两栖跑车。

其实水陆两栖的交通工具并不少见，我们在“八大另类交通工具”里面也有所涉猎。不过这款跑车的独到之处在于它的两个独特的滑水板设计。当需要在水上高速行驶的时候只要打开滑水板即可。

它的路上最高行驶速度为125英里/小时（约合201公里/小时）,0到100公里加速时间为5.9秒。而水上的行驶速度最快也能达到45节（约合83公里/小时）。足以与一般的快艇相媲美了。

丰田防瞌睡系统

发表于 2011-10-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

疲劳驾驶会带来非常严重的后果，因此采取必要的手段来提示疲劳驾驶的驾驶员朋友及时休息就显得非常必要。我们曾经介绍过一款可以根据驾驶员头部倾斜角度来判断其是否在打瞌睡的“瞌睡报警器”，不过那款产品是完全根据机械原理来设计的，难免有失精准。

而我们今天为大家介绍的这款来自日本丰田公司的防瞌睡系统则是通过安放在仪表盘上的摄像头来对上下眼皮之间距离进行监测从而判断驾驶员是否处于瞌睡的状态。一旦发现驾驶员出现瞌睡的现象就会发出警报提醒其注意从而避免事故的发生。

这套系统将被陆续安装在丰田公司旗下的汽车上。

可爱的数字存钱罐

发表于 2011-09-30 | 更新于 2018-07-24 | 分类于 环保&公益创意

几位大学生使用iPhone、木板、自己编制的代码系统制作了这款刷卡式数字存钱罐，它如同电子宠物一般需要主人及时的“呵护”与“喂养”，也就是要及时向其中存钱储蓄罐才会变得高兴起来。通过巧妙地程序设定与软件配合，存钱罐的显示屏上会根据自身或环境的变化而表现不同的表情：当您在其周围走动时，它会“含情脉脉”地注视着您，期待它的“食物”；而当您使用磁卡“喂养”它之后，它还会对您流露出浓浓的爱意。它还会在“吃饭”后打瞌睡直到下次吃饭时间。设计者称，这款产品旨在培养使用者的节约意识，即便是小钱也能积攒成巨额财富。

设计：Wang Chao, Maggie Kuo and Jordi Parra

水压升降椅

发表于 2011-09-07 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

我们之前曾经为大家介绍过一款方便老年人洗澡的“浴缸升降椅”，其实还有一些类似的帮助病人和老人洗澡的辅助性产品，但是大多数的产品都是需要通过电力驱动的。而我们今天为大家介绍的这款升降椅却可以完全依靠水龙头的水压来提供升力，而无需任何额外的电力驱动。

带滑翔伞的三轮车

发表于 2011-09-07 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款构思非常大胆的概念设计。与大多数概念车一样，它的外观非常炫酷。除了表面功夫之外，这款概念车还在每个轮子上都配备了一个电动马达，从而提供充足的动力。不过，这款设计最为奇特的地方还在于它的顶棚。其顶棚可以拆卸下来，作用与“滑翔伞帆船”中的滑翔伞类似，可以借助风力来牵引汽车前进。

设计师：Tsun-Ho Wang, Min-Gyu Jung & Sung-Je Do

蛋壳读书台

发表于 2011-09-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着城市人均居住空间的缩减，不少家具都采用了集约型设计，将各个家用器皿部件都巧妙地搭配在一起，节约居住空间，却又不影响日常生活的需要。这款蛋壳读书台便是一款经典之作。从表面上看，这款读书台充当的是书架的角色，只是它还藏有机关。将蛋壳打开，一张舒适的带脚垫的阅读沙发便呈现眼前，而破开的蛋壳边缘还有一圈LED灯，辅助照明，提供阅读所需光线，设计十分贴心。

设计：Ufulie
