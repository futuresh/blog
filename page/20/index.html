翻盖电脑桌

发表于 2012-03-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

传统的电脑桌大都是基于体积庞大的CRT显示器设计的。可是随着液晶显示器普及率的逐渐提高，这种原始的设计也越来越无法满足人们的需求。如何设计出更能充分利用液晶显示器纤细身材的电脑桌呢？来看看这款产品吧。

它的思路非常新颖，在电脑桌上设计了一个可以翻转的木板，而液晶显示器就被固定在这个木板上，平时可以将其扣上，将显示器隐藏在里面，就像是一个普通的办公桌。而需要使用电脑的时候则只需要将木板翻过来露出显示器，然后从桌子下面抽出键盘即可使用。

不仅非常简洁大方，而且充分的节约了空间。

动感三轮车

发表于 2012-03-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款名为Carve One的三轮车已经多次亮相于各大车展。它最特别的地方在于可以像赛车一样倾斜过来，这样一来就可以避免传统三轮车因为转弯过猛而导致的重心不稳的问题。

另外，根据亲身驾驶过该车的人的描述，由于具有这种独特的倾斜功能，这款车的驾驶起来的感觉就像是在坐过山车。

车身长11.2英尺，款4.3英尺，高为4.6英尺。最高时速115英里

新型概念对讲机

冰川橡皮

发表于 2012-03-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

全球变暖的问题日益严峻，而如何提高人们的环保意识，就成为减缓全球变暖趋势的重要因素。这款设计就巧妙的将冰川整合到了橡皮中去。

把橡皮做成冰川一样的形状，使用时，“冰川”就会逐渐变小、变黑，以此来提醒人们注意保护环境，减少碳排放。

设计师：Skeet Wang

相关产品：“消失在杯子上的冰川”
