空心轮摩托车

发表于 2011-11-22 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款摩托车的设计非常古怪，整个车身的重心非常低，车前轮距离驾驶者的距离也非常远。当然，最为奇特的还在于它放弃了传统的辐条，而采用空心的车轮设计。据称，这样的设计可以减小簧下重量和结构应力，提升制动倍率以及转向的准确性，同时减少震动并降低重心。

环保打印机

发表于 2011-11-21 | 更新于 2018-07-24 | 分类于 环保&公益创意

环保是当前最为突出的设计主题之一，各类IT产品也不例外。这款环保打印机就是一款非常有趣的设计。

首先，它并没有采用传统墨盒，而是利用咖啡渣或茶叶渣来制作墨水，只要将这些残渣放到配套的“墨盒”中即可。而为了达到省电的目的，打印机的喷头并没有利用电力来驱动，需要手动将其左右摇晃，即可将设定的图像和文字打印到纸上。

设计师：Jeon Hwan Ju

过山车果盘

发表于 2011-11-15 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款果盘的设计灵感来自于过山车的轨道，螺旋交错的线条勾勒出了这样一款精美的产品。虽然它最多可以容纳9个橙子，但是摆满水果之后的果盘，似乎就无法体现出其优美的线条了。所以，即使是不作为果盘来使用，这款产品仍然可以起到非常棒的装饰作用。

新型麻袋桌椅

发表于 2011-11-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

这套由荷兰设计师设计而成的软桌椅从外表看上去，颇似被重新缝合的麻袋，感觉有些简陋、粗糙，但是内部填充物却又让其表面蓬松，显得很是柔软。这套家具外层覆盖物为无纺布，很多环保购物袋都是采用这种材料制成的。而支撑的框架则是使用可100%回收使用的碳纤维树脂材料制成，质量轻便，结实耐用。

设计：Chris Kabel

反光自行车服

小涡轮有大功率

发表于 2011-11-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

通常而言，风力涡轮的体积越大发电量就越大，而小型的涡轮所产生的电量大多只能为便携设备充电或者作为装饰之用。不过我们今天为大家介绍的这款小型的涡轮却能够以数量取胜。

这种涡轮的直径为18英寸，每组由13个组成，涡轮的叶片由碳纤维制成，在保证强度的前提下最大限度的降低了涡轮的重量。在20英里/小时的风速下，其发电功率就能达到200瓦，风力越强功率也就越强。图中就是它的设计者利用一组涡轮点亮一个50瓦的汽车前灯。

在大的自然灾害发生时，这种小巧高效的产品将会派上很大的用场。

宝马智能导航系统

发表于 2011-11-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

导航系统的确能够给我们的出行提供很多便利，但是现在的导航系统大都需要很多人工参与。而宝马公司现在就在设计一款更为智能的导航系统。

这款产品不仅能够像传统的导航系统一样为你指示行车路线和当前的路况，而且还能自动学习你平日的行车路线，从而在你进入驾驶室后自动为你提供效率最高的行车路线，同时将座椅角度等参数调节到你习惯的位置。它可以通过用户手机中的蓝牙来识别当前驾驶员。

有了这样的系统，你的汽车就更像是一个老朋友了。

垂直太阳能充电器

发表于 2011-11-05 | 更新于 2018-07-24 | 分类于 电池

这款电池充电器的充电能源来自于身后支架上附有的太阳能板，通过吸收太阳能光，它可以为灌入充电口的充电电池补充电量。当电量补充完毕，从上投入的电池就会落入下部出口，你只要像在自动售卖机购买饮料时那样，将出口处的挡板按下即可取出电池。

设计：Tracy Subisak, Donn Koh & Herlinda
