开元未来创始人罗轶赢得GE绿色挑战

发表于 2012-08-11 | 更新于 2018-07-24 | 分类于 荣誉

通用电气(GE)董事长兼首席执行官杰夫·伊梅尔特日前在北京向开元未来创始人罗轶先生颁发了GE绿色挑战奖杯。
罗轶先生发明的摩天轮农场，采用温室技术和无土种植，借助风力发电、太阳能来提供能源。为高人口密度城市和未来城市的提供了一种有益的解决思路。
GE公司于2005年启动了“绿色创想”战略，致力于研发清洁科技，帮助客户解决环境挑战。2010年，基于“绿色创想”战略平台，GE在全球发起 了“绿色挑战”活动，联合风险投资企业共同承诺投入2亿美元，激发和呼吁全球各界提出新一代电网能源管理方案。在中国，我们启动了GE中国绿色挑战项目， 主题为“燃气能源的开发和利用”。 本挑战斥资一亿美元，公开呼吁企业、创业者、创新者和学生采取行动，寻找突破性创意，创建更加清洁、更加高效而且具有经济可行性的燃气能源的开发和利用。

背景资料：
绿色创想挑战赛主要关注寻找燃气能源开发和利用的创新的技术和流程，以及更出色的业务模式。在准备提交您的创意方案时，请考虑： 原创性——我们在寻求创新的燃气能源解决方案。您的创意是否独出心裁？ 可行性——我们在寻求能够在市场上获得成功的功能性解决方案。您的创意是否经济高效，或者它能否变得经济高效？ 影响力——如果成功实施，您的创意能否有助于将我们当前面临的能源挑战转化为机遇？ 牢记这三项标准将有助于您的创意方案更有机会获得评审的关注。

获奖者：罗轶，设计师 获奖项目：摩天轮农场

项目简介：

摩天轮农场，采用温室技术和无土种植，借助风力发电、太阳能来提供能源。它为高人口密度城市和未来城市的提供了一种有益的解决思路。

唯美的玻璃灯

发表于 2012-03-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

现在的节能照明产品可谓层出不穷，无论是外观还是能耗都吸引人。但是比起我们今天为大家介绍的这款灯具，恐怕就是小巫见大巫了。

这款灯具由一种名为Planilum的材料制成的，它是一种新型的“可发光玻璃”。不需要像传统灯具那样配置灯罩等设备，而只需要一块这样的“玻璃板”即可实现照明。每块Planilum的厚度仅为0.8英寸（约合2厘米），由四层特殊的玻璃组成。在空隙中充入稀有气体同时配以相应的花纹即可。它的照明寿命可达5万小时。而且90%的材料均可回收。不仅如此。当这种电灯点亮时，并不会像传统灯具那样发出高温，你可以直接用手去触碰，而不会被烫伤。

一块100瓦的“玻璃板”可以照亮约40平米的房间。

用这样的灯具装饰家居，效果一定很棒。

纸制手机

发表于 2012-03-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

电话卡可以随时买到，可是手机呢？前几天我有位朋友，坐飞机本来要在芝加哥降落，结果由于天气原因飞去了华盛顿，一个通宵都联系不上，搞得我们担心了半天。主要原因就是因为刚到美国没有手机。普通老百姓总不会为了临时需要专门去买个手机吧？所以，如果有一款造价低廉而且可以回收的手机该多好？

这款非常具有创意的纸制手机就是这样的产品，由于整体基本采用纸做原材料，所以成本非常低廉（不到10美元）。更加值得骄傲的是，这款手机是由几个台湾学生设计的，而且获得了有”设计界的奥斯卡”之称的德国iF设计奖梦幻空间设计奖。

液体玻璃

发表于 2012-03-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

这种神奇的液体玻璃不仅具备防水、防尘、防菌、透气、耐热、耐酸和抗紫外线等多种功能，而且易于使用，不会产生任何环境后遗症。只需在溶液中添加少量水或酒精，便可将其直接喷涂在任何物体的表面。它将迅速形成一层透明薄膜，其厚度不过是头发丝的百分之一。如果您将它喷洒在物体表面，那么清洗起来就更加容易了。此外，用液体玻璃加上热水来清洗其它的物体的表面，绝对不比漂白剂逊色。值得一提的是，这些液体玻璃十分环保，对环境没有任何危害！

节能花

发表于 2012-03-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款非常直观的节能提示器。它可以通过一个虚拟的花朵来提示你家庭的能耗状况。

这款产品由两部分组成，一部分是专门用来监测家庭用电、用水等资源消耗状况，并以此为依据生成相应的图形并给出相关的建议的，同时还可以根据相关的统计数据对未来的能源消耗做出相应的计划。第二部分则是一个形似花盆的EDP显示器，可以根据计划的完成情况来做出相应的反应。如果在计划耗能范围内则花朵一直会保持绽开的状态，而当用电超过计划量时，花朵则会逐渐凋谢，直至死亡。而且由于EDP显示器只有在变换图像时才会消耗能量，因此其耗电量是非常小的。

用这种非常直观的方式来帮助你节约能源、保护环境，的确是个不错的主意。。
