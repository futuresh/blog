超酷迷你环保车

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款由南澳大利亚大学的师生共同研发的新型环保汽车。虽然这款汽车采用了电力作为动力，但是它的性能却一点都不含糊：0～100公里加速时间仅为10秒，最高时速约为120公里/小时。相对于普通的电力汽车而言，动力性能大大提高。不过即使是这样的动力也仍然是非常省钱，每百公里的成本不到1澳元。每次充电可以行驶100英里（160公里）。

出于重量的考虑，这款车的前后排座舱均只能容纳一人。不过它的外形却非常酷，为了方便进入车身还采用了上掀式的设计，更使得这款汽车充满了时尚气息。

巨型空气净化器

发表于 2012-02-05 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

为了保护环境，还真是涌现出了不少新奇的想法。例如，我们之前就为大家介绍过“用喷雾给地球降温”的点子。今天的这个想法虽然没有那么宏大，但是同样非常夺人眼球。

这个巨大的钢筋混凝土建筑是专门为防治污染设计的，它的顶部配有大片的森林，可以将周围的污染气体进行过滤，生成适宜人类呼吸的氧气，为了更好的聚拢气体，它还设计有两个翅膀。植物生长所需的水和肥料都是通过风力涡轮获得的电力来运送上去的，因此不会消耗额外的能源。

在污染较重的区域建设这样的空气净化器，或许真的能够改善周围的空气质量。

节能空调窗

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

冬天取暖、夏天降温都不可避免的要消耗掉电力等能源，也就会对大气造成一定的污染。那么有没有办法，能够不消耗或者尽量少消耗能源却依然可以达到冬暖夏凉的效果呢？

这款节能空调窗就是这样一款产品，它的正反两面分别有透光和反光的功能，夏天，将反光面朝外，透光面朝内，就能够减少光照同时加强屋内热量的散发，从而降低房间温度；冬天则可以通过独特的旋转设计将玻璃进行180度旋转，则可以起到很好的保温作用。

有了这样的产品，相信一定能为你节约不少电费，同时也为环保事业做出一份贡献。

能追踪线路的机器人

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款机器人表面上看起来就像一个简陋的遥控器车。不过它并不是遥控的，它还具有一项非凡的本领。那就是它可以顺着事先画好的线路走下去。当然了，这条线路并不一定是直线，包括所有的曲线都可以。它甚至在线路交叉的地方都可以识别出相应的线路。

更为神奇的是，当碰到不会阻挡“视线”的障碍物时，它还可以主动绕开障碍物，然后重新返回自己的线路。说得这么神奇，还是来看看视频，领略一下这款机器人的风采吧。

用思想操控轮椅

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

日前，来自美国Ambient与芝加哥复原研究院（Rehabilitation Institute of Chicago）的研究人员共同开发了一套轮椅操控系统，可以通过判定特定的字符对轮椅进行操控。

研究表明，即使人们没有发出声音，但是只要他思维中出现了某个词汇，大脑仍然会向喉部发出相应的信号。而这套设备正式基于这样的理论开发出来的。所以即使使用者不发出声音，系统仍然可以探测到其发出的指令并做出相应的动作。

方便的耳机转接头

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

作为易耗品的耳机使用寿命有限，经常因为小部分的损坏就导致整个耳机报废了。为了节约资源，一位设计师构思了这款耳机转接模块，这套系统由一个三角式耳机插口转换器和带有独立接口的耳塞组成。如果某部分的组件损坏了，只需单独更换即可，十分方便。此外，为了满足高端用户需求，您也可以选择带有蓝牙设备的三角形播放控制模块，让您可以更加方便地收听歌曲。

设计：Choi Hyong-Suk

车把锁

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

骑自行车出行既方便又可以健身，受到不少环保人士的青睐，不过自行车常常受到小偷的光顾令人十分头疼。这辆防盗自行车除了在座椅处配备一把巨大的车锁用来锁住后轮之外，它独特的车把还可以拆卸下来当作环形车锁使用。用车把当车锁的最大好处就是，如果小偷破坏了车锁，那么自行车也将无法正常使用，一定程度上抑制了偷窃行为。

“红线”GPS导航仪

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

GPS导航仪的确可以为我们的出行提供巨大的便利，但是传统的GPS导航仪只是通过电子地图或语音提示来指示路径，未免显得不够直观，如果能够有一只手随时为你指示前方的路径那该多好。

这款来自美国新泽西州的新颖的“红线”GPS导航仪就可以实现这种功能。它通过将激光投影技术与GPS导航设备整合到一起来导航。在汽车行驶的同时会通过激光在汽车的挡风玻璃处形成一根弯曲的红线，而这跟红线会自动与GPS设备相连并探测汽车行驶的位置，从而不断的变换形态来指示行进的方向。
