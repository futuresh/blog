调料塔

发表于 2011-08-21 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着生活水平的提高，人们日常的调料也已经不再局限于油盐酱醋这些传统产品了，有些人家里的调料瓶甚至多达几十瓶，尤其是家里有小孩的朋友，更是要多备些调料，满足孩子的口味。

口味是满足了，寻找调料就没那么容易了。所以，你需要这样一款调料塔。

折叠自行车

发表于 2011-08-09 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

大多数折叠车为了便于折叠，减少占用空间，都选用了直径较小的轮胎，降低了骑乘舒适度。而折叠之后的自行车也不方便移动。这款新型折叠车使用了标准的26英寸轮胎。经过完全折叠之后，自行车的其他部件都收缩到两个车轮之间，节约空间的同时方便移动。官方网站上有该自行车的使用视频。

环保灯具包装盒

发表于 2011-08-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

原本被当作垃圾扔掉的包装盒其实还可以发挥更多的功用，例如我们之前为大家介绍过的“用包装盒做电视柜”就是很好的例子。今天的这款产品更是将这种思路发挥到了极致。

这个灯泡的包装盒将会伴随灯泡的“一生”，当灯泡被买回家时，它被作为包装使用。而当使用灯泡照明时，它又可以当作灯罩。等到灯泡“寿终正寝”时，它又可以发挥垃圾箱的作用，送灯泡最后一程。

设计师：Tim Edgeler

环保电动车

发表于 2011-08-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

相比于用汽油做燃料的机动车而言，电动车已经非常环保了，但是这款设计却整合了更多的环保元素。它的车身骨架外包裹着太阳能电池板，而两个轮子则是风力涡轮。因此要充电的时候，只要将电动车放到阳光充足、风力充沛的地方即可。

骨感的匍匐三轮车

发表于 2011-08-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款骨感的匍匐三轮车融合了健身与娱乐于一体。其关键部件为中部的机械扭转机构，此机构采用仿生学设计，构造类似人体的脊椎。驾驶者需要趴在上面，通过扭转身体——即依靠腰部扭转时的力量控制车的转弯。此外，这辆车使用两个电钻作为动力系统，并加装了单向离合器来保证其正常工作。驾驶者需要通过双手用来控制方向和刹车装置，同时还需要支撑上半身，而双脚则得到了放松。如果您想体验驾驶它的乐趣，那可要抓紧锻炼腹肌了。

人力“汽车”

发表于 2011-07-28 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意
