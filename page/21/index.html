节约地方的喷泉

发表于 2012-03-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了节约用水，喷泉的水都是需要循环利用的。所以，即使是开放式喷泉，周围的地面也会有许多回流孔，用于回收喷出的水流。但是这样工程似乎有些大，而且非常浪费空间。

而这款喷泉则是准确的计算了喷泉水流的落地点，并在地面上设计了多个较深的回流槽，这样一来，喷出来的水就既可以直接通过回流槽流回到地下，也不会占用过多的地面空间。

利用狗狗排泄物发光的长明灯

发表于 2012-03-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

带狗狗去公园散步，总是难免会发生这样的状况：狗狗四处嗅嗅，淡定地蹲坐，然后就便便了。有责任心的主人会把宠物的排泄物拾起来扔进垃圾桶。如何更好地利用这些排泄物呢？这个名为“公园火花”的项目，打算建立统一收集点收集狗狗的排泄物，然后利用特殊的装置处理这些排泄物，最终产生沼气，并点燃作为公园的长明灯。这一项目既保护了环境，还能够节约能源，十分环保。

设计：Matthew Mazzotta

平板电脑转换器

发表于 2012-03-11 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

你是否羡慕别人使用平板电脑（Tablet PC）时的那分轻松自在？是否希望自己也能体会一下手写触摸的乐趣，又苦于囊中羞涩？来看看这款刚刚在CeBit 2007上亮相的由NAVIsis出品的平板电脑转换器。

你只需要把它固定在普通笔记本电脑的屏幕旁边（台式机也可），然后就可以进行手写了。虽然比起真正的平板电脑还是有一定的差距，不过还是值得一试的！需要提醒的是，普通笔记本电脑的屏幕并没有保护和抗压的性能，所以使用的时候一定要注意保护屏幕。

预计上市价格130美元

壁挂式小菜园

发表于 2012-03-11 | 更新于 2018-07-24 | 分类于 都市农业

在高楼林立的钢筋水泥森林里，要想拥有一片属于自己的小菜园不再是难事。这款壁挂式小菜园的特殊设计，可以让你在家里、在办公室里、在公寓大楼里轻而易举地打造一堵绿意盎然的菜园墙壁。

这款壁挂式小菜园由单位大小的菜园模块组成，它们能通过特制的卡槽固定在墙上，且方便取下，对植物进行灌溉。经过特殊处理的菜园泥土每年只需要施一次肥。设计师称，这款壁挂式菜园适合种植香葱、罗勒、香菜、薄荷、番茄、草莓和迷迭香等植物。
