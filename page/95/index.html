折叠温室

发表于 2011-09-04 | 更新于 2018-07-24 | 分类于 都市农业

温室是用来栽培植物的设施。在不适宜植物生长的季节，能提供生育期和增加产量。不过对于普通人而言，如果家里的花卉不多，单独搭建一个温室就显得有些奢侈。而这款折叠温室就是专为这种情况设计的。

它采用了折叠的设计，因此不用的时候可以收藏起来，而需要时也可以很方便的安装。为了能够适应不同数量、不同大小的植物，它还采用了模块化的设计，可以根据需要进行拼接。

用废旧电池来照明的路灯

发表于 2011-09-01 | 更新于 2018-07-24 | 分类于 环保&公益创意

废旧的碱性电池如果得不到妥善的处理，就会对环境产生很大的破坏。而为了鼓励人们将废旧碱性电池集中起来，就有设计师设计出了这样一款充分发挥废旧电池“余热”的路灯。

这款路灯底部配有多个大小不同的孔，可以适应不同型号的电池，由于采用高效的LED灯来提供照明，因此可以将电池内剩余的电量充分耗尽。用户只需将自己用过的废旧电池放到相应的孔中即可。电量全部耗尽之后也方便统一回收处理。

设计师：Sungwoo Park & Sunhee Kim

竹材发泡复合防腐改性新发明

发表于 2011-08-23 | 更新于 2018-07-24 | 分类于 都市农业

复合发泡竹材

脱胎换骨 心满易筑

开元未来发明的复合发泡竹材是通过在原竹内腔填充现代高分子发泡材料，以加强原竹的物理性能、延长原竹的使用寿命，从而使竹子可以作为一种新型绿色建材在大跨度建筑结构、低碳建筑、快速搭建建筑、积木式建筑、临时性建筑以及新农村建设等领域得到应用。

竹子是一种能够适应不同气候且生长周期短的植物，因其加工过程中的低耗能、高产量的特性，是具有环保、可再生、可循环、经济且抗震的天然绿色建筑材料，竹制建筑材料作为钢筋、水泥及木材的替代品，能大大节约资源，改善环境，被视为未来的建筑材料。

根据波士顿咨询公司和美国自然资源保护协会《由灰变绿》报告中的数据，中国住宅和商业建筑运行能耗相当于中国水泥、钢铁行业能耗总和，而通过绿色建筑每减少一吨二氧化碳排放，可产生461元净收益，电力、工业、农业等产业相应的收益都是负的，分别为-287元、-236元、-133元。因此竹材在绿色建筑领域的应用具有巨大的发展潜力。

竹材的缺点是易腐、易蛀、不易连接，其纵向强度很好，但径向强度很差，这极大的限制了原竹的适用领域。


复合发泡竹材是骨力学、材料学与工程学交叉研究的结果。在继承传统竹材易获取、低成本、低技术、低能耗等优点的同时， 复合发泡竹材还耐腐蚀、耐霉菌、易连接、比强度高、隔音吸音、隔热防火并低弹抗震，能够建设超大跨度的建筑物。

复合发泡竹材建筑的低技术特征、平民化的造价、快速的建设周期以及可以用工业化方式建造的特点，具有绿色建造和绿色使用的双层意义，在低碳建筑建设、新农村建设、应对自然灾害、推广以竹代木等领域具有广泛的应用前景和适应性。
