概念保鲜瓶

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 都市农业

这款保鲜瓶是一款概念产品，旨在保持水果和蔬菜新鲜的同时，消除那些对人体健康有害的化学残留物。保鲜瓶采用玉米淀粉塑料为原材料，由主体、瓶盖和营养液三部分组成。在采集完水果或者蔬菜之后，立刻将其插入保鲜瓶内，然后向上推动瓶盖将其锁住。瓶内的营养液可以为蔬果提供营养，使得他们能够保鲜更长时间。当然，您也可以在保鲜瓶内插入您喜欢的花朵或者其它植物。

设计：Paul Frigout

触摸屏制作的新材料

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

德国斯图加特的研究机构近期推出了一款新型碳纳米管触摸屏，可以让触摸屏的价格更加便宜。因为该种新触摸屏由双层薄膜组成，而其中作为材料载体的薄膜便采用了与制作塑料瓶类似的便宜塑料材料制作。至于用来对触摸操作进行反应的工作层则使用碳纳米管材料混合电子传导聚合物制作，这样便可以节约成本，并保证工作质量和图像展示效果。虽说这种新材料若是遭遇湿度高的环境、大力挤压和紫外线直射，其表现并不稳定，但是研究者称该款新型触摸屏安装在当前的普通电子产品上，还是可以满足普通使用要求的。

设计：Fraunhofer Institute for Manufacturing Engineering & Automation

奔驰新式直升机

发表于 2012-02-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

最近，梅赛德斯奔驰公司和欧洲直升机公司合作，推出了这架豪华直升机。这架备有8个座位的豪华飞机，内部设计借鉴了奔驰R级轿车的内饰。地板和天花板由高档木料铺制而成；真皮缝制的座椅可以沿轨道滑动，必要时能腾出很大空间来安放行李；前排的多功能工具箱整合了小型冰箱、杯座、桌子、15英寸显示器和一台DVD机。此外，考虑到公私用户的不同需求，机舱内还配有配有一道可移动门，方便将私人空间和驾驶舱分隔开来。
