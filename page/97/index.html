折叠书架

发表于 2011-07-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款书架采用了独特的折叠结构，因此，不用的时候可以收起来放在一边，而不会占用过多的空间。

护理机器人

发表于 2011-07-25 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这是一款由日本东京大学的研究人员开发的护理机器人。这款名为Mamoru的机器人内置有广角镜头，配合着物体识别软件即可对用户所设定的物体进行监控。只要将它放在视野较好的位置，即可随时获知遥控器等经常会“躲藏”起来的物品的位置，防止为了找一个小东西而翻箱倒柜的情况发生。

年轻人尚且容易犯这样的错误，老人就更不用说了。

中暑报警器

发表于 2011-07-25 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

因为身体原因，抵抗力差的老人很容易中暑。这款中暑报警器可以测量周围环境的温度和湿度，估算出中暑的可能指数，将结果数值反映在显示屏上。除了显示数据，报警器上的LED灯还会直观地用指示灯表示不同等级的中暑指数，并发出警报。听到报警声的时候，赶紧让老人躲到阴凉的地方去吧？

售价：1050日元

折叠汽车

发表于 2011-07-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着人们生活水平的提高，有车族也在不断壮大，在获得交通便利的同时却也在被停车问题困扰着。所以也就有了这款专门为节省停车空间而设计的产品。

它的结构与我们之前介绍的麻省理工学院设计的“小巧的电动车”类似，也可以在停车的时候将车身做轻微的折叠，从而节约大约一半的停车面积。环保也是考虑的因素之一，因此这款汽车也采用了时下流行的混合动力。

除了实用之外，这款设计的外形也非常吸引人。开着这样的汽车上路，一定非常拉风。

光圈环保插头

发表于 2011-07-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

电视机、空调、电脑机箱、音响设备，我们用遥控器或者关机按钮关掉它们，就以为它们已经休息了，休息是不耗电的。果真如此么？事实上不管电器是否工作，只要插头还连接电源，都会会进入一种“待机”状态，看不见、听不见，却一直在耗电。所以，不管是出于拯救地球的环保节能大意识，还是出于拯救自己钱包的节约持家小意识，养成及时拔掉插头的习惯是必要的。

话虽如此，大多数的我们在大多数的时候都会忽略掉这个确实很容易被忽略的状况，如果待机状态下消耗的电量能够被我们看到，用最直观的、跳动的数字告诉我们电量的流逝，大多数的我们这次总不会再“忽略”吧？韩国设计师muhyeon jake kim的这个设计作品既是如此，只要插头还在插座上，就会有光环显示耗电状态，有数字指示电能消耗情况，你还能无动于衷吗？

用水供电的手机

发表于 2011-07-19 | 更新于 2018-07-24 | 分类于 电池

在2007年韩国电子展上，三星公司发布了一款以水为燃料的的微型手机燃料电池。当电池没电的时候， 你所需要的做的就是向里面加入一点水，它就可以利用水与内部的金属反映来产生氢气，之后，氢气与空气中的氧气反应就可以电力。而且其效率是目前市面上常见的充电电池的两倍。

据悉，该产品将会与2010年被广泛用于手机上，
