这款折叠灯的防护隔热外壳采用可折叠夹板制成，充分折叠之后的夹板高度与灯泡相同，能够在运输过程中保护灯泡。夹板、灯泡、电源线等灯具的各个组成部件可以放置在方形纸盒中，节约了运输和包装成本。

花园灌木修剪器

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 都市农业

要想让花园的灌木丛保持好风貌，那么定时修剪是必不可少的了。只不过，每次修剪之后，还得大费周章地收拾被剪切下来的枝桠，工作量着实不小！而这款花园灌木修剪器则参考了吸尘器以及电动刮胡刀的造型和原理，将电动修剪器进一步升级，让剪切下来的枝桠可以直接收纳。而您也不用在完成灌木修剪工作之后，还得花时间收拾花园了。

太阳能触摸屏

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 电池

太阳能电池已经在市场上流行了很长时间，但是这块却不一般。它能镶嵌在触摸屏内部，如此一来，只要在阳光下使用触摸屏，那么它就能持续为触摸屏提供能源支持。目前，科学家们研制出了第一块镶嵌有太阳能电池的触摸屏，但它的功效尚有待提高。而一旦该技术成熟，将会给触摸屏设备带来一场革命。

概念环保沐浴系统

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

此前介绍过的新月沐浴设备曾向我们推荐了一种循环利用水资源的环保理念。而这套淋浴系统除了水资源的循环利用以外，还可以利用水流出花洒所产生的能量来进行加热。地板上装有过滤层和按压发电系统，使用时能发电，把过滤后的水推送到顶部的储水箱。水流的大小是根据使用者的体重，由系统自动设定的，以免浪费。整套设备使用了环保的水泥、钢材和玻璃来代替不可回收利用的陶瓷。

设计：Paul Frigout

宝马全新电力跑车

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款电力概念跑车由一位德国学生设计。车身前部特别设计有两个尖状凸起，使其拥有良好的空气动力性能。位于尖端两侧的进气口能最大限度的吸进空气，充分冷却电机，有效提高电机的工作效率。这款车使用了下一代安全系统，配备了事故探测器，两个安全气囊在车身外部，能给驾驶员和路人提供最安全的保障。此外，这款车使用美国Better Place公司的电动车电池自动更换装置来进行燃料更换，您只需将车开进充电站，不到一分钟更换装置就可以为您更换好电池。电池更换之后，充电站会利用太阳能为电池充电，十分环保。

设计：Seungmo Lim

广角眼镜

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

通常而言，人眼只能看到180度范围内的物体，所以对于斜后方的潜在危险就无法及时做出反应。这款来自耐克的新型眼镜则可以帮助人们拓展视角。它在普通眼镜的两侧增加了两个菲涅耳透镜，从而使得骑车人可以将两侧的视角各扩大25度，这样就可以及时发现潜在的危险，提高安全系数。

食肉的生物能时钟

发表于 2012-03-23 | 更新于 2018-07-24 | 分类于 电池

大自然中有千奇百怪的植物，其中的食肉植物（如猪笼草、土瓶草等），它们依靠消化节肢动物的躯体生存。由此得到启发，设计师发明了这款“食肉”电子钟，它依靠细菌消化苍蝇躯体产生电力进行工作。时钟由沾有蜂蜜的滚动皮带、细菌生物电池与LCD屏幕组成。当苍蝇被粘到皮带上后，就会被传送到细菌消化室，特种微生物会将有机能转换为电能，为LCD供电。据称，8只苍蝇的尸体就能为LCD提供长达12天的电力供应。美中不足的是，依靠苍蝇尸体所产生的电力仅能供LCD屏工作，滚动传送带还需外置电源，还不能算是真正意义上的生物能电子钟。
