可调式多功能太阳能路灯

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

最早的路灯仅仅是用来为行人提供夜间照明的，但是现在的路灯已经不再局限于照明，而附带有许多其他的功能。例如我们专利之家之前为大家介绍过的“隐形叶片路灯”就具有很强的装饰性。而今天的这款路灯不仅具有装饰性，而且非常使用，也相当环保。

每个电线杆上都配有三个可独立旋转的路灯，每个路灯都可以根据行人的需要而定，无论是方向、颜色还是光照强度均可调节。为了更好的节约能源，这款路灯上还配有太阳能电池板，因此在大多数情况下无需额外电力供应。另外，这款路灯还可以当作小型投影仪来使用，由于其光照强度和颜色均可调节，因此可以将企业的标志或是简单的宣传图案投影到建筑物或是户外广告牌上。当节日来临时，还可以通过特定的软件对路灯的光照方式进行控制以增强节日的气氛。

设计师：Agustin Otegui

内置开关的门把手

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了节约用电，人们出门的时候通常都会关掉房间里的灯。但是晚上回家时，却不得不摸黑寻找开关。有没有既可以节约电，又不必摸黑找开关的方式呢？

这款内置开关的门把手就可以解决这一问题。门把手与室内灯具的开关相连，当旋转门把手打开门时，室内的灯也会同时打开。而出门时，由于需要逆时针扭动门把手，所以室内的灯也会随之关闭。有了这样的门把手，就再也不必担心下班回家时摸黑寻找开关了。当然，停电除外:-)

设计师：Huang Tzu-Yu，Cheng Hung, Wang Tzu-Wei,，Xiang Yu-Wei & Lin Yu-Sheng

节水喷头

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

无论你使用的是何种热水器，都不可能在启动喷头的同时就达到合适的水温，因此，很多人都会选择打开喷头，先做其他的事情，等到水温合适的时候就可以直接洗澡了。但是，如果你不是一直等在喷头前面，就会很容易浪费掉一些热水。而为了帮人们节约能源和热水，就有公司设计出这样一款节水喷头。

它内置有感应器，可以在水达到合适的温度（华氏95度/摄氏35度）时自动关闭喷头。之后再由使用者手动开启喷头即可直接使用热水。这样一来，既不必一直等在喷头旁边，又能够有效的节约热水。

柔软的花盆

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

日前，来自美国费城的一家公司计划向市场推广一种柔软的容器。这种容器采用废弃的帽檐和纯羊毛制成，可以用来做花盆、装饰品和其他的非液体容器。

之所以要推广这种产品，并不单单是因为它的外形好看，手感柔软，更重要的是这种产品非常环保。制作这样的容器要比烧制陶瓷所耗费的减少90%。而且由于都采用针织品制成，废弃之后很容易被降解，不会对环境产生危害。
