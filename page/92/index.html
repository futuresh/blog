用声波给手机充电

发表于 2011-12-01 | 更新于 2018-07-24 | 分类于 环保&公益创意

我们专利之家曾经为大家介绍过许多非常环保的充电方式，例如，太阳能、风能以及动能。而我们今天为大家介绍这项新技术则是利用声音来充电的。

这是由美国休斯顿大学的科学家提出的一种全新的充电方式，它的原理并不复杂，与“发电的马路”一样，都是利用了压电效应，只不过这一次是利用声波带动压电晶体振动并产生能量为手机充电。为了能够将声波的微弱振动转化称能量，压电晶体的尺寸就必须很小，约为23纳米（人的头发的直径约为10万纳米）。据悉，该技术将于几年内成型。到那时，无需额外的充电设备，一边打电话就可以一边为你的手机充电了。

新型智能导航系统

发表于 2011-11-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

日前，日本尼桑（Nissan）公司就开发出这样一款全新的智能导航系统，他利用现存的3G无线网络和GPS全球卫星定位系统来实时监控潜在危险路段上行人的情况，以便司机提前做出反应。

希望这样的系统能够早日普及，从而减少交通事故的发生。

车窗电玩

发表于 2011-11-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

坐在长途车里呆呆地看着外面的景物是否会觉得很无聊？相信不少人会选择用游戏机来打发时间，但是游戏机里的场景来回也就那么几种，很快也会玩腻的。那么有没有什么更加新颖的游戏方式呢？来看看这款车窗电玩吧。

它是由德国柏林艺术大学的几位学生设计的，它在传统的电玩游戏的基础上增加了一摄像头，从而可以将捕捉到的景物实时转换成游戏场景，只要将摄像头贴在车窗上即可获得动态的场景，而车速的快慢则直接决定了游戏的难度。

银质锅灶

发表于 2011-11-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

金属银通常都是用来制作首饰的，要跟厨房扯上关系，大部分人恐怕只能想到银质餐具。然而我们今天为大家介绍的这款设计则将银用到了了锅灶上。

这款锅灶底部由银制成，之所以这样做是为了利用银良好的导热性。为了更好的节约厨房空间，它还采用了柔软的硅橡胶一次成型制作而成，使用完后可以直接将其卷起来，腾出地方以做其他用途。另外，这款锅灶底部还设计有许多小孔，可以将热量循环喷到烹饪器皿上，从而减少了烹饪过程中的热量损失。

设计师; Maurizio Maiorana
