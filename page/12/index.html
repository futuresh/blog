可感应温度的变色花洒

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 环保&公益创意

我们之前介绍的“变色水龙头”曾经得到了网友们的一致好评。其实，即使这样的好产品依然还是有很多可以改进的地方，我们今天介绍的这款感温变色花洒就是一款改良的产品。

它也是通过在花洒中内置LED灯来实现一种梦幻的效果，不过，这款产品还增加了一些实用的功能。首先，LED所需的电力并不需要额外的电池来供应，而完全是通过水流来发电的，非常环保。其次，这款花洒还可以根据水温的不同而变换不同的色彩：白色表示低于32摄氏度；蓝色和深蓝色表示33～41摄氏度；粉色和紫色则表示42～45摄氏度；红色则代表水温大于45摄氏度。有了这个功能，你就能更为直观的了解水温的变化，也就不必频繁的调节水龙头了。

泊车位定位服务

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

美国加州圣莫尼卡（Santa Monica）停车场推出了一项泊车定位服务，可以让使用者通过停车场内安置的摄像头来找到自己的车子。大型停车场的构造往往酷似迷宫，将车辆停放其中再行找寻实在是麻烦事一件，而在推出定位服务的停车场，您只要将汽车车牌号码输入触摸屏操作台，它便能通过这个号码找到您的车子，并在屏幕上调出该停车位的实时摄影图像和找寻路线图供您参考。当然，这一技术所涉及的隐私问题还在讨论当中。

交互式景观灯

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

景观灯是城市亮化工程中必不可少的组成部分，而为了达到更好的装饰效果，很多景观灯都设计有动态的变化效果，在照亮道路的同时尽可能的发挥其装饰的作用。而我们今天为大家介绍的这款景观灯则将近来比较流行的交互式设计融入其中。

这款路灯可以通过内置的感应器感应到马路上的车流状况，并以此为依据来变换不同的灯光效果。这样不仅更加多变，而且还可以方便过往的司机更直观的了解道路上的车流状况。

高效燃料电池技术

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 电池

在便携设备上使用燃料电池已经不是什么新技术，只是由于种种技术限制，燃料电池的使用效率还没有得到根本的提高，因此一直没有被广泛应用到实际中来。

近日，来自台湾国立清华大学的研究人员开发了一种全新的技术，能够有效的提高甲醇电池的使用效率。每100个为一组的50微米的小孔序列来帮助二氧化碳散出，同时通过疏水性聚四氟乙烯材料来将水分子从甲醇中分离出来，通过这种方式可以将目前甲醇燃料电池的效率提高10倍。

该技术可以被广泛的应用于手机、笔记本电脑、MP3音乐播放器等便携设备上，看来燃料电池的普及真的离我们不远了。

糖水生物电池

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 电池

在2010年玩具论坛上，索尼公司将其新开发的生物电池展示在众人面前。与之前曾经介绍过的诺基亚可乐手机类似，这款生物电池也是利用含糖饮料来为相应设备供电的。该款电池名为依尼（ENE），意为娱乐新能源（Entertainment New Energy），将会运用在电动玩具上。在试验过程中，可乐、雪碧等含糖饮料都能驱动电池工作，而葡萄汁的效果最显著。只要利用3毫升可乐，即可让下图中的玩具螺旋桨持续工作1小时，由此可见其发展远景。

大轮雪橇

发表于 2012-03-22 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

在常年有积雪的俄罗斯，雪橇是人们出行常用的交通工具。这辆独特的大轮雪橇由一名退休老人设计制造，原材料全部来自废旧物品。雪橇引擎是旧摩托车上的，座椅则是游乐场内的旋转木马，连变速杆都利用了可口可乐的纪念品。车子采用的空心轮结构使质量大幅降低，同时在滑雪板表面覆盖了氟塑料以减小摩擦系数，即使冰面不够结实的地方它也可以平稳安全地行驶。这辆雪橇结构简单，维护方便，精心设计的电机位置还能够在一定程度上为驾驶者取暖，十分贴心。

诺基亚环保手机

发表于 2012-03-21 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款诺基亚手机的设计意图在于减少手机使用过程中所造成的能源消耗，让手机能够获得更高的循环使用比例。他的所有部件均由再生铝材作为制作原料；键盘安置了电容式传感器，无需单独生产各个按键，让手机部件精简许多；主显示屏采用了电子墨水显示，降低功耗；它采用了银锌电池替代了现在常用的锂电池，这种电池的可回收比例高达95%，其续航时间比锂电池高出40%。

竹艺椅子

发表于 2012-03-21 | 更新于 2018-07-24 | 分类于 环保&公益创意

这三款椅子都由竹子装饰、点缀，虽说如此，它们的风格却各不相同。人工痕迹对比自然装饰；手工作业对比机械加工；原始状态对比加工部件，各有各的风采。

第一款竹艺椅子是由不锈钢基座和可以更换的竹子靠背组成，冷冰冰的不锈钢基座和泛黄的竹子透露出的温暖感觉形成鲜明对比。

这款椅子的椅面由植物废弃材料压制而成，而起支撑作用的椅子腿还有椅背的用料，皆为通过烤制让其弯折的竹子。

这款躺椅主要用料为回收聚乙烯，但它的外形却模拟了藤竹，纯天然的气质脱颖而出。
