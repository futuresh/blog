陆空两用车

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这辆美国制造与众不同的汽车既可以在陆地上行驶，也可以在空中飞行，十分拉风。当它在空中飞行时，能展开一个宽大强健的类似滑翔伞的伞翼，同时靠尾部的螺旋桨装置提供前进的动力。这款车在陆空转换时简单方便，仅需一人就可操作。目前，驾驶这款车要求驾驶员同时拥有汽车驾驶证和相应类别的飞机驾驶证。预计这款车将于1年之内上市。

滚轮音箱

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这个看起来跟汽车轮胎尺寸类似的东西其实是一款无线室外音箱。由于采用无线传输方式，无需任何电线即可通过这款音箱进行音乐播放。而且轮型的设计也非常方便移动，适合在院子内和野外郊游时使用。

每次充电可持续播放8小时。

产品尺寸为22.8× 9.8英寸

设计师：Ron Arad & Frances Pellisari

新型自行车停放架

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

自行车是绿色出行的首选交通工具。这款外形独特的自行车停放架配色鲜明，黄色基座搭配白色的支架十分引人注目。花瓣样的支架可以围绕中间的轴双向旋转，方便在不同高度锁住自行车。这款产品不仅适合存放多种车型，而且给沉闷的街道增添了活泼的色彩，让城市生活更加生动。

设计：Yoann Henry Yvon

静止！

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

纽约市中央车站是全球最大的火车站。然而就在这座占地19公顷，日均旅客流量50万人的车站里却发生了一件非常不可思议的事情。某个周六的下午2：30，在中央车站的大厅里，有207人在这一瞬间突然静止了，他们的动作也凝固在了这一刻！这一古怪的事情令周围的人大为不解。但是当人们知道原因后都对这群人报以热烈的掌声！

究竟是怎么回事呢？

原来这是一个名为Improv Everywhere的组织发起的一次静止活动。除此之外，他们还先后发起过70多次类似的活动。如果哪一天你也有幸目睹这样的表演可不要太过惊讶哦。

标志三轮电动跑车

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这辆未来概念车吸取了四轮跑车的外形特点，但却另辟蹊径采用了三轮设计，看上去很是特别。它宽大的后轮中装有电动机，并采用类似摩托车上使用的单摇臂悬挂系统。此外，当这款车行驶在下坡路时，电机会还会自动充电，充分利用每一分资源。

设计：Juan Carlos

路边的自行车充气站

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

路边的汽车加油站很常见，可是想要在路边给自行车充气，可就没那么方便了。一位韩国设计师设计了这款自行车充气站。各种设施如气泵、胶皮管线等都与人行道融为一体，车胎需要打气时只需从接口抽出气管，接到自行车气门上即可。同时，路边明显的标志还可以方便需要人的寻找。

设计：Woon-kyeong & Kim Jong-buem

木质随身听盒

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 环保&公益创意

现如今似乎周围的每一样东西都是由冰冷、毫无个性的钢铁及塑料制成，所谓物极必反吧，越来越多的人开始追求风格怀旧、设计朴拙、手工的、自然的东西。PETER KINNE设计的这款木质随身听盒，很好的满足了几乎上述所有的要求。

内衬软垫，以保护您心爱的随身听；旁边有带夹，真有那么些马大哈连东西都拿不稳呢，还是背上放心；另有开口可用于插耳机及同步显示。材质自然环保，连里面的金属件都是废物再利用。在这个科技飞速发展的时代，这可真是件温情而个性的小东西。

套在书架上的床

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 环保&公益创意

全球各国的人们都在为居住空间越来越小而愁容不展时，设计师们也在尽其所能的设计出一些节约空间的家居产品。这款书架就是其中的一款非常有趣的设计。

它将书架与床融合在一起，白天可以把床直接与书架拼接在一起，让床体变成书架的外框。到了要睡觉的时候，则可以将“外框”拆下来拼接成床。非常的节约空间哦。

设计师：Karen Babel
