节油换挡器

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

同样配置下，手动挡的车要比自动档的省油一些。尽管如此，还是有人对手动挡汽车的耗油不满，于是这样一款省油换挡器就应运而生。它通过换挡器顶端的液晶显示来提示驾驶员换挡，从而以最佳的换挡时间使能源达到最大的利用效率。

当然，开车的时候最好还是少看这东西为妙。如果就为了省那么一点油而发生什么事故就得不偿失了。作为驾校里练习的工具是个不错的选择。

挂在天花板上的夹子？

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款看似不起眼的夹子会有什么作用呢？

下面是一点小提示：尺寸为：6×9.75×3.75英寸 理论最大承重为200磅。而且，它通常都是被固定在天花板上使用的。

答案明天（11月9日）揭晓

这是一款专门用于存放自行车的挂钩。它的操作非常简单，只要将自行车举起来，用车轮轻轻碰一下上面的凸起，夹子就会自动合上。要拿下来时，同样只需要轻碰一下凸起即可。用这样的方式存放自行车可以有效节约紧张的储藏室空间。

浴缸升降椅

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

泡澡是一种放松身心的好方法，可是对于腿脚不便的老年人而言，想要舒舒服服的泡个澡，可不是那么容易的，不仅要顾虑浴室湿滑容易摔倒，还要考虑到能否方便地进出浴缸。所以就有公司专门设计了这样一款浴缸升降椅。

它表面看来就像是一个造型有些奇特的椅子，而实际上它是一款专门为腿脚不灵便的老人设计的升降椅。老人进入浴盆之前，可以提前将椅子升到跟浴缸外壁等高，而且由于椅子两边采用了延展的设计，因此使用者可以直接从浴缸外侧做到椅子上，然后转身进入浴缸，再通过控制器来降低椅子的高度，就可以开始洗澡了。洗完之后在按照相反的顺序操作一边即可安全的从浴室出来。

汽车节能棒

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

据称，只需要将这款节能棒插入汽车的直流电插槽，它就能改善汽车18%的燃油使用效率，其工作原理在于它能够稳定汽车电子控制单元的电流，而电子控制单元又控制着发动机燃油的喷射和汽车的点火系统，所以，它可以提高汽车单位燃油的行驶路程。这款装置还可以有效缓和汽车音响、空调、车灯等之间的电子干扰，从而优化汽车电脑对燃油喷射和燃油使用的控制。目前，这款装置适用于使用两年以上且电压为12伏或24伏的汽车，暂不支持双动力汽车。

售价：79.95美元 【在此购买】

手动游泳圈

因为遗传或是后天遭遇等不幸的拖累，腿脚不灵便的小朋友往往会在夏天的时候看着朋友们在泳池里玩耍，艳羡不已，却又无法享受同样的乐趣。这款安装有特别装置的游泳圈可以让他们也能在泳池里尽情玩耍。安装有保护套的游泳圈可以固定住小朋友，用双手操作的把手能够控制小巧的螺旋桨转动。小朋友可以借助它的帮助，在水里好好玩耍一番。

智能地铁站台提示牌

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

普通的地铁站台提示牌只是会将下一趟地铁的到站时间公布在上面，或是给等车的乘客播放电视短片，以消磨空闲时光。在设计师们的创意改造之下，地铁站台提示牌也可以拥有更智能的功效。新型地铁的车厢内部会安置有扫描器械，能够识别车厢内的人脸。如此一来，地铁电脑控制系统便可以知道车厢内的人员乘坐数目，然后将这一数据通过不同数目、颜色的形象小人符号表示出来，让等车的乘客可以通过观察提示牌知道自己应该选择哪个位置等候列车。

设计：4-id Creative Network

瓶装水滤水瓶

发表于 2012-01-29 | 更新于 2018-07-24 | 分类于 环保&公益创意

目前，市场上的瓶装水包装都要通过回收处理才能够重新使用，而这款造型优美的水瓶能做到循环使用上千次而无需回收处理。其中的关键技术就在于瓶盖下那个带有微小正电荷的活性炭过滤器。这个过滤器能与自来水发生水合作用，从而能产生可饮用的纯净水。轻便的塑料材料让这款滤水瓶易于携带，有6种颜色可供选择。
