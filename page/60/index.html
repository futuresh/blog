时尚太阳能挎包

发表于 2012-01-26 | 更新于 2018-07-24 | 分类于 电池

太阳能作为新型清洁能源的代表已经被设计师们融入到很多日常用品中了：比如我们曾经介绍过的“夜光杯”和“太阳能比基尼”，包括刚刚介绍过的“卷轴太阳能充电器”也都是不错的产品。但是这款充电器并不方便实时获取太阳能，必须要单独展开来才可以。

而这款时尚太阳能挎包就将太阳能充电器与挎包结合在了一起，辅以时尚的造型，既实用又美观。由于太阳能电池板是被安放在挎包内侧的，不会影响到包的美观，当需要的时候将内侧朝外就可以实时的吸收太阳光为便携设备充电了。

健康跳绳充电器

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 环保&公益创意

现在，越来越多的大容量充电电池向市场推出，减少了一次性电池给处理工作带来的压力。而这一健康跳绳充电器，让充电电池的环保程度更上一层楼。您只要将充电电池放入跳绳的把手内，使用跳绳的时候，动能便会被转换为电能，给充电电池补充能量。除了减少家庭耗电量，将动能充分利用以外，这款跳绳充电器还量化了您的运动成果，鼓励您多做运动，让您爱上健康生活。

设计：Jooyong Kim and Sungkyung Kim

太阳能充电站

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

光是有“汽车充电站”恐怕还不够，最好连电力来源都能是环保的。这不，一家英国公司就设计了一座太阳能充电站。

它的外形跟普通凉棚类似，但是上面的太阳能电池板却可以吸收阳光并将其转化成电能储存起来。等到有汽车需要充电的时候即可为其提供电力。

可弯折的电池

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 电池

日前来自美国伦斯勒理工学院（Rensselaer Polytechnic Institute）的研究人开发出了一种全新的可以弯折的电池。

这款电池通过将特殊的“纸张”浸入电解液中，然后在纸张内植入碳纳米管来实现电池的功能。碳纳米管充当电极，利用纸张作为载体来通过电解液产生电能。

由于这种电池可以随意弯折而且可以制作的非常薄，所以如果应用于便携电子设备上将会有很好的前景。或许在不久的将来我们就能够享受到这项科技成果带来的便利。
