“内置”手机的PDA

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

目前市场上各种各样的PDA手机可谓种类繁多，但是PDA手机通常耗电量较大。使用起来未必很方便。

因此，有人就设计出这样一款内置手机的PDA。注意，我们说的可是名副其实的“内置”。平时携带的时候手机就藏在PDA的肚子里面，作为整体使用。

但是也可以分开使用，由于内置的手机做的比较精简，可以有效的节约电量。同时PDA也可以作为一个独立的单元使用，二者还可以通过无线设备同步信息。

相信不久的将来这种产品应该会很快普及开来。

不勒手的塑料袋提手

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

生活中，大家或许都有自己的办法避免塑料袋勒手，比如用橡皮管子套住提手。不过这款提手或许更加方便。

它通过一个塑料把手配合下面的几个小钩子，增大受力面积减小压强的同时，也能够增大将塑料袋之间的间距。有了它，你就再也不必担心塑料袋会勒手了。

不过，出于环保的考虑。还是建议朋友们少用塑料袋。

夹子树灯

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

设计师在可回收塑料衣夹内部安装了LED灯，给我们的生活提供了一款便捷的照明设备。这款衣夹灯的充电接口被设计成盆栽的样子，我们可以将衣夹夹到盆栽的枝桠上，给LED灯的内置电池充电。补充了半小时能源之后，衣夹灯便可以持续工作4到5个小时。

设计：Natalia Romanova

雪地专用安全气囊

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

在雪地里活动的时候，最可怕的莫过于遇上雪崩了。而雪崩中致死的人多半都是因窒息而死。而这款来自瑞士的安全气囊就可以有效的避免窒息。

它平日里就像是一款背包一样被戴在身上。而当遇上雪崩的时候，使用者只需要像展开降落伞一样拽一下拉绳，容量为150升的安全气囊就会在3秒钟之内迅速展开，围绕在头部周围，并在90秒钟之后逐渐缩小。

可不要小看了这个气囊哦，它不仅可以有效的保护头部，而且能够在雪崩中为你撑出足够的空间，以供呼吸，从而为救援赢得宝贵的时间。

售价1,100美元

可以站立的轮椅

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

对于坐在轮椅上的朋友来说，能够站立起来，无疑是他们最大的愿望之一。这款名为Pegasus（飞马）的概念电动轮椅就从一定程度上实现这个愿望。

它利用回转感应器配合电动装置，让使用者可以自由的“站立”起来（如图所示）。而且在臀部和膝盖等关键的关节部位配上相应的保护装置，让使用者不至摔伤。这样，即使坐在轮椅上，也可以轻松的完成很多只有正常人才能做到的事情。大大方便了残疾人的生活。

软碗

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

外出野营的时候虽然可以带一些一次性餐具，但是在提倡环保的今天这显然是不合时宜的，可是带着普通的餐具不仅重量太大而且也很浪费空间，我们之前为大家介绍过的“折叠餐具”就是专门为了这种情况而设计的。不过它毕竟还需要进行手工组装，有没有更为方便的产品呢？

来看看这款软碗吧。它采用柔软的硅树脂制成，不仅重量更轻而且可以随意挤压，不必担心它会因此而破碎，因此占用的空间更小。而且它还可以反复多次使用，不会造成环境的污染，绝对是出门野营的必备产品。
