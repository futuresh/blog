水中石–洗手池上的水龙头设计

发表于 2011-12-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

前者的排水方式与传统方式一致，而后者在石块顶部按照360度的角度将流水等分，这种方式不仅非常适合运用在洗手池中而且很好的达到了节约用水的目的。你可以通过调节圆盘改变流水的温度，但是得注意一些说明细节，当你使用第二种排水模式的时候你的手就湿了，而这样一来你就能凭感觉继续调节水温直到合适的温度，整个过程节省了更多的用水，并且避免了水温的过高或者过低引起用户的不适。

触摸感应变色灯设计

由Kenan Wang设计，灯体内置触摸感应器，每一次触摸灯体自动更换颜色，达到五彩的效果。手动充电，设置有充电把柄，打开把柄，仅需轻轻一摇，便可轻松为灯充电，节能环保。

满是插座的房子

发表于 2011-12-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

家用电器越来越多，满屋子窜的电线和插座着实让人烦恼，打扫起来也不方便。这套新式插座套件能节约不少拉线插座，让你的房间更加整洁。整套设备分为插座和插槽两个部分，其中插槽可以沿着墙边设置，你可以在需要插座的地方将通用插座插入插槽，这样就能就近为电器供电了。而插座可以单独开关，也可以通过总开关同时关闭整排插座。

设计：Ashley Kelly

一次性折纸蛋糕托盘

发表于 2011-12-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

大家平时在面包店或家里吃蛋糕是不是都用金属的或是塑料的餐具呢？用的时候取蛋糕很麻烦，用完之后清洗也是件很麻烦的事，现在看设计师Jung-Suk Choi设计的蛋糕托盘。一个三角形的纸，可以通过折叠，撕裂成盘子和叉子，盘子的形状正好可以取成块的蛋糕，既方便又环保。现在开始轻松的吃蛋糕吧。

节约空间的浴室设施

对于卫生间面积相对狭小的家庭来说，如何充分利用每一寸空间显得十分重要。体察生活的设计师发挥奇思妙想，设计了这两款可以扩展空间的浴室设施：可以折叠的淋浴房和带有盖子的浴缸。只是小小的改动，就给浴室带来了不一样的改变。

怀旧水灯

发表于 2011-12-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款采用水力电池作为驱动能源的优雅桌灯。作为一种环保的化学能源，主要成分为碳粉和清水的电池可以借助不时添加的水作为反应来源，以发出足够的能量支持灯具发光。为了节约能源，设计师还特地采用了节能的LED灯作为照明用料。这款外形类似老旧煤油灯的现代灯具，配上加水方能工作的启动方式，更有一种怀旧的感觉。

设计：Manon Leblanc

简易自行车反光灯

发表于 2011-12-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

来自澳大利亚的设计师利用收集来的废旧公路信号牌，制作了这款简易的自行车反光灯，包括轮辐专用灯与车架专用灯。反光灯依靠简单的固定装置与自行车固定在一起，别看它面积小巧，反光效果却非常棒。目前这款产品以其成本低廉、环境友好等特点在澳大利亚量产，受到市民的欢迎。

拉链沙发

发表于 2011-12-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款沙发的造型与我们前面介绍过的一款废物利用的“纸质沙发”非常相似。虽然这款产品中并不包含有什么环保的元素，不过却更为实用。

这款沙发由一个坐墩和拉链沙发组成，平时不用的时候可以将其整合在一起，形成一个圆柱形的小桌台。需要休息一下的时候则只需要将坐墩取下然后将拉链拉开即可。沙发垂下的部位还可以用来支撑手臂减轻疲劳，坐墩则可以用来放脚。非常的舒适。

这是一款专为喜欢出门野营的人设计的汽车。它将气传统汽车的车顶设计成轮船的甲板，这样一来，开车到海边游玩的时候，还可以舒舒服服躺在车顶，吹吹海风，晒晒太阳，充分放松身心。
