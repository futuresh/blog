太阳能充电电池

发表于 2012-02-09 | 更新于 2018-07-24 | 分类于 环保&公益创意

充电电池虽好，但是还要专门准备充电器却是比较麻烦的。所以就有了很多将充电器与电池整合到一起的设计。我们两年前为大家介绍过的一款“USB电池”就是一款非常不错的产品，而今天的这款产品则更加环保，因为它是利用太阳能来充电的。

这款产品在普通镍氢电池的外侧包裹上一层太阳能板，这样一来，当充电电池没电时，只要将它拿到太阳下晒一下即可充满电。

用抹布擦干净？这也太普通了吧。不如试试雨刷！对，没错，就是汽车上的雨刷！

环保衣料

发表于 2012-02-09 | 更新于 2018-07-24 | 分类于 环保&公益创意

伦敦中央圣马丁学院的设计师们已经研制出一项独特的技术。通过这项技术，设计师们可以在绿茶培养液中，借助细菌的发酵作用，培养出一种晒干之后材质类似纸张或蔬菜叶子模样的制衣原料，这种原料可以用来取代布料制成我们日常所需的衣物。这种制衣方式相比起现今常用的技术来说，要环保得多。

全自动喂鱼机器人

发表于 2012-02-09 | 更新于 2018-07-24 | 分类于 都市农业

喂鱼是一件很枯燥的事情，如果能够将这种事情交给机器人来做，应该可以节约不少的人力。

这款喂鱼机器人就可以通过底部的推进器在水上行驶，当行驶到特定区域后，它就会发出声波来吸引鱼群，然后根据不同的温度和鱼群密度来释放适量的鱼食，之后机器人就会前往下一个区域继续相同的喂食动作。而当喂食结束后，机器人就会返回原位，等待新的鱼食放入。

另外，这款机器人还引入了一个非常有趣的概念，它的水上部分可以当作花瓶来使用。针对不同的鱼塘种植不同的植物，一方面可以更直观的区分鱼的种类，另一方面，也可以通过植物的长势来判断鱼群的生长状况。

设计师：Ran Xiangfei

相关产品：“档栅机器人”
