林肯蝙蝠车

发表于 2012-01-31 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

花了3年半的时间，瑞典改装车达人将一辆1973年的林肯大陆车改造成为蝙蝠车，整个过程耗费100多万美元。除了外形酷炫，车内还安装有卫星导航系统、语音识别设备、摄像机、DVD播放器和等离子电视，车身高度也可调节。从实用的角度上看，该车功能与蝙蝠侠正版车不相上下。

两栖滑板车

发表于 2012-01-31 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

水陆两栖的跑车你已经见过了，但是水陆两栖的滑板车你见过吗？

这款滑板车就是一个水陆两栖的设计，由于轮子的辐条被设计成了涡轮的形状，所以平时它可以像普通滑板车一样在路上行驶。而要游泳的时候只要将扶手折叠起来，用电力驱动轮子转动，即可通过涡轮来提供游泳时的动力。

带着这样一款滑板车去水边嬉戏，一定能享受到更多的乐趣。

设计师：Yong Jieyu

标签酒杯

发表于 2012-01-31 | 更新于 2018-07-24 | 分类于 环保&公益创意

作为一种非常环保的设计思路，将产品的包装加以利用，使其直接为产品服务，已经成为一种趋势。大到电视柜小到蛋糕盒甚至药片的包装袋都可以用这种思路来提高利用率，从而达到环保的目的。可是像红酒这种在零售时通常都没有包装的产品，是不是就无法采用这种环保的思路了？别着急，不是还有标签吗？

这款新颖的设计就将红酒的标签稍作改进，变成了一次性的酒杯。酒瓶的标签不仅可以用来展示品牌和常用的产品说明，当它被取下后，还可以当作酒杯直接用来饮酒。同时，为了适应这种纸杯的厚度，酒瓶还被设计了一个凹槽，这样既方便存放酒杯，又不会因此影响到酒瓶的“身材”。

设计师：Viktor Pucsek

相关产品：“折叠餐具”

个性的私人车载空调

发表于 2012-01-31 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

夏天开车时，能够开空调享受一下，一定很爽。不过，对于警察等特殊职业，由于需要整齐的着装，所以，有时即便开着空调也难以享受到清凉的感觉。于是，有人就发明了这样一款空调气管，只要把一段接在空调出口上，另外一段通向你想要“冷却”的任何地方，就可以既不影响着装，又能够充分享受清凉了。

不过，记得不要过于频繁的使用这种东西，否则很容易感冒哦。

路灯公交车

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

如果你已经厌倦了城市里拥挤的公交车，那么这样一款新颖的概念设计或许会让你眼前一亮。

它的外形就像是一个巨型的气泡，由顶部的太阳能电池供电，可以同时容纳四个人以及随身的行李，内置有GPS导航装置和声控系统。

而这款新型公交车最具创意的地方还在于，它不会占用路面的停车位置！因为到了晚上当汽车处于闲置状态时可以直接挂到杆子上作为路灯来使用。

相信随着科技的发展，这种概念公交车总有一天会走入我们的生活。

设计师：Ross Lovegrove

环保花灯

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款设计与我们之前为大家介绍的一款“节能花”采用了类似的思路。所不同的是，这朵花更加真实。它同样是通过探测家里的用电量来改变不同形态的。当探测到你家里能耗过高时，它就会低下头去，如果你的用电习惯良好它则会绽放开来，并投射出美丽的光芒。

想要看到美丽的花朵吗？那就一定要养成良好的用电习惯，培养起环保的意识。

磁力耳机适配器

发表于 2012-01-30 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

你是否有过类似的经历：使用电脑时，要起身去做一些事情，却忘记自己仍然带着耳机，结果……

耳机损坏倒是小事，万一用的是笔记本电脑，把电脑重重的摔在地上，那可就损失惨重了。这款磁力耳机适配器就是专门用来解决这个问题的。

它的原理类似于苹果的MagSafe技术。通过磁力将适配器和插头吸附在一起，当遇到较大拉力时就会断开，从而保护耳机和设备都不受损坏。

该产品可用于所有的3.5毫米耳机插孔。

健康环保的水池清洁器
