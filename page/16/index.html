环保的电子秤

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款概念秤是专门用来称量一些厨房专用的食材的。它的造型非常特别，采用了环形的设计，刚好可以将碗的底部套住并使其悬空。

使用的时候，需要先将一个合适的碗放在上面，然后把需要称量的东西放入碗内，通过OLED显示屏读取数据。另外，这款概念秤虽然是电子产品，但是它却不需要任何的外接电源。因为使用的时候，你需要将带有OLED显示屏的一端旋转出来，方便读取数据。而旋转时内置的发电装置会将这部分动能转化为电能，供给电子称使用，从而达到节能环保的目的。

设计师：Martin Pråme

本田概念三轮车

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

城市扩张和人口结构老化等社会问题的突显给灵活代步工具提供了越来越广阔的市场需求。这款小型电动代步车正是专门为这一市场打造的。这款概念三轮车由安装在车身底部的锂离子电池供电，仅供单人乘坐。它采用电动透明顶罩设计，停车时顶罩可以用来遮盖座舱防止灰尘和雨水落入，在行驶时顶罩可以打开当做挡风玻璃使用。驾驶座椅两边半遮挡的车身不仅可以挡风还可以给驾驶者提供安全保护。这款车还在车身前部设计了一个储物仓可以用来携带各种物品。

感应式分类垃圾桶

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了方便对垃圾的回收利用，很多地方都已经配备有分类垃圾桶。但是人们还是很难以对垃圾的分类做出准确的判断，即便是垃圾桶上的提示很多时候也不够准确，例如，并非所有的金属制品都是可以回收利用的。因此，就很难达到垃圾分类的预期目的。

而这款新颖的分类垃圾桶则通过条形码识别技术解决了这一难题。它内置有条形码扫描器，可以通过对产品的条形码的扫描来识别产品的材料从而做出相应的判断。当你需要扔垃圾的时候，只要将条形码在垃圾桶的扫描器上晃动一下，垃圾桶就能识别信息然后自动打开相应的垃圾桶。

设计师：Woo Seok Park
