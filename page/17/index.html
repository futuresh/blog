汽车太阳镜

夏日到来的时候，汽车在太阳下晒几个小时车内就会变成蒸笼一般，似乎汽车也应该“防晒”才行。一家巴西公司别出心裁，在海滨停车场给汽车发放巨型太阳镜，顺便宣传他们的产品。这款特殊材质的太阳镜不仅可以遮阳，还可以防紫外线，两个眼镜腿刚好可以挂在后视镜上。海滨的汽车带上了各色的太阳镜，仿佛也在享受悠闲的假日时光。

超轻便的自行车

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

你能够想象一个自行车骨架却只有2.75磅（约合1.25千克）吗？

这其实是一款由碳纤维制成的自行车骨架，由于碳纤维独特的低密度、高强度的特性，才使得这款自行车实现了这种不可思议的超轻重量。另外，它还采用了独特的网状设计，在保证强度的同时，也最大限度的减轻了车体的重量，

车架售价6,995美元，整车售价11,995美元。

两用三轮车

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们最近已经为大家介绍过好几款三轮汽车了，但是今天的这款汽车与前面产品最大的不同在于它不仅可以作为三轮汽车来使用，还可以当作两轮摩托车来用。

如果在较为空旷的地区行驶可以将车体平放当作三轮汽车来使用，当遇到交通堵塞等比较拥挤的情况时，则可以将三轮车直立起来，变成一个两轮摩托车，这样一来，就可以顺利地在城市的狭窄道路中穿梭了。不仅如此，如果碰到车位紧张的时候，也可以通过这种方式来减少占用的面积。

最后，无论是作为三轮汽车还是两轮摩托车来使用，这款产品的外观都非常酷！

设计师：Seyyed Javad Ghaffarian

保持干燥的自行车坐垫

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

遇到雪天或者雨天，自行车的车座常变得潮湿，为了解决这一问题，设计师推出了这款特别的自行车坐垫。这款自行车坐垫底下附加了一个收纳箱坐垫套，如果您在停车的时候忘记给自行车套上防潮套，那么只要将收纳箱反扣，将由硬质塑料和柔软纺织品组合而成的坐垫套套上座位就可以了。这款坐垫套内里还有些许空间，可以让你存放物品，外壳上还贴心地准备了一排安全灯，让车子在夜间更加醒目。

设计：Sam Holmes

小型移动庇护房

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这辆移动“房车”专为那些无家可归的流浪者而设计，设计者希望以此为他们提供一个容身之所。整个车身重量很轻，仅为225磅，结构简单但不漏水。除了为使用者提供栖息之所外，白天它还可被当作移动便利店出售物品，让流浪者也能挣钱营生。内部装饰虽然相对简单，但是功能齐全：有小型厨房、厕所、可折叠家具，还有一个用来收集雨水的水槽。

设计：Paul Elkins

方便爬台阶的多功能婴儿车

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

推着婴儿车上下台阶可不是一件轻松的事。为了解决这个难题，设计师构思了这辆能轻松应对爬楼梯问题的多功能婴儿车。上楼时，连在一起的三个小轮轮换落地，整个小车就会平稳地一步一步“走”上台阶，保证宝宝的安全。这款车的拉手位置符合人体工程学，使用起来不会感到劳累。座椅和支架都可以调整角度，同时它宽厚的大轮也可以减少路面的颠簸。另外，这款车还可以轻松折叠、拆装，十分方便。

设计：Arturo Arino

一次性照片打印机

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款简洁小巧的照片打印机类似一次性相机，它使用环保材料制成，使用后既可抛弃处理。您在选购时，可以根据外包装上印制的介绍内容选择打印机的打印张数、打印尺寸、分辨率、打印色彩或是打印速度等。使用时，撕开两端的封条内置的电池就会启用，然后您只需将数码相机与这款打印机用USB线连接就可以开始打印照片了。虽然目前这款产品仍处于概念阶段，但是其简捷方便的特性肯定会受到不少爱拍照人士的喜爱。

设计：Yuexun Chen & Chia-Chen Hsiao

无链折叠自行车

发表于 2012-03-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

大多数折叠自行车为了节省空间都采用了小轮子设计，而这款自行车却拥有直径20寸的大轮子。不过，经过设计师的巧妙设计，大轮折叠车也不乏优点。这款车的无链条设计，使得脚蹬与齿轮盘合为一体，齿轮与轮毂直接啮合，大量节省了空间。简洁直观的折叠结构，让您在很短的时间里就可以方便地展开或折叠。此外，方便的的无链设计使得这款车折叠后仍能像行李箱一样拖着前进，非常省力。

设计：Sanghyun Jeong and Jun-Tae Park
