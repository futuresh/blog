有31个轮子的轮滑盔甲

发表于 2012-01-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

或许有很多朋友觉得我们之前介绍的“把骑车人变成自行车”的创意太过疯狂或者压根就无法上路，那么下面这个设计没准能改变你的看法。因为这个创意比那个自行车更加疯狂，而且还真的上路了！

这是由一名法国的学生设计的轮滑制服，它看起来就像是一件盔甲。但是最特别的地方在于“盔甲”上有整整31个轮子，穿上这样的盔甲你就能够以几乎任何姿势在路上滑行。

设计师：Jean-Yves Blondeau

电动越野自行车

发表于 2012-01-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款外形特殊的自行车与常见的越野自行车不同的是拥有两个超宽的轮胎，它几乎能够适应任何艰难地形，如沙地、雪地等，甚至在冰面上都可随心所欲的骑行。这辆车由600瓦的电机驱动，最高时速可达20英里。同时，它还配有14级变速箱和大容量锂电池，单块电池可供骑行的距离约为100英里。设计者还同时提供定制不同颜色和款式的服务，以满足不同人群的需求。

设计：Dan hanebrink

多功能生态橱柜

发表于 2012-01-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这套绿色环保的家用水循环系统能很好地解决厨房水资源的浪费问题。整套设备包含一个洗涤系统以及独立的原始废料、固体废料和液体废料微循环处理系统。悬吊瓶用来种花，如果您浇的水太多了就会从花瓶底下渗漏，从而进入液体废料循环系统。橱柜底部还有一个存储蚯蚓的容器，较难溶解的固体废料，如苹果皮，便可成为它们的美味佳肴。该橱柜还可以处理一些废弃的罐头或者瓶子。所有处理干净的水都将汇入蓄水池，方便用户重新利用。

设计：Victor Massip & Laurent Lebot 【官方网站】

患病儿童助力车

发表于 2012-01-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

脑瘫能使儿童产生严重的生理功能障碍，如无法控制肌肉等，导致他们不能像正常孩子一样玩耍和行走。设计师为脑瘫患儿研发了这款多功能助力车，不仅可以帮助孩子提高活动能力，还有助于让他们恢复自信，获得恢复。助力车由一个可升降的马鞍式座椅和带轮支架构成，通过无线遥控，带捆绑护垫的座椅能上下移动。坐在上面的儿童可以利用助力车进行“蹲下”、“站起”的活动，三轮支撑架也能在一定程度上帮助他们获得平衡能力。座椅上还可以安装许多附件，如脚凳或小桌板等，为患儿提供更多的方便。

设计：Roberto Reyes

不含电子元件的便携音箱

发表于 2012-01-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款便携音箱的最大特点在于它的主体不含有任何的电子元件。

由于内部通道采用了海螺的螺旋形设计，就像是管乐器一样，可以直接将声音放大，因此你只需要将耳机插到它的输入端，即可收听到外放的音乐。非常环保的设计哦。
