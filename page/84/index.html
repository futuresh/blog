环保超市推车

发表于 2012-01-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

设计师通过给超市推车安装小部件，将推车移动过程中产生的动能转化为电能，并将其储存起来，然后再通过传输设备，在推车停靠回超市入口的时候将储存的能量转移到超市供电中心，为超市用电提供支持。

设计：Kitae Pak & Inyong Jung

鸟瞰倒车系统

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

传统的倒车系统只能够对某一个方向的影像进行实时监控，即使在多个方向同时安装了摄像头，也只能分别观察，不够直观。为了帮助驾驶员更好的了解汽车周围的状况，更为方便的倒车和停车入位，日产公司（NISSAN）日前开发出了一套全景倒车系统。

它通过将车身四周的四台180度摄像机所拍摄到的影像进行拼接，从而构成完整的全景影像来帮助驾驶员以最为直观的方式进行实时的监控。就像是从天空俯瞰的影像一样。

环保三轮车

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款由柏林科技大学与宝马等著名厂商合作设计的新颖环保三轮车。

它的外形类似于国内一些城市常见的一种改造过的三轮摩托车。不过既然有宝马的参与显然不会是三轮摩托车的水平。首先，它与我们之前介绍过的一款来自印度的“零污染汽车”一样，也是完全由压缩空气提供动力的，因此非常符合环保的要求。

另外它的动力性能也相当不错，最高时速可达100公里/小时，0～60公里/小时加速时间为7秒。而能耗方面，每行驶1公里所消耗的压缩空气约为60克。该车长3米，宽1米，重约400千克，虽然看似很小，但是却可以轻松容纳两人。

虽然从动力和空间上来看，要比现在常见的汽车要逊色不少，但是在城市中使用，可以说是非常经济实惠了。

相关产品：“超酷环保车”“超酷迷你环保车”“三轮汽车”

折叠飞机

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 环保&公益创意

从莱特兄弟1903年第一次试飞成功，到现在已经100多年了，飞机也已经逐渐走入了人们的生活，甚至有一些富豪已经拥有了私人飞机。虽然在普通人看来，能买得起飞机的人已经非常富有，但是能节约的还得节约，譬如停机棚。所以也就诞生了这样一款折叠飞机。

遥控气垫车

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款看似普通的遥控汽车其实是全球第一款反重力遥控汽车。它内置有两个内外函喷气发动机，使得它可以悬浮于空中，从而减少了传统的轮胎所产生的摩擦力，因此速度更快，也更为灵活。另外，为了增强坚硬度，该产品的外壳由莱克桑聚碳酸酯制成，底部则由聚乙烯泡沫制成

这款遥控车采用单个无刷电动机驱动，最高时速可达25英里。而升级版的双无刷电动机的产品的最高速度则可以达到50英里/小时。

立体花园

发表于 2012-01-06 | 更新于 2018-07-24 | 分类于 都市农业

城市景观可能在建筑艺术上成就非凡，但常常缺乏亲和力和自然情趣。法国植物学家Patrick Blanc的立体花园给出植物改变公共空间的极好范例。这种立体花园由一层层的毡垫、用以防止根系破坏的塑料及金属架及一套滴水灌溉系统组成，曾成功地应用于塞纳河畔及巴黎的Pershing Hall旅馆。它不仅仅是一座美丽的活体雕塑，还能够很好地改进饱受污染的城市空气。
