骑自行车？不如变成自行车!

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是来自美国的一项非常古怪的专利。它的名字叫做“Be The Bike”,看过图片想必也不需要过多的解释了。

其实就是把自行车的骨架换成驾驶者本人，这样的话驾驶者本人就跟自行车合二为一了。

虽然这个设计在2004年获得了美国专利局的授权，但是要骑上这样的自行车上街恐怕还真得需要一定的胆量。

车用犬袋

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

想要带着自己的狗狗出门兜风，可是又不想让车里被狗狗搞得凌乱不堪，该怎么办呢？来看看这个来自1936年的设计吧。

这款设计最早刊登于1936年6月的Popular Mechanics杂志上。用一个大袋子将狗狗挂在车门外侧就可以避免上述的情况了。

如果现在有人再用这种方法带狗狗出门，不知道会是什么后果哦。

带触摸显示屏的华硕键盘

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款华硕键盘在原来安放数字小键盘的位置设置了一个触摸显示屏，它完全可以充当显示屏使用，运行键盘内置的系统。这样键盘实际上就变成了一款微显示屏电脑了。

这款键盘采用了1.6GHz Atom Z520处理器，内置1GB DDR2内存，可选配16GB或32GB硬盘，支持IEEE 802.11n无线网卡和蓝牙，高配版带有无线高清多媒体视频输出功能。此外，还拥有电视、移动设备、网络、音频、读卡器等多个接口。它的售价约为400美元，将于本月发行。下面就让我们欣赏这款键盘运行英特尔Moblin系统的视频吧!

注解：Moblin项目由Intel公司于2007年7月正式启动，最初主要集中于开发MID软件系统，能够访问Internet上丰富的内容，同时没有与PC相关的复杂性。这些设备从移动互联网设备、车载信息系统、便携式媒体播放器到个人导航设备、数码机顶盒到笔记本电脑等以各种形式推动新技术、新设备的发展。

快捷饮水售卖机

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款快捷饮水售卖机无需人工售货员干预。使用者只需要将随身携带的饮水瓶放到饮水机上，它就会自动给饮水瓶灌满水。饮水机通过RFID技术与使用者的手机沟通的，直接扣除使用者的购水费用。节约人工的构想与自动售卖机类似，而可重复使用、方便使用者携带的饮水瓶相比起自动售卖机里的饮料，可以节省更多的垃圾处理人工，使用过程更加环保。

设计：Oliver Craig

肥皂盒里的花园

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 都市农业

怎么才能用肥皂盒培育出一片自己的花园呢？别惊讶，没错，我的确说的是肥皂盒！

该技术由一家日本公司发明。你只须要购买他们的产品，然后将里面的一块类似于纱布的东西在放满水的肥皂盒里面浸湿，使他适于种植。然后往肥皂盒里面加入约一英寸厚的土壤。这时肥皂盒开始腐烂，没几天你就会看到肥皂盒变成了一朵盛开的苋菜。就是这么简单！

还有很多其他种类的花朵可以供你种植，不过都需要花一点银子哦。
