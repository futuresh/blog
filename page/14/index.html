感应式全能水龙头

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了节约用水以及避免顾客手上的细菌和污物通过水龙头的开关相互传播，许多公共场所的水龙头都被被做成了感应式的。你无须动手，只要把手放到水龙头的下方，就可以通过红外感应自动出水。而这款全能水龙头则是将这种设计思路进一步的延伸，将多种功能都整合在了水龙头上，而且全部都采用了感应触发的方式。

水龙头的感应面板被分割成五个区域，上方的两个分别表示洗手液（soap）和消毒液（disinfect），下方为普通的清水（water），左右的“+”“-”则是用来控制水温的。你只需要按照需要在相应的区域晃动一下手，就会得到相应的功能，而且全程不会与任何固体相接触，不仅使用便利而且非常卫生。

生态节能的长途汽车站

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着人们环境保护意识的提高，绿色建筑受到越来越多人的欢迎。汽车站作为人多、车多的一贯场所，其内部经常人声嘈杂、通风不畅，非常容易变成环境污染严重的地区。规划师和建筑师为了解决此类问题，设计了这个现代化的新型汽车站。它类似波浪形状的外形，给人一种流动感，十分优美。特别设计的宽大天顶和自由开敞的入口，让车站内部的光线更加明亮，保证了环境的宜人舒适。此外，良好的通风效果也在一定程度上节约了能源的使用。

设计：Austin-Smith: Lord

摇摆充电电池

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 电池

除了常见的充电器充电形式以外，这款镍电池还可以通过手动方式充电，无需借助其他配件的帮助。这款电池内部由五个部分组成：可反复充电的镍电池、线圈、磁铁、轴以及橡胶软垫。当使用者摇晃电池的时候，磁铁就会以轴为中心，在富有弹性的橡胶软垫的帮助下，迅速上下滑动，穿越线圈中心，磁感线被不断切割，从而产生电流，给镍电池充电。

三轮汽车

发表于 2012-03-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

汽车越来越普及，随之而来的车位紧张的问题也就日益凸显，因此方便泊车的小型车也就受到一部分人的喜爱。不过这款来自米其林汽车设计挑战赛的获奖作品却为我们提供了另外一种思路。

它的造型非常特别，采用了三轮的设计。前部为两轮，而独轮部分则安置在后部。这样一来，两部车相互交错就可以占用相当于一部传统汽车的空间，从而大大节省了停车的成本。而为了使乘客和驾驶员获得尽可能宽敞、舒适的内部空间，它也采用了类似于我们昨天介绍的一款飞机座椅的曲折设计。另外，它还采用了时下流行的电力驱动系统，从而达到节能环保的目的。

相关产品：“小巧的电动车”“奥迪灯泡概念车”
