铅笔打印机

发表于 2012-01-13 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款打印机的墨水就是铅笔头上的铅芯粉末。您只要将多余的铅笔头插入打印机上方的小孔内，打印机会自动将铅笔头削成粉末，然后运用电烧原理将木屑和铅芯粉末分开,并将铅芯粉末存入其墨盒内，这样它就能利用这些铅芯粉末打印了。同时，这款打印机还有擦拭功能，您只要将布满铅笔字迹的A4纸插入打印机的另一段，纸上所有的铅笔字迹就会被完全清除。这样，A4纸就可以反复使用了，还能充分利用铅笔头，经济又环保。

设计：Hoyoung Lee

立体自行车库

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

作为一种清洁的交通工具，自行车已经越来越受到各国的重视。不过要普及自行车，首先一定要解决自行车的停车问题。不仅要有足够的空间，还要保证停车的安全性以及便利性。

这款来自日本的立体自行车库就给我们提供了一种很好的思路，它的空间都位于地下，采用全自动的设计，你只要将自行车放到平台上，按下按钮，自行车就会被存储到地下车库。而需要取车的时候只需要将特定的取车卡在扫描仪上刷一下，您的自行车就会在半分钟内被自动调出。

一套这样的车库可以存储9400辆自行车，每部车每天的花费也仅需100日元。

本田代步器

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

本田公司近日推出了这款8字形代步器。它能感应到乘坐者的身体倾斜方向，从而控制代步器的前进方向。虽然看似独轮车，但实际上这款代步器的“大车轮”内部藏有若干个小轮，要在上面保持平衡不是难事。它仅重22磅，座椅可以折叠收纳起来，方便搬动和携带。

灯柱休息凳

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

散步是一种非常简单高效的健身方法，这款灯柱休息凳可以为散步的人们解决休息的后顾之忧。它由废旧木料和塑料复合制作而成，并经过环保处理和二次加工。整个凳子的造型简单大方，经过特殊处理的表面非常易于清洁。凳子的背面有两组圆形螺栓钢带，可以方便牢固地将凳子安装在路边的灯柱上。灯柱休息凳作为城市基础设施的一部分，将为城市的环境添加新的亮点。

设计：Nichola Trudgen，指导老师：Paulus Maringka

节电小精灵

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

节约用电成为全世界每一个家庭不得不面对的问题，比如空调，天冷了需要吹暖气，天热了又需要吹冷气，这些都是不可避免的，那怎么才能够节约电费的开支呢？试试这款节电小精灵，他一定会带给你惊喜。该产品通过探测线路中的碳的放射量来控制用电量，一旦检测到有处在待机状态的用电器，节电小精灵就会立刻切断正在待机的用电器。

这个小东西是即插即用的，因此你不需要对自己家里的线路做任何改动。只要将它插进插座就可以了。我不是很清楚待机状态的电器会用掉多少电，但是数据显示，每台节电小精灵一年可以节约的电费高达38美元（美国的电费大概0.02美元一度）。如果您会算账的话，肯定会对这个小东西有兴趣的。

夹子铅笔

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

如果使用传统的铅笔，当笔头太短无法书写的时候就该削铅笔了。但是要知道铅笔都是用木材制成的，这样一次又一次的削铅笔，每年所浪费的木材数量可不是个小数字。所以才发明了自动铅笔等替代品。

但是在许多需要绘图的专业领域，传统的铅笔还是无法用自动铅笔来替代。不过通过巧妙的设计还是一样可以节约木材的。像这款夹子铅笔就是其中之一。

它通过夹子式的笔杆设计来达到无需削铅笔却一样可以顺利使用的目的：笔芯不够了？那就打开夹子，把笔芯抽出来一段，继续使用。这样一来，你每次只需要更换笔芯而不必“削”掉笔杆了。

摩托车防盗罩

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款多功能防盗器。它采用了折叠的结构，展开后可以将摩托车整个扣上。内置的警报器会适时地发出警报以达到防盗的目的，不仅如此，它还可以起到遮阳、挡雨和防尘的作用。就像是给摩托车配上了一个小房间。

设计师：Marc Graells

混合充电器

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 电池

这款充电器有两个接口，可以通过普通电源或是USB接口给充电电池补充电量。也正因为有了USB充电接口，它还能给各种电子设备补充电量。而放置充电电池的凹槽也可以进行调整，能够为5号和7号电池充电。

设计：Hiroaki Tanaka 【官方网站】

卷筒纸的环保设计

发表于 2012-01-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

在将卷筒纸使用完毕之后，没用的硬壳卷纸芯往往会被我们丢弃。设计师便对其进行了改造，将废弃的硬壳卷纸芯重新利用起来。这款新型卷纸的纸筒可以沿着环保标签伸展开来，变成简洁的日程表。您可以将日程记录在上面，或把它当作便笺，沿着刻线将记载有待办事宜的纸片撕下，方便使用。

设计：Kim Ji-eun
