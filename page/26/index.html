便携太阳能充电器

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 电池

这款小东西看起来不起眼，不过在关键时刻，它绝对可以解你的燃眉之急啊。它使用太阳能充电，在你手机、MP3等便携设备没电，手边又没有电源的时候，这东西可是能派上大用场的。它内置的锂电池足够手机连续通话六个小时。怎么样，有这么一款小巧的充电器，那些整天为手机电池没电而苦恼的商务人士是否可以略微宽慰一些呢？

便捷拆装自行车

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款自行车尽最大可能方便了不同用户的需要。自行车车座下方、后轮和踏板都设置了警示灯，把手下方则留有照明灯，细心的设计提升了自行车的安全系数。最实用的莫过于自行车的可拆卸设置，根据不同需要，各个使用者可以调整车筐大小，选择适合自己的组合结构。

设计：LOTS Design

智能双轮车

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款骑行式的赛格威（Ｓｅｇｗａｙ，又译思维车），它由电力驱动，采用两个轮子做平衡，根据人体的倾斜车身可以智能调整平衡。与一般直立式赛格威不同的是，这款车采用了仿摩托车设计，使用者可以乘坐在座位上。设计者为这款车取名为“金牛座”，希望通过这样的设计改善城市交通拥堵的问题。

设计：Erik Lanuza

一体工具箱

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

其实，在家里配备一个简单的工具箱是个不错的选择，至少当遇到一些小问题的时候，我们也能借助工具的帮助，自行解决，以应对突发状况。聪明的设计师便将常用工具做了整理，并设计了这个内部配有相应卡槽的工具箱。为了节约空间和生产原料，充分利用工具自身的优势，当锤子的把手安放到工具箱内部的时候，它还能充当工具箱提手，可谓是一举两得。

镜框优盘

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

现在的优盘体积越来越小，这样虽然节约了空间，但是却非常容易丢失。于是就有不少厂家反其道而行之，增加了优盘的体积。而我们今天为大家介绍的这款产品不仅增大了优盘的体积还将它设计成了镜框的样子，增加了不少情趣。

世界第一款太阳能无线键盘

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

罗技公司近日推出了世界上第一款太阳能无线键盘，这款键盘顶部装有一排太阳能光电板，可以通过阳光或普通灯光为其充电。当电量充满时，它可以在黑暗的情况下连续工作长达3个月的时间。键盘上还附有光照量提示，能随时随地告诉您当前的照明强度。此外，这款键盘采用可回收塑料制成，在保证产品质量与寿命的同时，也体现罗技的节能、环保理念。预计这款产品将于本月15日上市。

售价：79.99美元

环保螺旋笔芯

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

设计师给常用的一次性笔芯更换了新造型，变成螺旋形状的笔芯不仅身形更加窈窕，同时还能增加墨水的储存量。因为计算过螺旋的弯曲角度，正常使用的话，出水并不会出现堵塞现象。

自行车租赁系统

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

行走于乡间小路，欣赏美丽的景色，的确是一件很惬意的事情。但是毕竟交通不便，走长了还是会感到疲惫，可是如果开车去又有失情调。如果能够租到自行车就好了。

于是，一家荷兰的公司就开发了这样一套自行车租赁系统。它可以通过内置的RFID芯片来定位自行车的位置以及存储相关的租赁信息，你从一个租赁点租到的自行车都可以很方便的在其他任何一个地方返还，而不必跑回原来的地方。上面的RFID芯片会记录相关信息并提示你支付相关费用。

对于外出郊游而言的人而言，的确一套非常贴心的系统。

太阳能空调

发表于 2012-03-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

北方大部分地方冬天都有暖气，因此空调通常都是用在夏天制冷的。但是在环保意识逐渐增强的今天，放着炽热的阳光不利用，让它白白浪费，还要额外花费能量来抵消太阳光产生的热量多少有点令人不解。别着急，我们专利之家今天为大家介绍的这款产品就是一款能利用太阳能来驱动空调的新型产品。

该产品不仅可以像传统产品一样，夏天制冷，冬天制热，还能够提供由太阳能加热的热水。有了这样的产品，既利用了太阳能、保护了环境还可以享受冬暖夏凉的舒适生活。

该产品由澳大利亚国立大学开发，目前尚处于测试阶段。
