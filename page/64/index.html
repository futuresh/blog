来自引擎的热电

发表于 2012-01-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

引擎在提供给我们动力的同时也浪费了很多的能源，据统计，将近60%的热量在引擎的热循环过程中被浪费掉了，如果能够将这部分能源加以利用，一定会为环保事业做出重大的贡献。

来自美国克莱蒙森大学（Clemson University）的物理学家Terry Tritt在今年的nanoTX会议上提出了一个伟大的设想，他希望开发出一套系统能够将这部分被浪费掉的能源加以充分利用。其实这种系统并不算新鲜，美国国家航空航天局（NASA）早在30年前就通过吸收外太空的热量和辐射并将其转化为电力用以供给空间设备的运转。而如今Terry Tritt所要从事的事业则是以更为低廉的成本将这项技术应用到更为广阔的范围中来。

据称，即使以目前7%～8%这种较低的转化效率来计算，如果这套系统能够被应用到重型卡车上，那么单单美国一个国家每年就能够因此而节约15亿加仑的柴油。由此带来的环境效益和经济效益则更为巨大。如果能够将这一效率进一步提升，那么它对环保事业所做的贡献则是不可估量的。

扫把自行车

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

想要像传说的女巫一样骑着扫把到处飞行大概比较困难，但是在陆地上驰骋应该要容易很多。这款扫把自行车就可以满足你的愿望。

它将一个扫把与自行车整合到了一起，扫把可以绕着车座底部的轴转动，通过机械结构的传动来控制控制自行车的方向。因此就可以省去车把，而直接通过扫把来控制前进的方向。

可以行使在铁轨上的汽车

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款尽集约之能事的单人概念汽车通过汽车底部的两个轮子以及旁侧伸出的带小轮的支架保持平衡，小轮和汽车之间的距离正好等于铁轨的宽度。如此一来，你就可以直接将汽车开到铁路上，省去了找停车位而错过火车的烦恼，为那些需要转乘火车上班的白领提供便利。毕竟，两列火车之间会留有5英里的安全间隔，这款汽车就可以行使在两辆火车之间，不必跟大家一起挤占公路，可以在一定程度上解决交通拥堵问题。

设计师：Matthias Pugin
