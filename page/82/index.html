环保婴儿床

发表于 2012-01-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

由于近来对因为塑料制品发生的事故频频见诸于报端，我也开始特别关注生活中可能对自己孩子构成伤害的塑料制品。聚氯乙烯和PVC几乎充斥着整个生活，无数的日常消费品都是由这些塑料制成的，甚至包括婴儿的玩具和长牙齿时咬的塑料环。这些塑料产品不仅不可回收而且还有毒。邻苯二甲酸酯是一种毒性很大的化学物质，可是它却用来作为塑料玩具的软化剂，例如橡胶鸭子、软塑料书。这些化学物质已被证明与生殖障碍、精子量过低、早熟以及癌症有一定联系。

想一下，您的孩子整天咬来啃去的那些玩具竟然含有这样的化学物质，这是多么可怕的事情？很可能在他们玩玩具的时候，这些有害的化学物质已经进入他们肚子里了。

我正在致力于这样一件事情——寻找、购买、派送那些不含有这类有毒物质的商品。其实很简单。很多公司，包括沃尔码，都在积极地承担环保责任，他们都会主动地推荐一些环保商品。

比如这款摇篮就是完全由纸板制作的产品，它由一家以色列公司制造。

这种环保摇篮质量绝对可靠。这家以色列公司用其特有的技术保证了其产品均符合欧洲关于婴儿床和婴儿摇篮的所有安全标准。最好的地方在于你不需要任何工具来组装这个摇篮，而且它可以折叠储藏，这样你就可以很方便的去郊外游玩了。

既然这东西这么好，而且又很环保，那为什么不给那些即将为人父母的朋友准备一套呢？事情虽小，但是却会对我们的健康和环境产生很大的影响……

环保口香糖包装

发表于 2012-01-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

做过街道清洁义工的朋友们，肯定都会对口香糖残渣的处理工作感到头疼。这款环保口香糖包装便是通过特别的包装，来防止口香糖随意丢弃的情况发生。设计师在桶装口香糖的外包装上附着了一圈包装纸，这些纸片已经预先做好了切割工作。如此一来，使用者即可轻松撕下包装纸，将废弃的口香糖吐到纸上，包裹起来，防止它黏在其他地方。此时，若是身旁没有垃圾桶，使用者还可以将包裹好的口香糖通过包装底部的十字开口，将其塞入储存槽中，等到发现垃圾桶的时候，再统一将口香糖残渣倾倒出去。

设计：Gonglue Jiang

无线烹饪盘

发表于 2012-01-07 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一套构思非常新颖的烹饪器具，它并不像传统产品一样采用相对固定的模式，而是被设计成了一个个独立的烹饪盘。其实它是利用了谐振感应的原理来发热。在台面下面有传导线圈，每个盘子中有接收受线圈，因此，烹饪盘可以随意被放置于台面的任意位置，而不需要被固定住。这样一来，就可以更加灵活的安排厨房空间了。

另外，每个烹饪盘都可以直接用水清洗。清洗干净后还可以像盘子一样摞起来，非常节约空间。

设计师：David Barry & Laurence Finnegan
