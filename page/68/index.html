折叠车轮

发表于 2012-01-22 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们专利之家曾经为大家介绍过像“环形折叠自行车”和“手提箱自行车”这样非常便携的折叠自行车。虽然都有着非常巧妙的结构设计，但是由于车轮是不可折叠的，所以它们的变形都受到了车轮的限制。

而我们今天为大家介绍的这款车轮就能够很好的解决这个问题。它是由一位来自英国皇家艺术学院的学生设计的。车轮的整体采用碳纤维制成，通过两根交叉的辐条来支撑轮胎，同时辅助轮胎的折叠。

如果这样的轮胎能够大量生产，我们就能够看到更多更为巧妙的折叠自行车的设计了。

地铁应急口罩

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

发生火灾逃生时的重要注意事项之一就是要用湿毛巾捂住口鼻再往外跑，不过如果在人多拥挤的地铁里突遇火灾，想要找到水或是湿毛巾却不容易。这款应急口罩就是为了应对地铁火宅设计的，它被安放在地铁列车的车门附近。当突遇紧急事件时，乘客只需将其从中间掰断，内置水囊便会破裂润湿口罩棉布，便捷地为使用者提供安全保护。

设计：Han Chi-hoon & Kim Yeon-soo

电动汽车充电站合集

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着世界能源问题的日益严峻，越来越多的国家开始重视电动力汽车及其配套设备的研发与生产。电动力汽车不仅使用的能源更加便宜，对环境污染的影响也更加微小，但是或许对电力系统提出了更高的要求。太阳能充电站可以很好地缓解这一矛盾，这里我们为大家介绍了9个各有特点的充电站，请您欣赏。

这是位于德国法兰克福的一座免费充电站，它可以为各种小型电动汽车提供免费的能源。它的站台是圆柱形的，以便在闹市区能节省一部分公共空间。

这座太阳能充电站设计的巧妙之处在于它的组装部件都是模块化生产的，便于快速安装和拆卸。此外，它还具备了电脑操作系统，安装了开源软件，可以接入互联网访问客户信息。

设计：Tom Miller

太阳能电站使用能源十分环保。纽约的一个环保组织利用废弃的集装箱，改造出这座充电站，希望充分发挥太阳能电站的环保主题。

日本丰田工业公司推出的这座充电站，可以在充电车辆过多，能源用完时，自动连接上普通电网获取能源，保证电能充足。

这座外形类似于飞船的充电站，是由普通的加油站改造而成的。为了保证外型整洁美观，一些基础设施如咖啡厅、服务区都位于地下二层。

设计：Geotectura

这个充电站为了满足市民多方面的需求，还特意增加了可以为便携产品，如笔记本电脑、iPod、小型电动自行车等充电的功能和空间。

设计：Michael Scherer

为了大规模地普及清洁能源，设计师将太阳能板改造后安置在了停车场。在给电动车充电的同时，也可以在夜间为停车场提供照明。

这座一体电站，同时使用太阳能和风能发电。它所产生的电能不仅可以为车辆充电，还可以为城市提供照明。

设计： Jaewon Sim

这座太阳能充电站是由废弃加油站改造而成的。由于过往车辆很多，所以将采用最新技术，将充电时间缩短为20分钟左右，提高效率。

旅行充电盒子

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 电池

乘坐长途飞机百无聊赖的时候，需要手持设备打发时间。如果这个时候手持设备的电力耗尽，你肯定会很郁闷。这款旅行充电盒子可以为你解决烦恼。它能通过飞机上的耳机接口为手持设备充电，如MP3播放器、手机、PDA、游戏机等等。它兼容所有的标准USB接口。如果飞机上没有合适的耳机插口，它还能将肚子里的AAA电池电量通过连接线转移到手持设备上。
