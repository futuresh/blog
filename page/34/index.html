车载便携马桶

发表于 2012-02-20 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们之前介绍过的一款“凝胶尿壶”虽然可以在一定程度上缓解因为堵车造成的如厕不便，但是，那东西似乎只适合男性使用。

而来自日本的Kaneko Sangyo公司计划在“东京车展”上展出一款更为方便的车载便携马桶。该产品体积非常小巧，可以直接放在车内，以供应急使用。

从图片上看，它应该是通过一种类似于尿布的高吸水材料将排泄物吸附住，然后再通过塑料袋收集起来。

具体的情况恐怕还得等到产品正式展出才能揭晓。

可溶性吊牌

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 环保&公益创意

据统计，全世界每年有13亿棵树被砍伐用来制造各类纸张，每分钟有近一个足球场面积大小的森林被毁坏。一位设计师受到常常被浪费的服装吊牌启发，设计了替代纸质吊牌的新型吊牌。这款吊牌采用固态活性剂制成，不仅节约了纸张，它还可以溶入水中当作洗衣剂，用来洗去为了保持新衣服质地和手感而添加的化学制剂，真可谓一举两得。

设计：Sangbyum Kim

用iPod包装盒制成的音箱

为了提倡环保，减少包装袋带来的污染，我们国家也即将要禁止超市免费发放塑料袋了。虽然对于一些常规的产品包装无法通过硬性禁止的方式来达到节约资源的目的，但是充分的加以利用，同样可以收到不错的效果。

车载旋转屏

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款专为汽车设计的液晶屏。它大小为7英寸，采用16：9的宽屏设计，分辨率为800×480。

为了方便携带，它还借鉴了笔记本的折叠设计，并且配置了可旋转的支架，可以进行90度旋转。该产品还内置摄像头和麦克风，可以直接进行视频通话。为了方便用户进行控制，它还采用了触摸屏的设计，可以直接通过手指的触控来实现基本的操作。

为了减少电线的数量，它将电源线和数据线整合为一根USB线，有效的避免了电线过多产生的凌乱。

超酷环保汽车

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款由德国Loremo AG汽车公司最新推出的概念汽车。与传统的环保汽车不同，这款车的环保概念并不是体现在混合动力系统上，而是通过减轻车体重量来降低油耗、提高燃料利用率从而达到环保的目的。该车采用两缸发动机驱动，燃油利用率达到150英里/加仑（百公里耗油约为1.56升），最高时速可达160千米/小时！而如此之高的柴油利用率都得归功于其超轻的车身重量，这样一部汽车的重量仅为450千克！

另外，这款车的造型也非常酷，它摒弃了传统汽车两边开门的设计，而采用了前后两向的上掀式设计，更使得这款车充满了现代的时尚气息。

该车预计于2009年在欧洲、中国、印度上市销售，售价约为13,000欧元。

环保节能开关插座

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 环保&公益创意

即便电器停止运转，只要电源与插座依旧保持连接，仍然会有少量的电量消耗。积少成多，也是一笔不小的损失。所以，设计师们便绞尽脑汁，想要通过直观、醒目的方式提醒我们在电器使用完毕之后彻底切断电源。这款开关插座便将插座与电源开关结合起来进行设计，它鼓励您使用这个电源终极开关，而不是贪图省事，仅按下电器开关了事。

设计：Daun Yun & Junil Choi

环保铅笔制造机

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 环保&公益创意

办公室的废纸处理一般只有两个途径，若不是交给碎纸机销毁，便是将只有单面用过的纸张用于复印、记录非重要文件。而一个中国设计团队却为办公室的废纸找到了一个新用途，让它们成为制作铅笔的新原料。这款办公室废纸处理机可以将废纸回收利用起来，用它们来包裹铅芯，制成废纸铅笔。而铅芯安装盒以及胶水承载盒都会自动检测内部存量，若是缺少铅芯、胶水，它们便会自动弹出，提示使用者更新耗材。

设计：Chengzhu Ruan, Yuanyuan Liu, Xinwei Yuan & Chao Chen

给地球降温的喷雾

发表于 2012-02-18 | 更新于 2018-07-24 | 分类于 环保&公益创意

众所周知，喷雾可以起到降温的作用，但是我们今天为大家介绍的这个设计可不是用来给人体降温的“便携喷雾器”，而是用来给地球降温的！

这个设想来自于美国国家大气研究中心的一位研究员。他希望组建一个由1500艘船组成的舰队，每艘船上都配有风力喷雾，可以源源不断的将海水喷出。这样就可以产生更多的雾气，甚至在地球表面形成一层雾气屏障，从而有效的阻挡红外线，最终起到为地球降温的作用。

不过，这个大胆的设想是需要有巨大的资金做后盾的，而且其作用也只是暂时性的。真要给地球降温，还是要从环保做起。
