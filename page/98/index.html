节水水槽

发表于 2011-07-19 | 更新于 2018-07-24 | 分类于 环保&公益创意

在宣传节水的公益广告中通常都会提到一点，那就是在刷牙或洗脸的间隙记得关上水龙头，避免浪费。但还是有很多人难以改掉这个“恶习”。而我们今天为大家介绍的这款设计就让你不必主动关掉水龙头却仍然可以避免浪费水资源。

这款设计在水槽下方增加了一个储水器，而入口则设在了水龙头的正下方。平时入口是关闭的，当有水流直接流下时，压力就会迫使入口的盖子打开。也就是说，即使你在刷牙、洗脸的间隙不关闭水龙头，这些干净的水仍然可以被储存到储水器中。而当用手接住水的时候，由于缺少压力，盖子又会自动关闭，避免脏水流入储水器。

储水器的外壳还采用了半透明的设计，方便用户了解水位。

设计师：Woo Sunhye

健身河水净化器

发表于 2011-07-19 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款新型河水净化器，而驱动净水器的工作能源则来源于平台上的健身器械。在大家使用健身器械锻炼身体的时候，净水器上的水泵就会利用器械所发电力进行工作，河水会灌入净水器的四个过滤器进行净化。整个设备十分环保，一举两得。

设计：Jakub Szczęsny 【设计师网站】

火柴日历

发表于 2011-07-19 | 更新于 2018-07-24 | 分类于 环保&公益创意

日历过期后就不能再用了，或许有人可以将废旧日历进行回收利用，但是毕竟还是废物利用。而我们今天为大家介绍的这款日历则具备了另外一个有趣的功能。

这款日历其实是由火柴组成的，每根火柴都代表了一天，这样一来，即使是废旧的日历仍然具有火柴的功能。当然，点燃火柴的同时也可以提醒我们，时间的流逝正如这跟火柴一样，烧完了就不再回来。

奔驰防瞌睡系统

发表于 2011-07-18 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

长时间驾车的司机很多时候都难以觉察到自己已经很困了，这样就就很容易引发交通事故。所以，各大汽车厂商也都纷纷加大这方面的投入，帮助司机及时意识到自己已经很困，应该停下来休息一下。例如，“丰田防瞌睡系统”就是通过摄像头来实现这种功能的。

而奔驰则会在旗下的E级轿车上安装一款更加先进的防瞌睡系统。研究人员首先研究驾驶员的驾驶行为与困意之间的联系，然后将具有较高相关度的动作输入到系统中。一旦系统探测到司机有类似的驾驶行为，就会提醒其停下车休息一下，避免因此产生交通事故。

小巧的电动车

发表于 2011-07-13 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

为了减少污染物的排放，节约能源，各大汽车厂商都纷纷推出了各自的混合动力车型甚至纯电力汽车，很多城市也都在大力推广“自行车租赁系统”。但是，很多时候自行车还是无法代替汽车的，而要完全更换成环保的汽车也不是短时间内能够完成的。为此，来自美国麻省理工学院的科研人员便开发了这样一款便于出租的电动车。

这款车的最大亮点在于它小巧的车身：仅容纳两人的驾驶室为车体瘦身提供了保障；另外，它还借鉴了超市购物车的思路，将电动车设计成可折叠的结构，如此一来，原本就很小巧的车身就能够节约更多的泊车空间。一个普通的停车位可以容纳6～8辆这种电动车。而且，该车还配有科研人员专门设计的全向车轮，能够很好的适应城市中狭小的路段。再以全电力驱动的马达，真正实现了环保的要求。

如果能够配合着一套很好的租赁系统，相信这种车会很快在大都市里面普及开来。
