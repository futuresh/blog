急救冷冻头盔

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这是一款急救冷冻头盔，它内部含有两个药包，分别为水和硝铵，当两者混合的时候，便会引发反应，从而能在头盔内部制造低温，保护使用者的头部。这项发明源于一次事故。设计者在与朋友们比拼篮球的时候，其中一位朋友的头部受伤。他们迅速找到冰袋为其消肿，但冰袋使用起来却十分麻烦。因为冰袋温度太冷，便很难长期持握冰袋让其固定在伤处。设计者便急中生智，将冰块倒入摩托车头盔，然后再让伤者带上。如此一来，便滋生了这一冷冻头盔的设计构想。经过反复试验，这款冷冻头盔除了能够制造低温以外，还附加了摄像头、GPS系统以及蓝牙设备，预计推出时候的售价为815美元左右。

设计：Jullian Preston-Powers

能量提示垃圾袋

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

废弃的垃圾中有不少可以利用的生物能，这些能源如果用来发电将是非常好的资源循环再利用途径。不过目前这种技术仍在探索之中，在德国，仅有40%的有机垃圾能够得到进一步的处理而非直接被填埋掉。这款可降解的垃圾袋，由PLA（以淀粉或甘蔗为原料）材料制成。它可以近似衡量垃圾中所含的能量，并用这些能量能够做多少事情（如可以烧开100杯水、为手机充电300次等等）直观地在袋子表面表示出来。由此提醒人们能够重视有机垃圾的回收与处理，充分利用资源，提高环保意识。

设计：Ahhaproject

大众超轻概念车

发表于 2012-02-04 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

通过降低车身重量来达到节约燃料目的的汽车我们已经多次为大家介绍过了。这似乎逐渐成为了一种趋势，而日前，著名的大众汽车公司也推出了自己的一款超轻便概念车。

这款汽车车身重量为660磅，油耗为282英里/加仑。采用单缸发动机驱动，但是日后有可能改进成为双缸混合动力发动机。该车采用双排座椅设计，每排只能乘坐一人。安全方面配置有安全气囊、ABS和稳定控制系统。空调则为可选配置。

在油价日益高涨的今天，如此经济实惠的汽车应该会受到不少人的青睐。

该车为限量版，计划于2010年上市。
