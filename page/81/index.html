水上过山车

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

水上滑梯和过山车这两种娱乐项目相信很多朋友都见过甚至玩过。不过将这两个项目融合到一起的水上过山车恐怕就没有几个人见过了。

这款来自德国的水上过山车是一个非常刺激的娱乐项目，它可以让你用通过水上滑梯的方式来体验过山车的刺激。

游戏开始之前，游客先被放入到一个准备仓内，而当游戏开始后，游客脚下的底板就会突然被抽开，这时游客就会掉入到一个全密封的管道中，而由于管道的前端是垂直的，因此在这一过程中游客就会处于自由落体的状态，从而获得足够的速度使得其能够顺利地通过后面的一段类似于过山车的环形弯道，最终像普通的水上滑梯一样滑入到水池中。

套装菜板

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 环保&公益创意

我们之前曾经多次为大家介绍过“可以折叠的菜板”，通过这种结构就可以将切好的菜直接倒入盘中或者锅中了。其实，除此之外，还有另外一种思路，那就是将切好的菜直接推入盘中。当然，普通的菜板要实现这种功能并不方便，这就要用到我们今天为大家介绍的这款设计了。

该产品由三个部分组成：两个带支架的菜板，两个盛菜的盒子，还有一个增高支架。由于两个菜板下面都配有支架，因此可以直接将盘子放在下面，切好菜后用刀将其推入盘中即可。如果要使用配套的盒子或者其他比较高的餐具则需要借助增高支架。另外，平时不用的时候还可以将这三者组合起来，不仅可以节约空间，还可以当作一个小储物空间来使用。

超轻便的单人汽车

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着油价的日益高涨，混合动力车逐渐成为一种趋势，其实除了改变动力源之外，还可以通过减小车身的重量来达到节省能源的目的。例如我们之前介绍的“超酷环保汽车”就是这样的设计。不过今天的这款汽车则是将这种理念发挥到了极致，因为它的重量仅为150千克。

为了尽量的减少车身的重量，节省能源。这款名为Huvo的电动汽车采用了单座的设计。车身则由钢、铝、碳纤维、强化塑料以及ABS树脂制成。据该公司CEO的介绍，其他公司也有过类似的计划，但是他们最终生产出来的成品却都在300千克左右。

相关产品：“小巧的电动车”

壁挂式洗衣机

这样一个东西出现在卫生间里，恐怕大部分人都会误以为是“手纸架”或是“香皂盒”。不过它的真实用途恐怕会出乎大家的意料之外。因为它其实是一款壁挂式洗衣机！

感应灯座

发表于 2012-01-08 | 更新于 2018-07-24 | 分类于 环保&公益创意

如果你总是在出门时忘记关灯，那电费肯定少不了。可是自己又总也不长记性，怎么办呢？这款感应插座就可以帮你解决这个问题。即使你忘记关灯了，在你离开房间后，过几分钟，灯就会自动关闭。

这款灯座内置有感应器，能够探测到方圆12英尺内的动静，所以，一般来说，只要你在房间里，感应器都可以探测到。只要4分钟内没有探测到有任何物体移动就会自动切断电源。是不是很方便？一定能帮你节约大量的电费开支，到时候可别忘了我吃饭啊。

该产品安装非常简单。首先，把你原来老灯泡扭下来，把这款灯座直接扭到原来的灯座上，然后把灯泡安到这款灯座上，就可以了。

不过我还想到一个问题，万一房间里面有只老鼠，或者苍蝇蚊子的，会不会被感应器感应到呢？

这款灯座售价为30美元。
