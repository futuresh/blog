全球最大的潮汐发电机

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 电池

一家名为Marine Current Turbines的公司计划于今年8月在北爱尔兰沿海安装1.2兆瓦的巨型潮汐发电机。据称这款涡轮机是目前为止全球最大的潮汐设备。它所产生的电力能够供应大约1000户家庭的用电。涡轮机的转速非常慢，大概为每分钟10-20圈，相比而言船只推进器的转动速度则可以达到这一速度的10倍以上。而由于生活在湍急水域的海洋生物通常都比较敏捷，因此这款涡轮机对于海洋生态环境所带来的影响并不会太大。

潮汐发电机的原理与风力发电类似，只不过把风力推动改为潮汐和水流推动，由此而产生更为环保的电力。

月亮锅

电磁炉已经越来越被人们所熟知，不少家庭的日常生活中已经缺少不了它的陪伴。但是，电磁炉属于大功率的用电器，使用的时候往往要消耗许多能量。设计师想出了这个十分精明的设计：这款锅的外形如同月牙一般，可以在您使用电磁炉时放在主锅周围使用。它既可以让您同时烹饪不同的菜肴，还可以帮您节约能源，真是一举两得。

可多次使用的方便筷子

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 环保&公益创意

尽管一次性筷子在餐饮业非常盛行，但是众所周知这种产品会浪费大量的木材，非常不环保。可是很多经常外出的人却的确难以放弃这种方便但不环保的产品，毕竟出门在外不方便清洗筷子。那么有没有一举两得的办法呢？

这款新颖的花瓣筷子就给我们提供了一种很好的思路。它在筷子的外层包裹上很多层的可降解薄膜，每次用餐之后只需将薄膜撕去即可再次使用，既避免了反复冲洗筷子的麻烦，又能够很好的满足环保的要求。另外，每一层的薄膜顶端都被设计成花瓣的样子，使得整个筷子看上去就像是一支花，不仅美观，还方便撕取。

因为制作工艺独特，这种筷子可以用普通的树杈来制作，而不必专门砍伐大棵的树木，这就使得这种筷子更加环保。

设计师：Hak-Byoung Kim

USB电池

发表于 2012-01-21 | 更新于 2018-07-24 | 分类于 电池

你肯定有过这样的经历，当你正玩得起劲的时候，电动玩具突然没电了。更可恨的是那该死的充电器也找不到了。没办法，要么不玩了，要么出去买块新的电池。就算最后解决了，恐怕也会没有了继续玩的兴致，这种事情确实非常令人扫兴。如果我告诉你，现在有一种电池，不需要充电器，就可以直接插在USB插孔上充电，你是不是会觉得很神奇？没错，就是这款USB电池。

哎，我已经不只一次地提到“21世纪是USB的世纪！”所以以后出现任何跟USB有关的东西，只要记住小编这句话就不会觉得过于出乎意料了。

这款电池每包里面有两个，均为镍氢电池，1.2V,1300毫安时。目前只有7号5号电池一款，不过陆续将会推出5号7号电池甚至手机电池。

打眼一看，跟普通电池没有什么区别，甚至可以说是完全一样。

不过，电池的正极实际上是一个帽，当你将这个帽拆下就会看到这款电池的秘密所在了，一个完整的USB插头。

帽并不能与电池本身完全分开，而是用连在一起，以免您不小心弄丢了帽，那这个电池就彻底报废了。

要充电时，只需要将电池插到任意一个USB接口里即可。充电时，绿灯将会亮起，当充电达到90%时，LED灯就会不停闪烁。当全部充满时，就会自动熄灭。（通常用USB接口充电大概需要5小时即可充满）。

当然，也可以像普通的充电电池那样，用充电器来充电。

总之，这款电池确实是非常方便实用。

方便残疾人使用的马桶

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

现代马桶的设计似乎并没有太多的为残疾人考虑，对于那些坐在轮椅上的残疾人而言，要使用坐便马桶并不是一件轻松的事情。这款马桶的设计则可以通过两侧的把手，让残疾人直接拉动把手进而从轮椅上滑到马桶上。而且水龙头就设计在正前方，方便之后，可以直接洗手。

而且这款马桶对于普通人而言同样适用，水槽处还被做成了靠背的样子，对于普通人而言，也可以获得更为舒适的体验。

更重要的是这款设计较之目前市面上常见的残疾人专用马桶要节约75%的空间。

设计师：Changduk Kim & Youngki Hong
