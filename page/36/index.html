李维斯方便晾干牛仔裤

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

烘干衣裤常常要浪费不少能源，这款李维斯的牛仔裤用设计的实际行动鼓励人们自然晾干衣裤，节约能源。这款牛仔裤背后的两个兜里设计了两条颜色鲜明的红色带子，穿着的时候可以把它们当成腰带或是绑在后面做装饰。经过洗涤后，这两条带子则可以系到前面的裤扣上，方便将裤子挂在任意的地方晾干。这项红色带子的节能运动，不仅引领潮流呼吁人们享受低碳生活，也凸显了商家的促销能力和社会责任感。

寻找章鱼的脚

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

这个设计乍看起来似乎没有什么实际用途，就是一个没有脚的章鱼，把空瓶子放在插到口上，帮它组成完整的身躯，这样它就可以漂浮在水面上了。

但是如果给小朋友一个这样的“玩具”，它或许就可以帮忙清理海滩上的废弃塑料瓶了。

通过这种寓教于乐的方式来教育孩子，更容易从小培养起他们的环保意识。

设计师：Jaehyung Hong

雨水收集器

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 环保&公益创意

相对于石油、煤炭等化石燃料而言，水的重要性要大得多，但是很多人却并未养成良好的节约用水的习惯。其实，除了可以通过改变生活中的一些用水习惯来达到节约用水的目的之外，还可以改变一些现有的设施来达到同样的目的。比如我们今天为大家介绍的这款雨水收集器就提供了一种非常好的思路。

它的一端可以插入现有的建筑物的排水管，另一端则可以接上废旧的饮料瓶。这样一来，留下的雨水就可以被收集到矿泉水瓶中。人们可以利用这些水来浇花，在外面玩得满手是脏东西的小朋友也可以用瓶子里的水来洗手。

实景驾驶模拟器

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款由昆士兰大学研究者担纲设计、研究的实景驾驶模拟器。它由按照实体汽车模样设置的驾驶台、虚拟路况屏幕，以及相应的电脑处理系统组成。驾驶者需要根据虚拟路况操作汽车，而驾驶台上的操作将会反馈到电脑系统中，电脑系统再将实体驾驶感受反应到操作台上。例如，当驾驶者高速过弯的时候， 汽车也会有相应的倾斜，以帮助驾驶者感受到更加逼真的驾驶感觉。这一系统可以真实地模拟各种状况下的驾驶行为，以了解交通事故的造成原因，让驾驶者避免各种驾驶过程中发生的危险行为。

设计：CARRS-Q

人力滑翔艇

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

如果你总是认为所谓的“超级自行车”之类的东西仅仅是发明，但并不实用，那么这款即将上市的人力滑翔艇应该可以改变您一贯的看法。目前来说，帆板是最实用的不需要任何燃料的水上运动项目，但是如果没有风，帆板也就失去作用。我们现在为大家介绍的这款新型的滑翔艇就很好的解决了这个问题，在锻炼身体的同时可以得到很多乐趣，更重要的是，它不需要风。

这款滑翔艇售价为495.95美元，重量仅为26磅（11.8千克），非常容易携带。最快时速达到17英里（27.4公里）。您只需要站在上面，不停的上下摇动把手即可推动滑翔艇前进。据说这种艇驾驶起来非常轻松，而且更重要的是，它可以为你赚来不少的眼球啊。

导航井盖

发表于 2012-02-17 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

在陌生城市的街道上迷失了方向，而又不好意思或者不方便问路的时候，井盖可以帮你忙。这款导航井盖可以显示附近火车站或汽车站的信息，你只需要用脚按动井盖按钮就可以查看。而井盖的工作能源则来源于从天而降的雨水，雨水透过井盖缝隙流入下水道的时候，就会带动井盖下方的风扇旋转，从而为井盖提供工作能源。当然，正是因此，这款井盖就需要安置在雨水丰沛的南部城市。另外，还有一个前提，那就是治安良好，否则警察们可就有得忙活了！

设计：Cheolyeon Jo & Youngsun Lee
