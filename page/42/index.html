折叠的环保袋子

发表于 2012-02-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

塑料袋虽然大大方便了我们的生活，但是由于其难以降解的特性，会对环境造成严重的破坏。因此，随着人们环保意识的不断增强，可反复循环循环使用的袋子就又重新流行开来。

不过很多布袋子确实不怎么美观，尤其是对一些爱美的女性而言，更是难以接受。这款来自日本的产品或许就能够提供一个折衷点。

它采用环保材料制成，平时不用的时候可以折叠在一个圆筒内；需要的时候展开来，同时圆筒也可以作为提手减轻袋子对手部的压强。由于圆筒外部配备有磁铁，平时还可以将它吸附在家里的冰箱上，节约空间。
该产品有多种颜色可选

驾车小助手

发表于 2012-02-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

麻省理工大学的研究者们与奥迪公司合作研发这款驾车小助手。它乖乖地坐在仪表盘上方，静静观察司机的驾驶状况，并通过脸部表情和符号提示帮助司机驾车。例如，当它观察到司机疲劳驾驶的时候，会对其进行提示，并自动降低车速；它还会给司机提示路况信息，指出建议行车路线，充当导航系统。可爱的小助手可比冷冰冰的GPS要亲切得多呢！该产品的研究目标是实现小助手自动驾车，让司机朋友彻底地解放。

太阳能折叠摩托车

发表于 2012-02-12 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

“折叠自行车”的设计已经非常普遍，其中也有许多非常经典的创意。但是折叠摩托车相对而言就比较少见了。我们今天为大家介绍的就是一款环保的折叠摩托车。

这款摩托车可以被折叠成一个梭形，不仅如此，它还充分融入了环保的元素，折叠后其外表就会完全被太阳能电池覆盖，可以通过吸收太阳光来获得动力。为了充分吸收太阳光，它还配有一款类似于太阳灶的底座，可以起到聚光的作用，从而吸收更多的太阳光。当阳光不足时，还可以通过普通的交流电源进行充电。

设计师：Cao Dawei

后花园办公室

发表于 2012-02-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

对于常在家庭办公的人来说，整天闷在家中难免会感到乏味。而这款后花园办公室却可以临时充当“世外桃源”的角色。这个办公室的外形是一个直径为3米的圆球。它表面覆盖着成型胶合板，内层应用可回收玻璃纤维的保温隔热材料。它与乐高玩具类似，可以拆解成若干块，方便安装者将其转移、搬运。办公室的门采用上悬式开启方式，内部设有一个巨大的环形办公桌，还有符合人体工程学的内饰、电源插座、数据端口和电热取暖等设施。除此之外，顶部的天窗还可以给室内提供较好的采光和通风条件。

可伸缩的燃气灶

发表于 2012-02-12 | 更新于 2018-07-24 | 分类于 环保&公益创意

房价的居高不下的确令不少人感到头痛，所以，能够节约空间的多功能家居用品也就成了一种设计趋势。这不，这一次又轮到了燃气灶。

这款燃气灶的最大特点是它是可以伸缩的。当做完饭后，锅架部分可以直接收缩到燃气灶内，这样一来，整个灶台就变成了一个平板，可以用作其他用途。同时，它的底座还采用了黑色的陶瓷，从而使得灶台变得非常容易打扫，只要轻轻一擦即可光洁如新。

设计师：Fisher and Paykel
