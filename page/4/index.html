鲜蔬保鲜罩

发表于 2013-07-10 | 更新于 2018-07-24 | 分类于 都市农业

切了一半剩下的新鲜水果蔬菜怎么继续保持新鲜呢？放在冰箱里，再铺上一层保鲜膜？这个方法普遍适用但又不那么绿色环保。这款硅胶保鲜罩无毒无害，它利用水果蔬菜本身的汁液密封空气出口，保持新鲜。使用方法极其简单：切下来的水果或蔬菜只需对准开口轻轻一按，便扣上了“大帽子”；从大到小，四种不同的直径罩子适合厨房里的多种水果。有了它，可以天天吃新鲜食物咯！

垂直厨房

发表于 2013-07-10 | 更新于 2018-07-24 | 分类于 都市农业

面对日益紧张的住宅环境，很多人“愿意”以牺牲厨房或浴室面积为代价获取更宽敞的卧室或客厅。这套所谓的垂直厨房将厨房常用工具和设备集合在一起，通过旋转厚薄不一的面板而选择不同的工具，如水槽、切菜板、电磁炉、冰箱等等。此外，水资源的循环、各功能模块间资源共享与交换以及面板外侧的私人小果园等都是该厨房的设计亮点，等它上市的那一天估计会很抢手吧！

设计师：Massimo Facchinetti

Blacktrail电动自行车

发表于 2013-07-10 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

电动自行车一直以来都凭借其清洁的骑车环境受到人们的好评，但是有限的速度也大大制约了它的发展前景。于是，一个高速自行车的出现就显得非常重要。针对这种情况，德国的PG自行车制造商设计出了一款既环保又迅速的Blacktrail电动自行车。该款自行车以先进材料碳、 镁、 钛等作为结构物质，大大减小了整体的重量；同时又以CM 48V 锂离子电池组作为发动引擎，使其成为了世界上最快的电动自行车。

适合藻类生长的有机生物混凝土

发表于 2013-07-10 | 更新于 2018-07-24 | 分类于 都市农业

爬满藤类植物的墙面在夏天与秋天会格外的美丽，但它们的根茎会插进墙体的缝隙中，久而久之，这些缝隙便成为了害虫们的大本营。而即便是当代的一些人工墙体花园在施工与维护方面也大有欠缺。西班牙的加泰罗尼亚理工大学（Universitat Politècnica de Catalunya）的研究者研制了一种适合藻类、菌类和藓类生长但又不会破坏墙体结构的有机生物混凝土，不同于传统的硅酸盐水泥，这种水泥的主要含量是磷酸镁，这让墙面呈适合上述植物生长的酸性环境，非常适合温和且潮湿的地中海气候。此外，实际建造中所用的生物水泥内含防水层，不必担心下雨天墙体湿透而带来的麻烦。

监测植物生长环境的智能助手

发表于 2013-07-10 | 更新于 2018-07-24 | 分类于 都市农业

在家里种植花草或者蔬菜不仅仅需要耐心，更需要了解大量的栽培知识。如果您还是个植物小白，那么这根智能探测棒便是您的最佳助手。插在土壤里的探测棒会收集各种数据，包括太阳光照强度、湿度、温度、土壤营养程度等等，再将其发送至手机端的应用程序中，经过简要分析之后，一些提示如何时浇花、何时施肥等便会推送至桌面，时刻提醒您不要顾着工作而忘了自己的小花园。相信有了它，您一定会收获丰富的果实的！

垂直“森林”

发表于 2013-07-09 | 更新于 2018-07-24 | 分类于 都市农业

现代化都市正一步步蚕食着自然界宝贵的绿色资源，在一些大城市想要找到适宜散步的公园或绿化区已是相当困难。不过，在米兰，一座号称“垂直森林”的住宅楼马上就要诞生了，经过特殊设计的阳台可用来“种植”多种树木、灌木或花草，预计在年底正式建成之日，将会有900棵大树、5000丛灌木以及1.1万余种花草在这个钢筋混凝土“树干”上繁荣生长，为整个城市带去勃勃生机！

盆栽“树”

发表于 2013-07-09 | 更新于 2018-07-24 | 分类于 都市农业

闲来无事之时，不妨在阳台上种点花草或者自食蔬菜，养眼养胃的同时又陶冶了情操。这款亮相于本届米兰设计周的盆栽“树”出自美国一位年轻设计师之手，它科幻、有机的外形让人眼前一亮：卵形“巢穴”为植物提供栖息地，两个“脆弱”支架的功能分别是为植物提供养分和排出多余水分。由于采用无土栽培，您只需在“树根”处注入营养液，再用水泵将其输送至单个巢穴即可，日常照料十分方便。

玻璃农舍

发表于 2013-07-09 | 更新于 2018-07-24 | 分类于 都市农业

这座玻璃农舍位于荷兰的Schijndel小镇，是为了重现与纪念二次世界大战之前当地的繁荣市井而建造。不过，建筑师并非单纯地“重建”一座红砖青瓦的农舍，而是用玻璃、钢铁来完成这个项目：他们将农舍的外墙与屋顶图像“印”在玻璃上，仿佛是农舍被玻璃外罩所覆盖；而走近之后我们甚至能清楚地看到红砖的纹理！当然了，玻璃墙上还是会留出一些透明区域让人们一探里面的究竟。建筑师希望通过这种看似“虚假”的装饰主义引发人们对复古建筑的反思，并唤起当地居民对历史的回忆。
