胸针垃圾袋

发表于 2012-02-03 | 更新于 2018-07-24 | 分类于 环保&公益创意

胸针可以对服装起到画龙点睛的作用，因此很多人都会非常注意胸针的款式以及与特定服装的搭配。然而我们今天为大家介绍的这款胸针则更多的是考虑了环保的因素。

它的实际作用其实是一款小垃圾袋。在“胸针”外侧配有一个圆环，可以将透明的垃圾袋直接套到圆环上。吃过的口香糖等体积较小的垃圾就可以直接扔到垃圾袋里面了。

折叠候车椅

发表于 2012-02-02 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

很多城市的公交车站都有候车棚，但是一些规模较小的公交车站通常只有一个孤零零的站牌而已。而为了用更为低廉的成本为人们提供休息的场所，就有人设计出了这样一款折叠候车椅。

它的结构非常简单，与站牌或者电线杆整合到一起。需要的时候只要轻轻拉出即可使用。而且为了鼓励人们多使用公共交通工具，减少汽车尾气的排放。这款折叠椅上还特意写上了类似于“乘坐公交，保护环境！”的标语。

设计师：Catherine Pena

太阳能四轮车

隐藏车牌信息的逃逸喷雾

发表于 2012-02-02 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

新手上路，不熟悉交通路况，即便是一个小错误也很容易被隐藏着的交警部门摄像头逮个正着。若想逃过此劫，您可以考虑使用这款逃逸喷雾。这款透明喷雾可以喷洒在车牌上，此后并不影响人眼对车牌的正常观看，只不过摄像头捕捉车牌信息的时候，喷雾薄膜表面就会与闪光灯对着干，造成反光效果，影响摄像头清晰捕捉车牌信息。当然，这种做法会对交通安全造成损害，切勿滥用！
