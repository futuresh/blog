跑步机自行车

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这款自行车的驾驶动力来源于车身上安装的跑步机履带。您必须滑动自行车，然后迅速站到跑步机履带上，坚持跑动，方能让履带旋转，并带动自行车向前滑行。要想在两轮自行车上保持跑动平衡，同时又得掌握好行进的方向，可不是件容易的事呢！

保姆机器人

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这是一款由美国马萨诸塞大学的机器人专家开发的一款新型保姆机器人，他的头部是一个显示器，可以作为远程视频系统与远方的家人联系，同时他还可以让使用者通过互联网照顾家里的老人。它还可以在紧急情况发生时拨打911。

隐形墨镜

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

夏日阳光十分耀眼，在户外运动时要注意保护好眼睛，一失手把墨镜碰碎了可不得了！即便不做剧烈运动，随着城市玻璃建筑的增加，晃眼的折射光也容易刺激视觉神经。那么，你可以考虑使用这款隐形墨镜，它的功效等同于框架墨镜。有绿、红、黄、灰四种选择，各色墨镜的遮光效果不一，你可以根据具体用途选择适合自己的墨镜。

设计：Jin-young Yoon, Jun-kyo Lee & Young-ho Lee

新型氢气充气站

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

氢气作为一种清洁能源，有望被应用于汽车领域。虽然，氢气动力车全面普及还需时日，但是已经有不少科学家开始为如何储存氢气并为汽车提供“充气”服务而忙碌起来。

这款全新充气站就是其中一个非常具有创意的设计。它通过底部的海藻池产生氢气，并储存在上方的直径为10米的汽球中以备氢气动力车充气之用。每个这样的充气站可一次性为12部充满气。

竖式太阳能停车场

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

还记得威尔·史密斯主演的《机械公敌》么？想必大家对那个大型自动停车场记忆犹新吧？这款竖式太阳能停车场虽说没有电影里那么炫，但是它也能将汽车竖起来安放，节约空间，而驱动停车设备的能源来源于太阳能。除了泊车，它还能给电力汽车充电，提供绿色汽车燃料补充服务，一站两用！

超声波清洁刷子

发表于 2012-02-24 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款概念刷子应用超声波技术清洁盘子，可以有效节约用水，并能在减少使用清洁剂的情况下，将厨具清洗干净，防止厨房废水中含有过多化学污染物，增加处理难度。刷子覆盖纳米涂层，可以防止细菌在刷缝中滋生，有自清洁功能。它还搭配有一个无线充电底座，而刷子边上的显示灯可以告诉你电量的充沛程度。

设计：Jinsoo Park for Cube2 Design
