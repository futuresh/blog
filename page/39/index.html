土豆搓皮手套

发表于 2012-02-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

创意的内涵就在于，用简单的设计解决大问题。这款土豆搓皮手套就很好的体现了这一内涵。

将橡胶手套的手心面做成毛刺状，这样你就可以通过揉搓的方式将土豆皮轻松去除掉，经过实际测试，只需8秒钟就可以“搓”完一个土豆。省时省力而且节约原料。

以后您再要做土豆的时候就会说“我去搓个土豆……”，哈哈。

节电插座

发表于 2012-02-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

该款插座是专为PC机和它的一些外围设备设计的。当你打开电脑时，智能感应器就能探测到并自动开启其他的外设。

有了这种插座，那些耗电“大户”可以有效地节约不必要的用电了；您也就不需要再为那些无谓的浪费而烦恼，可以安然进入梦乡了。当然了，如果我们房间里都使用直流电源的话，鬼才用那些跟砖头一样的插座呢。

该产品即将在英国投入使用，但是我们在想，这个东西似乎对节约用电并没有多大作用。或许其他地方有人希望引进这种产品，当你这么做的时候，切记售价一定不能高于60美元。

创意酒瓶杯

发表于 2012-02-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

被倒空的酒瓶除了当废品卖掉大都能够用作什么呢？一家回收公司将废弃酒瓶重新利用起来，把一个个带有漂亮图案的酒瓶子切割、抛光、打磨，在原有基础上重新改造，让酒瓶变身杯子，继续一展所长。如此回收利用，不仅可以将酒瓶原有的装饰继续保持下来，还能让其另做它用，更有创意！

便携洗衣机

发表于 2012-02-15 | 更新于 2018-07-24 | 分类于 环保&公益创意

现代人的住房面积正在逐渐缩小，小户型也日渐流行起来。因此也就诞生了不少专门针对小户型家庭设计的产品。这款洗衣机就是专为居住空间狭小的朋友设计的。

它不仅体积很小，而且为了方便使用者在家里移动洗衣机，还特意设计上了一个把手，这样一来就可以像拉手提箱一样移动洗衣机了。有了这项“便携”的功能，也就不必为没有地方摆放洗衣机而发愁了。只要将它直接放到水槽旁边接通水电即可开始使用。使用完毕还可以将它放到空闲的地方。不仅方便实用，而且节约空间。
