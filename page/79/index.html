塑料瓶衣架

发表于 2012-01-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

“寻找章鱼的脚”那款设计或许能够起到一定的教育意义，但是却未必有实际的用途。那么，能不能换种思路让废旧的塑料瓶发挥更大的作用呢？

这款新颖的衣架与那款用来教育环保的“章鱼”相同，主体设计都是没有“脚”的，而当你将两个空的矿泉水瓶插上去之后，它就变成了一个衣架。用这样的衣架来挂衣服，不仅环保，而且由于塑料瓶较粗，受力面积较大，还不容易使衣服变形。

设计师：Xuan Yu

竹子+塑料瓶=运动服

发表于 2012-01-11 | 更新于 2018-07-24 | 分类于 环保&公益创意

如果把竹子与塑料瓶放在你面前你能够想象它们竟然可以用来制作透气性绝佳的运动服吗？

这是一家名为Nulethics公司的最新产品。他们将竹子加热并研磨成为纳米颗粒，同时将回收的废弃塑料瓶制成高质量的聚酯纱线，然后将二者混合便制成了这种非常舒适的透气面料，并且加工成运动服。

据称，通过这种工艺制作运动服，不仅可以提高废弃塑料瓶的回收利用率，而且与传统工艺相比还能够将整个制作过程中的能源和化学品的消耗量减少80%！

可爱的助力扶手配件

发表于 2012-01-11 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

对于居住在山城的老人家来说，外出采购日常生活用品实属不易，那一层层陡峭的台阶考验着老人肌肉、关节的灵便程度，即便侧边有扶手作为助力辅助，返家时候手里提着的袋子依旧会增加老人的行走难度。所以，设计师给扶手设计了这款卡通助力配件，用鲜亮的颜色和结实的材料将一只卡通小鸟固定在栏杆上，如此一来，老人便可以在借助扶手向上攀爬的时候，将袋子挂在这个卡通配件上，减轻负担，更加轻松地走完台阶。

设计：Kim Bo-kyung & Baek Eun-ha

概念家庭菜园

发表于 2012-01-11 | 更新于 2018-07-24 | 分类于 都市农业

都市生活让我们远离了农田，这款概念家庭菜园让您在家中也可以实现栽种蔬菜的愿望。这款装置采用无土栽培技术来种植蔬菜。整个装置的下部设有喷嘴，用来对植物的根部喷水或是喷洒营养液。同时，它的内部还设有水循环系统，以保持蔬菜的新鲜。它需要使用外部电源为水泵和喷嘴供电，同时还配有一块备用电池以供不时之需。在家中就可以享受种植的乐趣，还可以吃上亲手栽培的无污染蔬菜，您动心了么？

设计：Jim Ruck

无翅目电动车

发表于 2012-01-11 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款三轮电动车，但是它的速度却有别于普通电动车。据称它可以在10秒钟之内完成0—100公里/小时的加速，最高时速可达137公里/小时。整车的流线形外观是根据空气动力学的要求，为降低风阻而设计的。这款车是由一位船舶制造师和一位生物科技工程师共同设计完成的，他们的设计目标是想让电动车像鸟儿一样利用空气滑翔，尽可能少地产生阻力。车的顶部带有一块太阳能电池板，其产生的电能将应用于热泵系统。车的内饰采用可再生材料和有机染料制成，十分环保。此外，为了降低成本和更容易被推广，这款车还加入了混合动力发动机。预计这款电动车将于今年上市。
