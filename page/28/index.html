八大另类交通工具发明

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

交通拥挤已经成为一个世界性的问题，为了解决这一问题，各种各样的交通工具层出不穷，今天我们专利之家就专门为大家选出了近期最为另类的八款交通工具让大家欣赏一下。其实他们其中有些可能没有什么实用价值，但是创意值得借鉴。同时，您或许还能得到一些乐趣。要知道究竟有多搞怪么？近来看看……

（1）滑板摩托车

这是一款将滑板与摩托车结合在一起的新型交通工具。其内之的43cc 两冲程发动机能够使其食宿达到16-19英里每小时（1英里约合1.6公里）。

（2）四脚摩托车

这款由日本人年发明的四脚摩托车则是用四肢取代轮子来驱动摩托车行走，虽然没有实际意义，但的确非常搞怪。

（3）履带滑板车

这款履带滑板车与前面的滑板摩托车有着异曲同工之妙，只不过这里采用的是履带作为驱动，这样就能适应更多不同的路况。

（4）人力滑翔艇

之前我们已经介绍过这款产品，但是并没有能够看到它具体的操作过程，今天通过一段视频来更好的了解一下他是如何工作的。

（5）水陆两栖摩托车

这款摩托车绝对酷毙了，摇身一变，摩托车便成水上摩托车。无论旱路水路，都畅通无阻。

（6）全球最小的单人直升机

这款由飞行爱好者自行研制的直升机最高飞行高度为1000米，速度90公里每小时，最长飞行时间为30分钟。虽然很简陋，但创新精神值得钦佩。

（8）蝙蝠侠气垫船

未来世界的智能设备

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

科技的发展在不断给我们带来惊喜的同时，也在不断的改变着我们的生活。那么，未来世界还会有那些有趣的发明会使我们的生活更加方便呢？今天我们就给大家介绍几款非常有趣的小东西。

智能鸡蛋

“不列颠鸡蛋信息服务中心（British Egg Information Service）”发明了一种新型的鸡蛋，可以帮助您更好的烹饪鸡蛋——当然，并不是对鸡蛋本身做什么改动，而是在鸡蛋壳上涂上一层特殊的“热感应墨水”。

只有当鸡蛋达到相应的温度，这层墨水才能显示出来。通常分为三种类型：熟透（hard-boiled）、半熟（medium）、不熟（soft）这样您就可以根据自己的口味来选择相应的墨水，从而烹饪出自己喜欢的口感。

看！图中的鸡蛋是不熟的哦！

智能空调

三菱公司今日推出了新款的智能空调。内置的感应器能够感应人体的运动，只有当探测到附近有人走动的时候，空调才会自动开启。

当然，这种空调并不适用于家用，但是对于人流比较集中的公共场所还是很实用的。据说这种方式可以节约50％的用电。通过一段视频来看看这款空调的效果。

智能桌子

这款智能桌子其实就是一个扫描仪。表面由玻璃制成，上方有摄像头，下面有投影仪和镜子。只要把物体放在桌子表面，然后按下按钮，物体的图像就会被“扫描”下来。非常方便。

智能电脑控制器

这个是一个非常特殊的摄像头，他可以通过感应人的动作来达到控制电脑的目的。通过这个小东西，你就可以通过不同的手势来控制电脑，而不需要再使用鼠标了。具体的操作情况还是通过一段视频来了解一下

智能饮料

这是一款可以自己调节口味、颜色、气味的饮料。再没有进行任何操作之前，它只是一瓶普通的苏打水，当你选择了自己喜欢的口味、颜色、气味之后它就会根据用户的需要调节出非常具有个性的饮料。当然好喝与否还是取决于您自己。

据说通过这种方法可以调节出32种不同的饮料。

环保台灯

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

又是一款用包装直接作为框架的产品。都不需要拆包装，直接打开来插上电源就可以用了。配上节能灯泡，就是名副其实的环保台灯了。

随着人们环保意识的不断加强，这种节约的环保设计一定是大势所趋。

设计者：Gumdesign

节能插线板

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

传统的插线板通常只能够简单的用来控制电源的开关，而为了达到更加节能的目的就有公司专门开发出了这样一款节能的可编程插线板。

它是专门为那些经常使用电脑的朋友设计的，可以通过USB接口与电脑相连，并通过配套的软件来对插座的开关进行相关的设置，从而使得其在必要的时候可以自动控制电脑和周边设备的用电量，以达到节能的目的。

另外，软件内部还配有一棵虚拟树，他可以根据你的用电量和通过这款插线板节约的电量来估算你对环境的影响，并通过虚拟树反映出来。通过这种更为直观的方式来提高你的环保意识。

压缩方便面包装

发表于 2012-03-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

桶装方便面是不少就餐时间不固定或是就餐条件简陋的差旅人士必备食品之一。虽说桶装方便面节省了清洗成本，提高了就餐效率，但是它的储存依旧是个问题，会占用掉不少行李空间。所以，设计师给它们设计了一款新型包装，让外包装可以在不使用的情况下保持压缩状态，减少空间占用体积。如此一来，不仅仓储、运输成本降低了，就连垃圾收纳也能节约不少空间，还方便了使用者持握面桶，可谓是一举多得。

设计：Liu Yi, Jiang Yuning & Luo Jing

机场座椅辅助充电设备

发表于 2012-03-04 | 更新于 2018-07-24 | 分类于 电池

随着数码技术的不断更新换代，设备功能越发完善，我们在出行的时候也越来越离不开这些电子设备。也正是因此，在长途旅行中，若是数码设备的电池用完，难免烦躁，觉得失掉了一项重要的娱乐工具。针对这一情况，设计师推出了这款便捷的充电设备，它可以将乘客候机休息的时候，对座椅施加的压力转换为电量，储存在电池里。然后，游客们可以通过座椅下方盒子电池上的USB和普通充电接口进行使用。此外，您还可以通过方盒子LED灯观察该设备剩余多少电力。

设计：Ryan Klinger
