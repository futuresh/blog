新型残疾人自行车

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款专为残疾人设计的自行车，它通过座椅两侧的杠杆操作杆调整、操纵车轮，让腿部肌肉受损但手臂依旧完好的残疾人也可以轻松地驾驶自行车。自行车巧妙的7齿轮结构可以让驾驶者节省不少力气，操纵杆上下摆动即可驱动车辆，左手操纵杆负责控制车轮转向，右手操纵杆负责控制刹车。它的操作设计十分简洁，驾驶者即使是初次接触，也能轻松上手。

设计：Tom Robbins Design

“零污染”汽车

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

2007年3月17日，全球第一款的零污染汽车终于下线了。

这款汽车通过储存的压缩空气推动活塞运动从而产生动力，整个运动过程中几乎不产生任何的有害气体，因此被誉为“零污染”汽车。

由印度最大的汽车生产商Tata 制造。

补充：这种汽车的行驶成本为每100公里不到1欧元，仅为普通汽油车的十分之一（欧洲标准），最高时速可达68英里。通过储气罐存储压缩空气，每次充气可以行驶200－300公里。

自行车空中基站

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

对喜欢自行车旅行的背包客们来说，最为难的莫过于停车了。大城市里寸土寸金，给自行车停车的地方越来越少。而设计师则提出了一个全新的设想，将越来越小的自行车停放处进行改造。自行车借助锁链和轨道，可以垂直地悬挂在半空。而节省下来的地面空间则可以安置快餐厅和浴室。如此一来，除了能够安全地帮助背包客收纳自行车以外，还能让他们就近解决就餐问题，也能简单地通过热水浴放松一下疲惫的身体。

设计：Yinnon Lehrer

环保过滤水瓶

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 环保&公益创意

针对日益严峻的饮用水短缺、浪费问题，设计师们推出了这款新型环保水瓶。与水瓶名字“321”相互呼应，它的使用方法也十分简单，只要按照水瓶使用指南，通过简单的步骤即可将普通水源过滤，成为干净的饮用水。您只要将水瓶打开，取出与瓶盖相连的树枝状活塞部分，然后把普通水灌入水瓶当中。接着，像推动注射器那样，再将“加长”版瓶盖部分缓慢推入瓶中。这个时候，原先灌入的水就会透过过滤活塞底部，渗透上来。这时，经过活塞过滤的普通水便已经满足饮用卫生要求了。水瓶活塞头可以单独购买，而一个新活塞至少可以满足50次过滤要求。

售价：44.6美元 【在此购买】

弧形人行道

发表于 2012-01-19 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

我们在过马路的时候，往往会不自觉地按弧线轨道行走，这样会更方便地抵达街对面的目的地。既然习惯如此，为什么人行横道不能因此做出改进，以保证行人安全呢？所以，韩国设计师便给人行横道做了改革，将人行横道的侧边改为弧形，让它更加符合人们的行走习惯。与此同时，设计师还打算给人行横道加上LED显示面板，以提高人行横道的醒目程度，而这块面板发光所需的能量则来源于车辆和行人对人行横道施加的压力，如此便无需额外的电力支持了。

设计：jae min lim
