创意竹筒环保袋

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 环保&公益创意

容易生长的竹子被用于各种场合，其中最著名的莫过于日本世博会的竹子会场了。设计师们当然不会放过这块环保材料。将竹筒稍做加工，便可以收纳一个环保袋，而将环保袋拿出来之后，竹筒还可以充当袋子把手，让提重物的时候手部压力减轻不少。

智能环保单车

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

在首届国际概念汽车设计比赛中，这辆精巧的节能电动单车获得金奖。它的独特之处在于，它拥有两种驾驶模式：在较窄的空间，您可以采用二轮驾驶模式，平时则可以三轮行驶。车内装载了智能自动控制系统，可以由电脑监控车间距，防止发生事故。对于上班族来说，这种小车既为上下班提供了便利，又可以彰显自己的个性。

设计：Han Jing

新型多功能环保自行车

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款新型多功能环保自行车，它由铝和可回收材料制造而成，比普通的自行车更加绿色。设计者考虑到不同人的出行需求，在自行车上设置了丰富的功能模块。你可以在后轮上方放置儿童座椅，也可以放置小型储物箱，还可以为平衡性差的人设置两个后轮以增加平衡。有这么一款功能丰富并且环保的自行车，相信会吸引更多的人加入环保行列！

设计：Nicolas Nicolaou

苔藓地毯

发表于 2012-01-25 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一种由日本人设计的可生物降解的苔藓地毯。它使用的是一种叫Terramac的环保材料，其制品废弃后在土壤或海水中经微生物作用可分解为二氧化碳和水，这种材料不但可以固定植物的根，还可以保护种子，并能将苔藓聚集到一起，从而形成这样美丽环保的苔藓地毯。
