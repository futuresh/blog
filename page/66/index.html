聪明的洗碗水池

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

大多数人在收拾饭后餐桌的时候，总会习惯先将脏污的餐具堆砌到洗碗水池里，然后返回餐厅擦拭桌椅，递送餐后水果，待送走客人之后再来清洗餐具。这款洗碗水池便可以为您节约时间，省掉清洗餐盘的工作，当然，也因此省掉了因为搁置时间过久餐具难以清洗的烦恼。它配有两个备用水池，将脏污的餐具放入洗碗池中，您便可以按动水池边上的旋转开关，将餐具送入水池洗碗后台清洗。这时，另一个干净的水池便就开始工作，方便您处理、清洗餐后水果。而清洗水果的水则会送入后台，循环使用，调用为脏污餐具的第一次油污冲刷用水，十分环保。

设计：AHHA Project

带发动机的旱冰鞋

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

作为一种非常环保的代步工具和娱乐项目，旱冰鞋在全球都颇受欢迎。不过如果要作为代步工具，有的时候（比如上坡）还是比较麻烦的，那么是不是也可以给旱冰鞋来一点额外的动力呢？

这款旱冰鞋通过在右边的鞋子上安装一个汽油发动机，帮助你以更快的速度行驶。发动机重约10磅，可以在几秒钟内加速到20英里每小时。大大方便了那些喜欢旱冰鞋的朋友。不过这款设备也有一个缺点，那就是它并没有制动系统，如果要停下来，你所能做的只有关掉发动机，然后用普通旱冰鞋的方式来刹车。

该产品为中国制造，预计上市后在英国的售价约为150英镑。

悬挂式盘子

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

相对于客厅和卧室而言，厨房空间显得比较狭小，因此如何充分利用这部分空间就显得非常重要。

我们之前曾经介绍过的“折叠盘架”和“模块化盘架”都是很不错的节约空间的产品。但是这些设计也都是需要占用一定的平面空间的，假如能够直接将盘子挂起来岂不是就能够节约更多的空间了？

这款悬挂式盘子就可以通过边缘的开口将盘子直接悬挂起来。如此一来，不仅可以有效节约空间，而且更容易保持盘子的干燥。

设计师: Marc Ballve & Victor Vinyamata

防盗钱包

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这是一款简洁轻便的钱包，尤其适合RFID卡用户使用。

无需通过机械或者光学接触，相应设备即可读取RFID卡的数据信息，因此出现了RFID窃贼。他们通过秘密启用相关设备，读取附近使用RFID技术信用卡、身份证的用户信息，进行犯罪活动。因此，尽管RFID技术方便了用户的使用，但是同时，其安全也受到威胁。

而这款钱包可以保护RFID卡上的信息不被他人窃取。你只需要将卡片放入钱包，钱包合上的时候，它就会自动屏蔽卡片上的RFID信号，这样窃贼的犯罪行为就不能得逞了。这款采用了可回收材料制成的钱包可以放置6张卡片，和15张纸币。另外，它还预留了一个相片插槽。至于钱包的开合由一个按钮控制，轻按按钮，钱包就会弹开，方便使用。以下有Youtube产品介绍视频可供欣赏：

售价：39.95美元 【在此购买】

补充：
射频识别即RFID技术，又称电子标签、无线射频识别，是一种通信技术，可通过无线电讯号识别特定目标并读写相关数据，而无需识别系统与特定目标之间建立机械或光学接触。

下水道发电机

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 电池

一名大学生受到水电站启发，设计发明了这款水力发电机。这款发电机安装在楼内的垂直下水道上，而不是水库内，依靠废水从高处下落的重力，推动涡轮机叶片转动，从而产生电能。这个想法付诸实践后得到了很多人的认可，有望在将来走入居民家中，为节约能源提供新的解决办法。

设计：Tom Broadbent

概念无线鼠标

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一个“绿色”主题设计比赛的参赛作品。设计者从传统无线鼠标需要大量电池的现实问题出发，结合鼠标使用时会被不停点击的特点，设计了这款利用压电效应为自身提供能量的鼠标。这款鼠标使用时不需要任何电池，仅通过点击左右键来产生电能供自身使用。整个鼠标使用软木材料制成，配合可回收的塑料配件，十分绿色环保。

环保的数码产品

发表于 2012-01-23 | 更新于 2018-07-24 | 分类于 环保&公益创意

顺应工业设计环保化的大趋势，索尼公司也推出了几款环保的概念数码产品。由于采用动力学和太阳能供电的设计，所以这些数码产品都不需要使用电池或外接电源，有效的节约了能源，从而达到环保的目的。

这是一款通过抽拉的方式供电的照相机，拉开时，镜头便展开了

右边的指示器还可以用来显示剩余电力。用手指拉住两边的小孔，不断抽拉就可以发电。

背面有一个凹槽，在拍照时就不会硌到鼻子。拉开时，刚才用来抽拉的小孔就变成了取景器。

这是一款滚轴发电的播放器。只需要像图中那样不停地滚动滚轴就可以获取电力。

这款制作精美的太阳能电池板，平时可以这样折叠起来。需要的时候展开，就可以像普通的太阳能电池一样为上述设备提供额外的电力。
