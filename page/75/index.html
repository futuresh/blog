微型风轮机

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

一家意大利公司研制出这系列微型风轮机。革命性的造型设计和家庭使用定位是其最大的两个特点。当风速为14米每秒的时候，中空的四边形风轮机能输出400瓦电量，而螺旋形风轮机能输出1000瓦电量。该系列微型风轮机的售价为2500欧元和3500欧元，每款风轮机的安装费用大概为1000至2000欧元。一个普通家庭只要安装上3个微型风轮机就能够满足日常使用了，实惠又环保。

设计：Philippe Starck 【官方网站】

太阳能音箱赏

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着太阳能板的日益普及，各种太阳能产品如雨后春笋般涌现出来。太阳能音箱给室外活动提供音响支持带来了很多便利条件，下面，我们就向您展示近年来一些杰出的太阳能音箱，请您和我们一起感受清洁能源带给我们的便利。

这款音箱外观类似于手风琴风箱。在太阳能板和扩音器面板之间，有可折叠的低音炮，打开的角度不同，其所发出的低音效果就不同。此外，如果您的iPod 或iPhone的喇叭音量小，音箱的内置扩音器便会在有光源的情况下自动提高功率，增强音效。

设计：Tom Pritchard

将无线功能融入太阳能音响，让音箱变得随处可用。这款全天候无线音箱可以连接到任何一个音频设备，并且音质不会受损，无线信号可以透过墙、地板甚至天花板。由于音箱被置于室外，可以确保电量充足。

这款可折叠扇形音箱，表面贴有太阳能板，既节约能源，又节省空间，还便于随身携带。

这款音响拥有可转动角度的太阳能板。您可以根据太阳照射角度，调节最佳受光角，以获取最多的能量。另外，它还带有防水盖，防止室外使用时意外降雨损坏机器。

这款类似展开的卷轴的音箱，除了太阳能板之外还带有一个USB充电接口，以确保在阴天的时候也能正常工作。

这是一款高达3英尺的太阳能音箱，拥有巨大的太阳能板和一个背投LCD显示屏。研发公司称，在阳光下，这款设备充满电需要20小时，室内则需40小时，而且充电时间等同于使用时间。

看了这么多设计巧妙、造型优美的音响，想必您也跃跃欲试，想DIY一个吧？这款太阳能音箱就是纯手工打造，它仅仅使用了一种材料就完成了整体构建。此外，它体积窄小，方便搬运。

风力充电器

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

之前有过“太阳能充电器”、“手摇式充电器”，如果还要找到一种环保的方法来充电，你会想到什么呢？

对了！风力充电器。与前面的产品类似，也是专为一些特殊情况设计的临时充电设备。只要风够大，随时都可以为你的手机进行充电。譬如在风力较强的地区进行野外考察的时候，这东西一定能派上用处。

该充电器由德克萨斯大学与美国无线运营商Orange共同开发，重量仅为150克，可以很方便的固定在帐篷上。

折叠车座

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

在地铁里，尤其是上下班高峰期的时候，人潮拥挤。如何才能够合理地安排座位，让地铁车厢显得更加人性化呢？设计师给我们提供了一个新点子。如果地铁车厢里的座椅能够折叠起来，那想必可以增加更多的空间，同时还能够在不减少车载人数（相比起未改装前）的情况下，让车厢中部也增加一些座椅，这样地铁乘客人少的时候也就可以让更多的乘客有了休息座位。即使座椅翻折上去，它依旧可以起到让乘客依靠、休息的作用，只不过坐起来没有那么舒适就是了。

设计：Jun Yasumoto

可双面书写的明信片

发表于 2012-01-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

虽然在提倡环保的今天，明信片已经渐渐淡出了人们的视野。但是仍然有很多人对这种传统的联系方式情有独钟。可是又不得不考虑环保的因素。

于是有人就想出了这种相对折中的办法——把明信片的两面都做成可以书写的，别人收到明信片之后，可以使用另外一面进行回邮。在如今花样繁多的明信片里，这样一款双面书写的没有任何修饰的明信片或许会收到意想不到的效果。

每十张售价6.24美元

多用自行车停靠架
