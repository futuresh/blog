波纹布艺保温垫

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

这款造型简单的波纹布垫可以为您的热饮容器提供保温服务。柔软的布垫织物密封着若干盐分，正是因为有了盐分的保护，菜锅和水杯的热量散播才会变得缓慢，从而达到保温的效果。有了它的保护，即便忙于工作忽略了手边的新泡咖啡，您也不用担心咖啡会马上变凉，损坏口感。

消失在杯子上的冰川

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款杯子看上去似乎没有什么特别的。而实际上，它外层的冰川图案是由热感应涂料绘制的。当热水被倒入杯子中时，冰川就会从杯子上消失，只剩下一望无际的“大海”。

在展示创意的同时，也警示人们全球变暖将会带来的后果，从而提高环保意识。

太阳能充电器

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 电池

现在地球的能源是越来越匮乏了。所以各种各样的新能源已经开始渗入到我们的生活中了。这不，又出现了太阳能充电器。它通过内置的7伏太阳能电池板对普通充电电池进行充电，只要阳光明媚，把它放在阳光下就可以象普通充电器一样使用了。当然，阳光强度不一样，充电时间也不一样。通常情况下，阳光比较好的时候可以再6个小时内完成对四节7号电池的充电。

太阳能窗户

发表于 2012-03-08 | 更新于 2018-07-24 | 分类于 电池

阳光和窗户的关系非常简单：阳光透过窗户照进室内，使得室内更加明亮。但是这种关系似乎也只限于白天。不过，一位来自澳大利亚的设计师却将它们的关系延长到了全天24小时！

这款窗户表面看来非常普通，但是它却能够在白天吸收太阳光并储存在电池内，到了晚上再利用储存的能量来产生相当于60瓦白炽灯泡的亮度。由于这款窗户是通过OLED发光的，因此，能耗更低、亮度更强。4小时的日照就能提供约6小时的夜间照明。

有了它，不仅能够为家里节约不少的电费，而且更为奇妙的是，没开灯之前，你甚至根本无法察觉到这盏“太阳能灯”的存在。

设计师：Damien Savio

这款造型别致撞击记录器有三种不同的型号，可以适应不同的高尔夫球杆。
