USB土壤分析器

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 都市农业

想要在家里养点花，可是又没有经验，该怎么办呢？别着急，有了这款USB土壤分析器，一切都迎刃而解了。

这款产品，可以直接被插到土壤中，放置24小时之后，它就会从土壤中搜集到一些重要数据。只要将它与电脑相连，即可通过配套的软件了解到这里的土壤是否适合种植某种植物了。不仅如此，它甚至会给出一个列表，向你推荐一些适合在所测土壤中种植的植物。

蜗牛形太阳能摩托车

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

相比摩托车而言，骑车更容易实现由太阳能来供给动力。这是因为摩托车的构造相对更难以获得充足的日照，而这也决定了太阳能摩托车注定是奇形怪状的。

这不，一款名为SunRed的外形酷似蜗牛的摩托车就通过特殊的外表来尽可能多的获取太阳能。这款摩托车的太阳能板面积达到25平方英尺，它所存储的电力可以供给摩托车以30英里的时速行驶约13英里，如果艳阳高照，还可以边行驶边充电。由于马达整合在了车轮中，所以不需要任何额外的传动装置。

带护手的围裙

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

围裙是用来防止油烟的，护手则是在端热锅的时候保护手部防止烫伤的。这两个东西都是厨房常用的，如果能够将这两个产品组合在一起，就不必再系着围裙找护手了。

这款护手围裙表面看起来跟普通的围裙一样，但是在围裙的两侧增加了两个护手，这样一来，你只要穿着围裙，就可以随时找到护手，既可以防止油污，又能够保护双手。一举两得。

其实我们之前曾经介绍过这款产品，这次之所以又把它拿出来，是因为这款带护手的围裙刚刚荣获了2007德国红点设计大奖。不愧是一款非常实用的设计。

设计师：Lee Jongoh

可控温床垫

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款床垫适合在任何季节使用。床垫温度的调节主要是由一个类似半导体装置的电脑系统来控制，最终通过床垫内部的水循环系统来实现温控。您可在华氏48至118度之间任意调节。除此之外，这款床垫还是一款非常环保的产品，它是由百分之百的大豆纤维制成，可以为您提供一个优质的睡眠环境。

盘子架

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 环保&公益创意

用现有的材料做一下简单的改装就变成了这样一个盘子架，原理跟之前的“鞋架（夹）”类似，当然普通的家庭或许用不上。根据设计的说明来看，应该是用在饭店里的，大概是这样一种场景：

事先把盘子放在上面，然后厨师做好菜，就可以直接把菜倒进盘子里，如果是一次好几盘相同的菜就更加方便了，然后服务生负责上菜，直接从架子上取走，后续人员接着将盘子“插”上。最重要的一点就是有效的节约了桌面空间，尤其是准备大餐的时候，经常会为了盘子过多无处摆放而烦心。有了它，这些工作应该会更加有条不紊。

拐杖变椅子

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

对于腿部残疾或受伤的人而言，拐杖是必不可少的。但是相比轮椅而言，拐杖要费力的多，尤其是走累了要休息一下的时候更是麻烦。如果能够把拐杖变成椅子该多好？

这款椅子拐杖的外形酷似我们前面介绍的一款“S形拐杖”，不过它的功用并不仅仅在于减少冲击。当使用者需要休息的时候，就可以将两个拐杖拼合在一起，靠在墙上就变成了一个可供临时休息的椅子。

设计师：Yong-Rok Kim

彪马长轴自行车

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

以设计销售运动服、运动鞋起家的彪马公司如今也开始涉足自行车设计领域。初入行业，便显示出它的超凡设计能力。这辆加长版自行车便是由彪马公司设计，专为那些喜欢购物、出游的青年人打造。它最吸引人的地方在于其特殊的支架部分：四边形宽大支架，超长下管，以及那结实、容量巨大的前车筐，让这辆车子显得与众不同。经过精密的力学设计，看上去不稳定的支架实则非常牢固，在崎岖的道路上也表现出色。此外，车筐可以盛放比一般自行车多好几倍的物品，且不必担心由于物品过多造成车把难以操控。铝制车身减轻了由于结构的增长而造成的额外重量，使整车质量轻便，易于运输。这辆新型自行车将在本月的欧洲自行车展上亮相，预计将于明年春季上市。

海滩清洁车

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

海滩经常被人们当做夏日度假的首选之地，但是大量的游客往往给海滩带来各种垃圾、污染，难于处理。一位德国设计师设计了这款海滩清洁车，希望能够通过它更好地保护海滩环境。这款车使用了一种不含空气的特殊车轮，这种车轮不怕刺扎、更加耐磨，不仅抗水漂性强，同时还具有减震功能，非常适合在海滩使用。由于配备了这种特殊车轮，可以降低驾驶室的位置使驾驶员获得更加宽广的视野。同时，这款车还可以通过遥控进行控制，使得清洁范围进一步扩大。

设计：Alexander Muller

便捷充电器

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 电池

这款小巧的方形充电器拥有两种充电方式。它除了能够通过表面的太阳能板为内置电池补充能源以外，通过拉拽充电器拉环，它也将动能转化为电能储存。当随身设备缺少电能的时候，只要通过连接线，即可将充电器内存电量转移至相应设备。这款充电器的侧边还留有LED灯，可以为你提供照明服务。

售价：1980日元 【在此购买】

新型无空气轮胎

发表于 2012-02-14 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

与米其林概念轮胎类似，这款新型轮胎的设计在技术上又是一次革命性的创新。轮胎内部无需空气作为主要支撑元素，可以让汽车的行驶安全性能得以提升，还能节省燃料损耗。此外，它还采用碳纤维原料和开夫拉（kevlar）材料塑成，能够在减轻重量的同时提高自身的坚韧程度。虽说这款新型轮胎目前还在测试当中，但是它的上市仍然值得期待。

设计：Brian Russell [官方网站]
