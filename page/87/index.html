超级翻斗车

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

昨天刚刚介绍过一款获得2007红点设计大奖的“旋转叉车”，其实其他的工程车辆上也存在着或多或少的问题，下面我们就来介绍一款同样获得了2007红点设计大奖的超级翻斗车。

这款翻斗车看起来非常的酷，就像是一个太空车。它的驾驶室由两个巨型的力臂支撑，可以调整角度和位置，使得它的全部四个方向都可以装卸货物，从而省去了不停的调整卸货方向的麻烦，尤其是可以直接从前方装卸货物，更能够显著增强安全性。另外，它的轮胎也通过力臂与卡车相连，更容易适应不同的地形。

设计师：Deng Haishan

全木质自行车

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

这是一款由加拿大的一名16岁高中生制作的全木质自行车，这款自行车的独特之处在于，不仅仅是框架和车轮，连齿轮和链条以及整个自行车的所有零件完全由木头制成。而更为神奇的还在于，这款自行车竟然是可以骑的！

要制成这样一款自行车的最大难点就在于链条。因此这位高中生经过对多种不同硬度的木材进行反复测试后最终制作出了这样一款可以骑的木质自行车。

用旋转门发电

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 电池

随着人们环保意识的不断增强，各种各样的环保产品和替代能源也是曾出不穷。人力发电也是其中的一种比较流行的思路，我们也曾经介绍过“让行人来发电”的装置。其实生活中还有很多的东西都可以利用类似的思路来发电。

这款新型的旋转门就将发电机整合在里面，每当有人进出而推动旋转门转动时，都可以将这部分动能转化为电能储存起来以备不时之需。尤其是对于那些身处闹市的写字楼和商场而言，这种产品一定会产生不错的效果。

哥本哈根环保自行车

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

正当哥本哈根大会如火如荼进行的时候，麻省理工的研究学者们推出了这款环保车轮。这款车轮可以将您乘骑时候使用手闸等设备产生的制动力储存起来，然后在您上坡或者加速的时候给您提供辅助，让您省下不少力气。除了贴心的辅助设计，它还能将沿途的路况、空气质量和其他乘骑信息进行统计，并将结果发送至您的手机上，根据其变化的数据，您可以更加合理地安排出行计划，选择空气更加舒适的时段出行。

发光的自行车踏板

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 环保&公益创意

自行车的行车安全也越来越受到人们的重视，“腕式转向灯”就是专门为自行车设计的用以保证夜间行车安全的产品，同时可以作为扮酷的工具。

如果你还是觉得自己的自行车不够安全，也不够酷，那就试试把踏板也换一下吧。这款新颖的踏板就内置有三个LED灯，分别对前后和侧面三个方向起到警示的作用，同时也可以使你的自行车更加吸引眼球。而且，这款产品非常环保，它的电力是完全由踏板自身的转动来提供的，内置的电池也可以存储一定的电力，所以，即使停车也仍然可以提供5分钟左右的照明。

纸质头盔

发表于 2012-01-05 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

据调查，市面上的塑料头盔其实并不能起到很好的减震防护作用，因为多数制造商将主要注意力集中在头盔的外观和空气动力学设计上，而忽略了安全性。设计师用包装用的减震纸板制作了这款头盔，希望通过纸板内的空隙来吸收碰撞过程中的冲击力。经过试验，得到了令人振奋的结果：这种纸框架结构的头盔在撞击过程中吸收能量是普通塑料头盔的四倍。这种头盔不仅防护效果出色，而且制造方便、环保轻便。此外，还可以根据用户头型与要求不同，定制款式和规格不同的头盔，以便给使用者提供更贴心的保护。

设计：anirudha rao

新款抽拉式储物架

发表于 2012-01-04 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了利用较少的空间来存储尽量多的东西，很多家庭都会利用多层的架子来存储各种各样的调料瓶。但是直立放置的架子不能够达到很好的节约空间的目的，如果是平放却又不方便拿取。如果能够将这两种架子的优点结合到一起一定很不错。

表面看来，这款储物架没有什么特别，但是当你将每层抽屉拉出到尽头后，就会自动垂下，使得瓶子更容易拿取。从而吸取了直立和平放两种架子的优点。
