绿色环保吸尘器

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 环保&公益创意

作为大型家电产品制造商，伊莱克斯公司每年都需要消耗大量的塑料制品。为了响应环保的号召，他们发起了一项海洋塑料垃圾回收运动，呼吁人们关注海洋生态环境，唤起公众的环保意识。与此同时，伊莱克斯公司将收集来的塑料垃圾进行重新利用，制成了这款限量版的拥有五彩斑斓外壳的吸尘器。其中的彩色部分由回收塑料直接压制成型，省去了二次加工带来的污染。这款吸尘器70%的原材料来源于回收塑料，最大限度地将塑料垃圾“变废为宝”，不仅节约了资源，还打开了新产品的大门。

挂在桌边的打印机

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 环保&公益创意

随着电脑外设和办公设备的逐渐增多，办公桌也成了寸土寸金的地方。能多节约一点桌面空间对于上班族而言就意味着桌面能够更加宽敞，心情也就更为舒畅，工作效率自然也能得到相应的提高了。

我们之前介绍的一款“壁挂式打印机”就是一个很不错的设计，而我们今天为大家介绍的这款设计虽然不会完全的脱离桌面，但是也的确可以节约很大的空间，更重要的是这种设计的成本更低，也更容易实现。

它其实就是将一个普通的打印机设计成了L形。大部分的元件和设备都被设计在L的长边上，而进出纸口和控制面板则位于短边上。只要将短边直接搭在桌面上固定好即可。这样的设计不但可以很好的节约桌面空间，而且在使用过程与普通的打印机几乎没有什么区别。

设计师：Jin Hee Kim, Hyung Il Kim & Woo Seok Park

腕式转向灯

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

“无线自行车转向灯”的确可以有效的保护夜间行车的安全，但是要在每一部自行车上都安装这样的设备不仅花费较大，而且非常麻烦。

那就来试试这款新颖的腕式转向灯吧。它可以像手表一样直接戴在手腕上，当你需要转向的时候只要将手臂扬起向后面的车辆示意，转向灯就能够自动探测手臂位置的变化并开始闪烁从而提醒后面的车辆。有了这样的产品不仅非常方便，而且能够大大增强骑车的安全性。

售价19.95澳元

手表上的信用卡

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

在万事达卡的Paypass系统以其方便、安全、快捷的特性深受广大消费者的喜爱。日前，又一款方便的Paypass设备应运而生。将Paypass系统整合在手表内，这样你就可以不必携带信用卡，只需要带着这只手表就可以很方便的支付各种费用了。

小知识：

_

PayPass是什么？

PayPass是MasterCard（万事达卡）国际组织2003年推出的一种无线支付技术，从2004年开始 在全球推广。从表面上看，这种技术与目前已开始使用的非接触式IC卡技术类似，但却有着全新的技术含义。这种卡（以下简称PayPass卡）集成了传统意 义的可编程IC芯片以及目前事实标准ISO 14443支持下的RFID，同时使用芯片、PIN以及RFID技术实现非接触支付。

在这些技术的支持下，持卡人只要将银行卡在专用非接触式读卡机（可以理解为专用POS）前挥动，即可完成传统形式的银行交易，而不用像以前一样，需要将卡片从读卡器插槽中划过。

PIN中可以保留持卡人详细的信息，这样当交易过程需要核实身份时，特约商户可以方便地进行查看，同时PIN中还 可以保留一定数量的历史交易明细。IC芯片则继承了传统IC卡存储容量大、可编程的优点，银行可以利用它开发出诸如安全保障、信息存储、用户管理等一些在 普通磁条卡上无法实现的强大功能。同时PayPass又具有传统IC卡所没有的功能。比如，IC卡不能与银行后台直接交易，需要通过圈存、圈提等交易与银 行进行转账后才可以使用，而PayPass卡则可以像磁条卡一样，与银行后台直接关联。看似简单的一个技术实现，将为我们开辟一个崭新的应用方向。

以前这种技术概念之所以未获得推广，主要是因为无线数据传输的速度太慢，而且有信息外泄的可能，最近几年，新技术的发展特别是RFID技术的发展使这一切成为可能。

PayPass技术的核心突破是通过RFID技术将IC卡与传统的银行卡应用结合起来。它集成了IC卡的优点，带 来了极高的安全性，在交易的整个过程中，卡片一直在持卡人的手中，一些传统的伪卡犯罪手段不再行得通。此外，万事达组织还许可在一定条件下，可小额交易免 签单、免输密码等，使其更简单、方便、快捷。另外传统磁条卡刷卡时，对卡片的磨损、对刷卡动作的额外要求以及磁条信息读取时发生的时间延误等缺点也都不同 程度地被克服了。据国外统计，Paypass卡单笔交易时间可以最高缩短达64%！使得银行卡这一“电子钱包”的推广价值迅速提高。

环保海水淡化器

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 环保&公益创意

在缺乏淡水的沿海地区，抑或遭遇海难的时候，一个实用的海水淡化器就显得格外重要了。这款新型海水淡化器可以漂浮在海面上，吸取海浪的能量，驱动内置的过滤、淡化系统。海水由底部的孔洞进入环绕四周的三个淡化器内部，海水不断冲刷，而其中的盐分则被这种力量配合过滤设备阻隔下来，从而获取淡水。这款淡化器无需其他能源驱动，唯一需要注意的就是因为海水的侵蚀和使用磨损，得定时更换过滤网杯。当然，它也是采用可回收环保材料制成的，无需担心环境污染问题。

设计：Yu-Hong Chen

节水的浴室底座

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 环保&公益创意

乍看起来这似乎只是一个普通的浴室，只不过在底部增加了几个涟漪形状的装饰。不过也正是这样几个装饰，使得它还具有了节水的特性。

平时这些“涟漪”还保持比较平整的状态，而随着洗澡时用水量的逐渐增多，涟漪就会逐渐升高变得非常硌脚。用这种方式来提示你，节约用水。

设计师：Tommaso Colia

最直观的节水开关

发表于 2012-01-16 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款专门为节水设计的花洒开关。它采用了多节的旋转设计，每当打开后，它就会像发条一样逐渐的旋转恢复到原位。虽然每次归位后你仍然可以拧开，而不会直接的限制你的用水量，但是眼看着水龙头慢慢的旋转归位，应该也会对使用者形成一定的心里暗示，提示使用者节约用水。

设计师：Julia Treutiger
