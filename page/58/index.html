折叠旅行拖鞋

发表于 2012-01-28 | 更新于 2018-07-24 | 分类于 环保&公益创意

为了节约原材料避免浪费，很多酒店都已经取消了传统的六小件。因此，我们出门的时候就需要自己携带牙刷、拖鞋等生活用品。我们之前也介绍过“旅行牙刷”和“带水龙头的牙刷”可以有效节约行李空间。但是相对于牙刷而言，拖鞋这样的东西要更加占用空间。因此，我们需要一款专为旅行准备的节约空间的折叠拖鞋。

这款新颖的拖鞋平时可以像平板一样储存在行李箱里面，就像两个纸板一样只会占用很小的空间。当需要的时候只要稍作弯折即可变成一双拖鞋。非常方便。

该产品由EVA泡沫制成。

头盔喇叭

透明的厕所

发表于 2012-01-26 | 更新于 2018-07-24 | 分类于 生命科学&第三龄

折叠自行车欣赏

发表于 2012-01-26 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

自行车可谓是最健康环保的交通工具。传统的自行车占用空间多且不便携带，不能满足一些有特殊要求的使用者。许多设计师便把目光投向灵活性很高的折叠自行车——它只会占据汽车后备箱很小的空间，有的甚至还可以方便地随身携带。这种便利使折叠自行车受到更多消费者的欢迎。这里，我们精选了一些折叠自行车，请您一起欣赏。

这款自行车折叠之后的尺寸为580毫米×463毫米×406毫米，可以轻松放入后备箱里。

这款自行车的重量仅为5公斤，折叠起来十分容易，只需花费几秒钟即可。

这款可折叠的电动自行车，其设计获得了一项移动设备类的金星奖。预计今年将上市销售。

这款自行车折叠后可紧实成一个圆柱体，车中间的两个圆形结构不仅是整个自行车的结构支架，而且还是折叠后的外部保护。

这款背包形式的折叠自行车重量为5.5公斤。为了减轻其重量，主体部分多为塑料制成。

这款自行车折叠后可变成手推车，便于行动，且节省空间。

这款自行车前部有一个控制栓，只要搬动控制栓便可以轻松折叠前轮。

这款电动自行车没有任何机械驱动装置，即没有链条。轮毂位置配有电机，踏板转动时可以为电池充电。

这款折叠电动自行车是由越南胡志明市一名建筑系学生设计的。其时速可达每小时30公里。充电两个小时可行驶12公里的路程。

这款自行车的造型很特别，外形显得有几分“悠闲”之外，还有几许“高傲”。它还可以通过踏板产生电能备其使用。

这款自行车折叠后的体积是原来大小的一半。它是一款很实用通勤自行车，可轻松放置在火车或汽车座位下。

这款行李箱自行车，折叠后它就像一个真实行李箱，底部带滚轮，可以手提也可以使用拉杆，出行更加方便自如。

史上最快的汽车

发表于 2012-01-26 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

你能想象世界上最快的汽车有多快吗？答案是340英里/小时（约合547公里/小时）！

这款来自Acabion的新款GTBO跑车是由一位前保时捷工程师Peter Maskus博士设计的。它拥有无与伦比的加速性能和极限速度，它能够在30秒内从0加速到300英里/小时(约合483公里/小时)！而最高时速更是可以达到340英里/小时（约合547公里/小时）。这样一部全球最快的汽车的动力是由1300cc的800马力涡轮增压发动机提供的。

该车预计于3年内在英国、美国上市。售价约为150万美元！
