外置安全气囊的汽车

发表于 2012-03-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

安全气囊通常都是被安装在汽车内部用来保护车内人员安全的。可是当汽车撞到行人时，更需要保护的是被撞的行人。因此，通过在汽车的车体上增加一些相关的设备来尽量减少被撞行人的伤害就显得尤为重要。我们之前也为大家介绍过的一款在2007东京车展上展出的“本田PUYO”就通过柔软的材料制成的车体来达到这种目的。

而日前，丰田公司也推出了一款非常新颖的产品，它将原本只在车内配置的安全气囊移植到了车外，这样一来，再有事故发生的时候，安全气囊就能够及时的弹出，从而减小车祸对行人造成的伤害。

泡泡自行车

发表于 2012-03-03 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

随着人们环保意识的不断加强，许多原本依靠机动车出行的人也逐渐开始采用自行车作为代步工具，不过要使得骑行的过程充满乐趣光靠传统的自行车恐怕就有些力不从心了，我们之前为大家介绍的“风笛自行车”可以让你享受到音乐的乐趣，而今天的这款泡泡自行车，则可以带给自己和路人更多的视觉上的享受。它能够在骑行的同时释放出“尾气”，不过这种尾气并不是汽车那样的污染气体，而是一个个充满了童趣和浪漫情调的泡泡。它可以利用骑车时的气流将泡泡吹出，伴随着自行车的前进，留下一路的泡泡。

燃烧脂肪，不燃烧汽油；释放泡泡，不释放尾气。不仅锻炼了自己，也可以给路人带来更多的好心情。

货板办公室

发表于 2012-03-03 | 更新于 2018-07-24 | 分类于 环保&公益创意

在阿姆斯特丹一家临时工作室内，没有现代时尚的办公桌椅，也没有装饰豪华的室内空间，全部内饰都是用货板搭建而成的：包括办公桌、楼梯、沙发等等。设计师采用了对比强烈的装饰风格，纯白色的墙面与地面，搭配过渡效果的货板、纯黑的座椅、靠垫以及楼梯扶手，给人留下印象深刻的现代感及空间感。对于这间临时工作室来说，使用货板材料既美观大方，同时也省时省力，节约资源，给客户留下了良好的印象。

设计：most architecture

太阳能台灯

发表于 2012-02-25 | 更新于 2018-07-24 | 分类于 环保&公益创意

这是一款非常环保的台灯，它的思路与我们之前介绍的一款“太阳能桶”相同。同样可以在白天吸收并储存太阳能，以供给夜晚的照明。

不过这款产品还有一个非常有趣的地方，白天的时候可以将它倒放过来就好像是一盆植物一样在吸收着太阳光。而到了晚上再将它翻转回来就可以当作台灯来使用了。

在光照较强的夏天，每充电10小时（户外），最多可以提供长达40小时的照明。而即使是在光照较弱的冬天，每天充电8小时（室内），也可以提供约2小时的照明。

售价90欧元

可原地旋转的汽车

发表于 2012-02-25 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

如果没有过硬的驾驶技术，是很难让汽车实现360度的原地旋转的。然而有了我们今天为大家介绍的这款概念车，这种“特技”就变成了家常便饭。

这款概念车的四个轮子与车体的连接处都被设计成了弧形的，因此四个轮子均可以获得更大的旋转角度，较之普通的汽车更为灵活，即使是普通人也可以驾驶着这部汽车实现原地360度旋转。
