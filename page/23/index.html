超节能灯泡

发表于 2012-03-10 | 更新于 2018-07-24 | 分类于 环保&公益创意

现在LED灯作为一种全新的照明器具已经非常普及，不过由于LED灯的亮度不够始终无法取代传统的灯泡。这款灯泡由150个LED灯组成，亮度达到308流明，相当于一个70瓦的灯泡，更令人称奇的是，他的功率只有9瓦。还有另外一款加强版本，能够达到594流明。不过需要注意的是，这种LED灯泡所发出的光与传统的白炽灯泡略有不同——LED灯的光比较刺眼（不过加强版可能已经克服了这个缺点）。这款灯泡的单价在60到70美元之间。对于一个灯泡来讲的确有点贵，不过从节约能源的角度来讲还是个不错的选择。

隐藏式陈列柜

这款陈列柜出自英国伦敦设计师之手，是用破旧的木地板全手工制作而成的。它不仅利于环保而且复古时尚。设计师巧妙搭配颜色，精心挑选出一块块大小合适的可回收木地板，才能展现出图中的复古效果。值得一提的是，这款陈列柜还有隐藏功能。打开柜门，您可以将珍贵的艺术品放入里面的盒子，不同形状的盒子可以存放不同款式的物品，非常实用。

紧凑的厨房

节约空间的设计已经成为当下家居设计的主流之一，今天我们再来为大家介绍一款非常紧凑的厨房设计。

平时就是一个比较普通的厨房操作台，包涵有水池、台面、电炉和烤箱。不过当需要额外的一些干净的空间时，就可以将操作台侧面的滑板抽出来，将整个台面覆盖住，同时从下面的缝隙中抽出几个设计非常简约的凳子。由于滑板和椅子都是整合在操作台内部的，所以平时不会占用任何的多余空间。关键时候还能一物多用。

一次性酒杯

发表于 2012-03-09 | 更新于 2018-07-24 | 分类于 环保&公益创意

这款一次性酒杯可以为聚会之后负责清洁工作的人员减轻工作负担。采用可回收材料制成的高脚杯内部密封有各种口味的酒精饮料，使用者只需根据易拉密封盖上的介绍将各色杯子分类摆放，与会者即可随意取用饮料。此后，只要撕开密封盖就可以品尝美酒了。用过之后，酒杯即可丢弃。这款一次性酒杯不仅免除了餐具清洗所耗费的功夫，还通过定量供应饮料而免除了美酒打开之后却又喝不完的浪费情况。

净水三轮车

发表于 2012-03-09 | 更新于 2018-07-24 | 分类于 车辆与交通系统创意

对于我们专利之家的大部分读者而言，自来水、纯净水都是生活中再普通不过的东西了。但是在很多缺水的地区，人们都需要走很长一段路到附近仅有的水源去取水以供一家人全天的用水。这样不仅费时费力，而且水源的卫生也难以保障。即使可以采用煮沸的办法来消毒，也会因此而消耗大量的燃料。

而这款名为Aquaduct的净水三轮车则是一款一举两得的产品。首先它可以像普通的三轮车一样用于运输生活用水，其次它内置的净水装置可以通过脚踏板的转动让你在运水的过程中就将水过滤干净。

这样的产品，对缺水地区的人们而言一定非常的实用。
