多汁草莓的外皮十分脆弱，稍微用力便会碎裂，所以，采摘草莓的工作往往还是由人工完成，以提高草莓收集的完整度，避免造成浪费。而日本科学家们却让“粗鲁”的机器手也能完成这一需要灵巧技能才能完成的活动。这个心灵手巧的机器人不仅可以将草莓从藤上采摘下来，放到包装箱子里，还能通过摄像头识别草莓的颜色，从而保证包装箱内收集的草莓都已经成熟。正是因为机器人摄像头可以多方位观察草莓果实的颜色变化，保证采摘的草莓表面有80%的面积都是红色，成功地辨别果实生熟，所以，只要在软件上稍加修正，这个摘草莓农夫机器人还可以完成其他果实的收获工作。

设计：National Agricultural and Food Research Organization（日本国家农业和食物研究组织）

延长蔬菜保鲜期的保鲜膜

发表于 2014-03-05 | 更新于 2018-07-24 | 分类于 都市农业

据统计，由于包装不当而导致的食物腐烂过期让英国每年浪费高达720吨的新鲜蔬菜。日前，这令人担忧的状况或许得到了新的缓解办法。在英国的乐购以及阿斯达超市，部分蔬菜正在使用一种名为EVAP的新型保鲜膜包装，这种通过提取植物精华获得的薄膜材料具有天然的紫外线过滤功能，同时还能释放包装袋中的潮气，减缓蔬菜“衰老”，并双倍延长蔬果保鲜期，有效减少食物浪费。

iPhone5的功能拓展背壳

发表于 2014-01-16 | 更新于 2018-07-24 | 分类于 电池

电量、闪光灯、存储容量，这是iPhone手机三大令人头疼的疑难杂症，即便是最新上市的iPhone5也未提出更好地解决方案。于是，这款为iPhone5量身定做功能拓展背壳诞生了。它不仅是一个防护性外壳设备，更是为用户提供了多一倍的电池电量、micro-SD卡插槽、性能更出色的闪光灯系统，还包括一个30针接口的内置转接设备等等。虽然这些拓展不可避免的增加了手机厚度，但在实际应用中却没有改变多少用户体验，反而更便捷、更好用。如果您对iPhone自带软硬件不满意的话，不妨尝试一下这款产品哦。

IBM研发会“呼吸”的锂电池

发表于 2013-12-12 | 更新于 2018-07-24 | 分类于 电池

在燃油紧缺的今天，电动汽车成为各大汽车厂商和科技巨头们的重点研发对象，而其中首要的便是需要解决汽车充电以及电池等问题。IBM公司最近正在研发一种锂离子空气电池，该电池工作时通过吸入空气中的氧气，并让氧气与锂离子反应后产生电力，而在驻车充电时电池则会将氧气释放出来，犹如会“呼吸”一般。据称，这种电池单次充满电可供汽车行驶500英里，续航能力优秀。此外，这种锂电池质量很轻，大大减少了汽车负载。据悉，IBM将要与化学巨头旭化成公司（Asahi Kasei）和中央哨子（Central Glass）公司展开合作，预计在2030年将其推入市场。

宏伟的人工树林

发表于 2013-11-24 | 更新于 2018-07-24 | 分类于 都市农业

在新加坡的南湾花园，18棵高耸入云的巨大人工“铁树”让城市的天际线变得异常壮丽。这些高达50多米的钢铁框架上爬满了绿色的藤类植物，种植着各种颜色的鲜花，加上夜间五颜六色的灯光，更是让这些城市中的钢铁巨人多了一丝柔情与浪漫。此外，其中的11棵“大树”上装有太阳能板，可以为植物提供温暖的环境，其漏斗结构同时还可用来收集雨水。该景观预计本月29号向公众开放，到时候让我们一起去观赏人类的杰出创作吧！

DIY“温室”机箱

大功率台式机电脑不仅耗电，其碳排放量也相当惊人。为了尽量减少对资源的损耗与浪费，国外一位DIY达人制作了一台可种植、培育麦芽的“温室”机箱。他利用风扇等电脑部件产生的热量为土壤“加热”，促进麦苗茁壮成长。同时，这些绿油油的小嫩芽不仅吸收了电脑产生的各种有害气体，也美化了室内环境。设计者给出了详细的制作教程，如果您感兴趣的话，不妨也从改造居住环境开始吧！

太空中的蔬菜种植系统

发表于 2013-10-23 | 更新于 2018-07-24 | 分类于 都市农业

与浩瀚无垠的太空和冰冷的机器为伴，让长时间待在太空站的宇航员倍感孤独。同美国航天局与休斯顿太空中心联手，设计师研发了一套可让宇航员吃上新鲜蔬菜的种植系统，该系统包括内含蔬菜种子的包装盒、经过特别设计的封口、浇灌工具以及专门用来存放、培养蔬菜苗的太空舱。简单易行的设计既丰富了宇航员的食谱，也让他们从遥远的太空感受到地球的勃勃生机。

设计：Mirko Ihrig
